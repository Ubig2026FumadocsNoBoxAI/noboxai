API (_Application Programming Interface_) adalah protokol yang memungkinkan aplikasi NoBox untuk berkomunikasi secara aman dengan pelayan (server) Telegram. Untuk menggunakan fitur Telegram Automation, Anda wajib memiliki **API ID** dan **API Hash** milik sendiri agar aktivitas pengiriman pesan Anda memiliki identitas aplikasi yang unik dan tepercaya.

Proses pembuatan API ini gratis dan hanya perlu dilakukan satu kali melalui portal resmi pengembang Telegram.

## **Tahap 1: Autentikasi Akun Telegram**

Langkah pertama adalah masuk ke portal manajemen API Telegram menggunakan nomor telepon Anda.

1. Kunjungi situs resmi: [https://my.telegram.org/auth](https://my.telegram.org/auth).
2. Masukkan nomor telepon yang terdaftar di Telegram (Gunakan format internasional, misal: +6281xxxxxx).
3. Kode konfirmasi akan dikirimkan melalui **chat Telegram** Anda (bukan SMS). Masukkan kode tersebut dan klik **[Sign In]**.

## **Tahap 2: Membuat Aplikasi Baru**

Setelah berhasil masuk, Anda harus mendaftarkan identitas aplikasi untuk akun Anda.

1. Pada menu utama **[Your Telegram Core]**, pilih opsi **[API development tools]**.
2. Anda akan diarahkan ke halaman pembuatan aplikasi (**Create New Application**).
3. Isi kolom yang tersedia sesuai dengan petunjuk berikut:

| Isian           | Uraian Fungsi                                | Contoh Data |
| :-------------- | :------------------------------------------- | :---------- |
| **App title**   | Nama aplikasi buatan Anda (Isi bebas).       | NoBox.AI    |
| **Short name**  | Nama pendek aplikasi.                        | NoBox.AI    |
| **URL**         | Alamat website Anda (Dapat dilewati).        | id.nobox.ai |
| **Platform**    | Platform penggunaan aplikasi.                | Desktop     |
| **Description** | Deskripsi singkat aplikasi (Dapat dilewati). | -           |

4. Klik tombol **[Create Application]** untuk mengonfirmasi pembuatan.

## **Tahap 3: Mendapatkan dan Menyimpan API**

Setelah aplikasi berhasil dibuat, sistem akan menampilkan rincian API Anda.

1. Anda akan melihat informasi **App api_id** (berupa angka) dan **App api_hash** (berupa kombinasi huruf-angka).
2. Salin (_copy_) kedua data tersebut dan simpan di tempat yang aman.
3. Gunakan data tersebut saat Anda menambahkan akun Telegram baru pada aplikasi NoBox Desktop.

<Kendala>
## **Penanganan Kendala Pendaftaran API**

Jika menemui hambatan saat membuat API, silakan cek poin berikut:

- **Gagal "Create Application"**: Telegram terkadang membatasi pembuatan API pada nomor telepon baru. Coba gunakan nomor yang sudah aktif minimal 1-2 minggu.
- **Error "Invalid Name"**: Pastikan _Short Name_ dan _App Title_ hanya menggunakan alfabet dan angka tanpa karakter khusus atau spasi berlebihan.
- **Situs Tidak Merespon**: Gunakan browser Google Chrome terbaru dalam mode _Incognito_ (Pribadi) untuk menghindari konflik cache.
- **Kode OTP Tidak Masuk**: Pastikan aplikasi Telegram di ponsel Anda tetap terbuka, karena kode hanya dikirim melalui sistem chat internal Telegram.

</Kendala>
