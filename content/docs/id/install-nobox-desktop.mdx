## **Mengunduh dan Menjalankan Aplikasi**

Bagian ini menjelaskan proses pengunduhan hingga menjalankan aplikasi NoBox Desktop.

### Langkah-Langkah Unduh dan Instalasi Manual

Ikuti langkah-langkah berikut untuk mengunduh dan menjalankan NoBox Desktop:

1. Unduh aplikasi melalui Google Drive pada tautan berikut:
   [NoBox Release](https://bit.ly/mynobox-release)

2. Setelah file berhasil diunduh, lakukan ekstraksi folder.

3. Buka folder hasil ekstraksi, kemudian masuk ke direktori:
   **NoboxDesktop → Bin**

4. Cari file aplikasi NoBox Desktop.

5. Klik dua kali pada file tersebut untuk menjalankan aplikasi.

Setelah aplikasi terbuka, Anda dapat melanjutkan ke tahap login.

### Catatan Unduhan

- Pastikan proses unduhan selesai 100%.
- Hindari memindahkan file sebelum ekstraksi selesai.
- Simpan folder aplikasi di lokasi yang mudah diakses.

## **Login NoBox Desktop**

Untuk menggunakan NoBox Desktop, Anda harus login menggunakan akun yang telah dibuat melalui website resmi NoBox.AI.

  src="/images/id/Login%20NoBox%20Desktop.png"

### Langkah-Langkah Login

1. Buka aplikasi NoBox Desktop.
2. Masukkan username dan password akun NoBox.AI Web.
3. Pastikan data yang dimasukkan sudah benar.
4. Klik tombol **Login**.
5. Tunggu hingga proses autentikasi selesai.

Jika belum memiliki akun, silakan mengikuti panduan registrasi melalui halaman berikut:
[Registrasi Akun](crm-user-registration)

### Proses Setelah Login

Setelah berhasil login, sistem akan:

- Memuat data akun pengguna.
- Melakukan sinkronisasi awal.
- Menyesuaikan pengaturan aplikasi.
- Menampilkan halaman utama NoBox Desktop.

---

## **Catatan Penting Penggunaan**

Perlu diperhatikan beberapa ketentuan berikut selama menggunakan aplikasi:

- Aplikasi NoBox Desktop hanya dapat dijalankan pada satu komputer dalam satu waktu.
- Penggunaan di lebih dari satu perangkat dapat menyebabkan konflik data.
- Pastikan selalu logout sebelum memindahkan penggunaan ke perangkat lain.
- Jangan membagikan akun kepada pihak lain.

## **Mengatasi Kendala Instalasi**

Jika mengalami kendala selama instalasi atau login, perhatikan hal-hal berikut:

**Pemeriksaan File dan Sistem**

- Pastikan file hasil unduhan tidak rusak
- Gunakan versi terbaru aplikasi
- Pastikan sistem operasi mendukung
- Periksa kapasitas penyimpanan

**Pemeriksaan Akses dan Keamanan**

- Jalankan aplikasi sebagai administrator jika diperlukan
- Nonaktifkan antivirus sementara jika terjadi pemblokiran
- Pastikan firewall tidak memblokir aplikasi

**Solusi Tambahan**

- Restart komputer setelah instalasi
- Coba instal ulang aplikasi
- Periksa koneksi internet
