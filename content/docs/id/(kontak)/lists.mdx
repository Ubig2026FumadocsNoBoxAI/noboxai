List adalah salah satu menu pada NoBox.AI yang digunakan untuk mengelompokkan dan mengorganisir data kontak pelanggan ke dalam daftar-daftar tertentu berdasarkan kebutuhan pengguna.

Menu ini membantu pengguna dalam melakukan segmentasi kontak untuk keperluan kampanye promosi, komunikasi massal, atau pengelolaan pelanggan berdasarkan kategori tertentu. Dengan fitur List, pengguna dapat membuat daftar kontak yang spesifik, menambahkan kontak ke dalam list melalui berbagai metode, serta mengelola keanggotaan list secara fleksibel.

## **Tambah List**

Untuk membuat data list baru, ikuti langkah-langkah berikut:

1. Klik modul **[Kontak]**, lalu klik menu **[List]** ➔ **[Tambah List]**.
2. Maka akan tampil dialog **[Tambah List]**, kemudian isi nama list yang akan dibuat.

## **Edit dan Hapus Data List**

### Edit Data List

Untuk mengedit suatu data list, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan nama list akan langsung tersimpan dan berlaku pada seluruh data kontak yang tergabung dalam list tersebut.

### Hapus Data List

Untuk menghapus suatu data list, pilih data list yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]**. Jika tidak, klik **[Tidak]**.

<Callout type="info" title="Informasi Penting">
Menghapus list hanya akan menghapus pengelompokan, data kontak yang ada di dalamnya tetap tersimpan di database. Namun, pastikan list tersebut tidak sedang digunakan oleh kampanye promosi yang aktif.
</Callout>

## **Menambah Kontak ke dalam List**

Terdapat beberapa metode untuk menambahkan kontak ke dalam sebuah list. Setiap metode dirancang untuk kebutuhan yang berbeda:

### Tambah dari Kontak

Metode ini digunakan untuk menambahkan kontak yang sudah tersimpan di database ke dalam list tertentu.

Cara menambahkan kontak dari database:

1. Pilih list yang akan digunakan pada halaman List.
2. Klik tombol **[Tambah Kontak]** ➔ klik tombol **[Tambah dari Kontak]**.
3. Pilih satu atau beberapa kontak yang ingin ditambahkan dari daftar yang muncul.
4. Klik **[OK]** untuk menambahkan kontak yang dipilih ke dalam list.

<ImageZoom
  src="/images/id/tambah%20dari%20kontak.png"

### Tambah dari Target Promosi

Metode ini digunakan untuk menambahkan kontak yang pernah menjadi target promosi ke dalam list tertentu. Fitur ini sangat berguna untuk mengumpulkan kontak yang telah merespon kampanye sebelumnya.

Cara menambahkan kontak dari target promosi:

1. Pilih list yang akan digunakan pada halaman List.
2. Klik tombol **[Tambah Kontak]** ➔ klik tombol **[Tambah dari Target Promosi]**.
3. Pilih data promosi yang kontak targetnya ingin Anda tambahkan.
4. Klik **[OK]** untuk menambahkan kontak dari promosi tersebut ke dalam list.

## **Hapus Semua Kontak dari List**

Fitur ini digunakan untuk menghapus seluruh data kontak yang ada di dalam suatu list sekaligus. Fitur ini berguna ketika Anda ingin mengosongkan list untuk digunakan kembali dengan set kontak yang berbeda.

Untuk menghapus semua kontak dari list, ikuti langkah berikut:

1. Pilih data list yang diinginkan pada halaman List.
2. Klik **[Hapus Semua Kontak]**.
3. Maka akan tampil dialog konfirmasi apakah Anda ingin menghapus semua data kontak yang ada di list tersebut.
4. Jika ya, klik **[Ya]**. Jika tidak, klik **[Tidak]**.

<Callout type="info" title="Informasi Penting">
Menghapus semua kontak dari list hanya akan mengeluarkan kontak dari pengelompokan tersebut. Data kontak itu sendiri tetap tersimpan di database dan dapat ditambahkan kembali ke list kapan saja.
</Callout>

## **Penggunaan List pada Promosi**

List memiliki peran penting dalam fitur Promosi NoBox.AI. Saat membuat kampanye promosi, pengguna dapat memilih list sebagai target penerima promosi.

Dengan menggunakan list, proses penargetan promosi menjadi lebih mudah karena pengguna tidak perlu memilih kontak satu per satu, melainkan cukup memilih list yang sesuai dengan segmen target promosi.

<Tips>
## **Tips Penggunaan**
 
Berikut beberapa tips untuk memaksimalkan penggunaan menu List:
 
- Gunakan List untuk mengkategorikan kontak berdasarkan minat, lokasi, status pelanggan, atau periode pembelian agar segmentasi promosi lebih akurat.
- Manfaatkan fitur **[Tambah dari Kontak]** untuk memasukkan data kontak yang sudah ada ke dalam list baru tanpa perlu input manual.
- Gunakan **[Tambah dari Target Promosi]** untuk mengumpulkan kontak yang merespon kampanye tertentu ke dalam list baru untuk follow-up.
- Buat nama list yang deskriptif dan konsisten (contoh: "Pelanggan VIP Surabaya", "Reseller Aktif 2024") agar mudah diidentifikasi.
- Lakukan pembersihan list secara berkala menggunakan **[Hapus Semua Kontak]** jika list tersebut ingin digunakan kembali untuk kategori yang berbeda.
- Manfaatkan fitur list saat membuat promosi untuk mempermudah penargetan tanpa perlu memilih kontak satu per satu.
 
</Tips>

<Kendala>
## **Penanganan Kendala**
 
Jika mengalami kendala saat menggunakan menu List, lakukan langkah-langkah berikut:
 
- Refresh halaman jika jumlah kontak di dalam list tidak segera berubah setelah penambahan kontak baru.
- Pastikan Anda memilih kontak yang benar saat menggunakan fitur **[Tambah dari Kontak]** atau **[Tambah dari Target Promosi]**.
- Periksa kembali koneksi internet jika proses penambahan kontak ke dalam list terasa lambat.
- Periksa hak akses akun Anda jika tidak dapat membuat, mengedit, atau menghapus list.
- Logout dan login kembali jika data list tidak ter-update setelah perubahan.
- Bersihkan cache browser jika tampilan tidak sesuai.
- Hubungi administrator atau tim support jika fitur **[Hapus Semua Kontak]** tidak dapat dijalankan (mungkin terkait hak akses).
 
</Kendala>
