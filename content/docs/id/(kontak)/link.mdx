Link adalah salah satu menu pada NoBox.AI yang digunakan untuk menampung dan mengelola akun-akun komunikasi (channel) yang terhubung dengan data kontak pelanggan.

Menu ini membantu pengguna dalam melihat seluruh link atau akun eksternal seperti nomor WhatsApp, alamat Email, akun Telegram, dan platform lainnya secara terpusat. Setiap link yang terdaftar memungkinkan sistem untuk mengirim pesan, melacak percakapan, serta memantau status pengiriman pesan siaran (broadcast) yang ditujukan ke akun tersebut.

## **Tambah Link**

Untuk menambah data link baru, ikuti langkah-langkah berikut:

1. Klik modul **[Kontak]**, lalu klik menu **[Link]** ➔ **[Tambah Link]**.
2. Maka akan tampil dialog **[Tambah Link]**, kemudian lengkapi seluruh isian pada dialog tersebut.

## **Edit dan Hapus Data Link**

### Edit Data Link

Untuk mengedit suatu data link, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan berlaku pada seluruh fitur yang menggunakan link tersebut.

### Hapus Data Link

Untuk menghapus suatu data link, pilih data link yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]**. Jika tidak, klik **[Tidak]**.

<Callout type="info" title="Informasi Penting">
Penghapusan link akan memutus koneksi antara kontak dengan platform komunikasi tersebut. Pastikan data yang akan dihapus benar-benar tidak diperlukan lagi.
</Callout>

## **Percakapan**

Tab Percakapan pada data link digunakan untuk menampung seluruh histori percakapan yang terjadi melalui link tersebut.

Tab ini mencakup percakapan dari semua status (Baru, Ditugaskan, dan Selesai). Semua percakapan yang belum diarsipkan akan tetap termonitoring dan terhubung dengan data link, sehingga pengguna dapat melacak seluruh riwayat komunikasi melalui kanal tersebut.

Pada tab ini, pengguna dapat:

- Memantau status percakapan (Baru, Ditugaskan, Selesai).
- Mengidentifikasi percakapan yang belum ditangani untuk tindak lanjut segera.

## **Kirim Pesan**

Tab Kirim Pesan pada data link digunakan untuk menampung informasi pengiriman pesan siaran (broadcast) yang dikirim ke link tersebut.

Melalui tab ini, pengguna dapat memonitor hasil pengiriman pesan siaran, termasuk pesan yang sudah terkirim, belum terkirim, atau gagal terkirim ke akun komunikasi yang bersangkutan.

- Memantau status pengiriman setiap pesan (Terkirim, Dalam Antrean, Gagal).
- Mengevaluasi efektivitas kampanye promosi yang ditujukan ke kontak melalui link tersebut.

<Tips>
## **Tips Penggunaan**
 
Agar pengelolaan Link lebih optimal, perhatikan beberapa tips berikut:
 
- Tambahkan berbagai kanal komunikasi (WhatsApp, Email, Telegram) pada setiap kontak untuk menjangkau pelanggan dari berbagai platform.
- Gunakan ID Eksternal yang benar dan lengkap (nomor HP dengan kode negara untuk WhatsApp, alamat email valid untuk Email) agar pesan dapat terkirim dengan baik.
- Pantau tab Percakapan secara berkala untuk melihat histori interaksi yang belum ditangani.
- Manfaatkan tab Kirim Pesan untuk mengevaluasi status pengiriman pesan siaran dan mengidentifikasi pesan yang gagal.
- Pastikan setiap kontak memiliki minimal satu link aktif agar dapat dihubungi melalui kampanye promosi.
- Perbarui data link jika terjadi perubahan nomor telepon atau alamat email kontak.
 
</Tips>

<Kendala>
## **Penanganan Kendala**
 
Jika mengalami kendala terkait menu Link, lakukan langkah berikut:
 
- Periksa kembali format ID Eksternal jika pesan siaran terus-menerus gagal terkirim (pastikan kode negara sudah benar untuk nomor telepon).
- Refresh halaman jika tab Percakapan atau Kirim Pesan tidak menampilkan data terbaru.
- Pastikan koneksi internet stabil saat menambahkan atau mengedit data link.
- Periksa hak akses akun Anda jika tidak dapat menambah atau menghapus data link.
- Logout dan login kembali jika sesi Anda berakhir saat melakukan pembaruan data.
- Bersihkan cache browser jika tampilan tidak sesuai.
- Hubungi tim support jika link yang sudah terhubung tetap tidak dapat menerima pesan promosi.
 
</Kendala>
