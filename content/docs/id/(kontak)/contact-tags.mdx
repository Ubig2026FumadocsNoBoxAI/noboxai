Tag Kontak adalah salah satu menu pada NoBox.AI yang digunakan untuk mengelompokkan dan mengkategorikan data kontak pelanggan berdasarkan label tertentu.

Menu ini membantu pengguna dalam mengorganisir database kontak agar lebih terstruktur, sehingga proses pencarian, penyaringan, dan pengelolaan kontak dapat dilakukan secara lebih efisien. Dengan penggunaan tag yang tepat, pengguna dapat mengidentifikasi jenis pelanggan, status interaksi, atau segmen audiens dengan cepat dan akurat.

## **Tambah Tag Kontak**

1. Untuk membuat data tag baru, silahkan klik menu **[Tag Kontak]** ➔ **[Tambah Tag Kontak]**.
2. Maka akan tampil dialog **[Tambah Tag Kontak]**, kemudian isi nama tag yang akan dibuat.

## **Edit dan Hapus Data Tag Kontak**

### Edit Data Tag Kontak

Untuk mengedit suatu data tag, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan nama tag akan langsung tersimpan dan berlaku pada seluruh data kontak yang menggunakan tag tersebut.

### Hapus Data Tag Kontak

Untuk menghapus suatu data tag, silahkan pilih data tag yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]**. Jika tidak, klik **[Tidak]**.

<Callout type="info" title="Informasi Penting">
  Pastikan tag yang akan dihapus tidak sedang digunakan oleh kontak yang masih
  aktif. Penghapusan tag bersifat permanen dan tidak dapat dibatalkan.
</Callout>

## **Cara Penggunaan Tag Kontak**

Fitur Tag Kontak dapat digunakan secara langsung saat membuat atau mengedit data kontak.

Untuk menambahkan tag pada kontak, ikuti langkah-langkah berikut:

1. Buka menu **[Kontak]**, pilih salah satu data kontak yang sudah dibuat atau tambahkan kontak baru.

2. Klik pada bagian isian di kolom **[Tag]**, maka akan muncul daftar tag yang sudah dibuat sebelumnya.

3. Pilih satu atau lebih dari data tag tersebut sesuai kebutuhan pengelompokan kontak.

## **Filter Tag Kontak**

Filter tag kontak digunakan untuk memilah data kontak dan menampilkan data berdasarkan tag tertentu yang dipilih. Fitur ini sangat membantu ketika pengguna memiliki banyak data kontak dan ingin menampilkan hanya kontak dengan kategori tertentu.

Cara memfilter data kontak berdasarkan tag:

1. Buka menu **[Kontak]**, klik tombol filter **[Tags]** yang tersedia di bagian atas halaman.
2. Pilih data tag mana yang ingin Anda gunakan sebagai filter.
3. Maka akan tampil data kontak yang telah terfilter berdasarkan tag yang dipilih.

<ImageZoom
  src="/images/id/4.%20Filter%20Tag%20Kontak.png"

Untuk menghapus filter dan menampilkan kembali semua data kontak, klik tombol **[Reset]** atau hapus pilihan tag yang sudah diterapkan.

<Tips>
## **Tips Penggunaan**
Untuk pengelolaan tag kontak yang lebih optimal, perhatikan beberapa tips berikut:
- Buat nama tag yang spesifik dan mudah dipahami agar tidak membingungkan (contoh: Pelanggan VIP, Hot Prospect, Reseller Aktif).
- Gunakan lebih dari satu tag pada kontak jika diperlukan untuk segmentasi yang lebih detail.
- Kelompokkan kontak dengan tag yang relevan untuk mempermudah pencarian dan targeting promosi.
- Manfaatkan fitur filter tag untuk mempercepat pencarian kontak dengan kategori tertentu.
- Buat tag berdasarkan status pelanggan (Baru, Aktif, Tidak Aktif) untuk memudahkan monitoring.
- Lakukan pembersihan tag yang sudah tidak digunakan secara berkala agar database tetap rapi.
</Tips>

<Kendala>
## **Penanganan Kendala**
Jika mengalami kendala saat menggunakan Tag Kontak, lakukan langkah berikut:
- Refresh halaman jika tag yang baru dibuat tidak langsung muncul di daftar atau di kolom tag pada formulir kontak.
- Periksa koneksi internet jika data tag gagal disimpan atau dihapus.
- Pastikan nama tag tidak duplikat agar tidak membingungkan saat melakukan filter.
- Periksa hak akses akun Anda jika tidak dapat menambah atau menghapus tag.
- Logout dan login kembali jika fitur filter tag tidak berfungsi dengan baik.
- Bersihkan cache browser jika tampilan tidak sesuai.
- Hubungi tim support jika penghapusan tag mengalami kendala teknis atau mempengaruhi data kontak yang sudah ada.
</Kendala>
