Kontak adalah salah satu menu pada NoBox.AI yang digunakan untuk menampung, mengelola, dan mengorganisir seluruh data kontak pelanggan yang diperoleh dari berbagai sumber.

Menu ini membantu pengguna dalam menangani data target promosi, menyimpan informasi pelanggan secara terstruktur, serta mendukung kebutuhan CRM seperti pengelolaan tugas, penawaran, dan catatan interaksi. Dengan fitur-fitur yang tersedia, pengguna dapat melakukan impor data secara massal, menambah kontak manual, serta mengekspor data ke platform eksternal seperti Google Kontak dan Facebook Audiences.

## **Cara Impor Excel & CSV**

Fitur ini digunakan untuk mengimpor data kontak secara massal dari file Excel atau CSV yang sudah Anda miliki ke dalam database NoBox.AI.

Untuk mengimpor data kontak, ikuti langkah-langkah berikut:

1. Buka menu **[Kontak]** ➔ klik fitur **[Impor Excel & CSV]**.

<ImageZoom
  src="/images/id/impor%20excel%20&#x26;%20csv.png"
  alt="Fitur Impor Excel & CSV"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

2. Kemudian **[Unduh Model]** terlebih dahulu untuk mendapatkan template format yang benar, atau upload file Excel/CSV yang sudah Anda miliki. Klik **[Pilih File]** ➔ pilih file Excel atau CSV Anda, kemudian klik **[Impor]**.

<ImageZoom
  src="/images/id/Impor%20data%20excel&#x26;csv.png"
  alt="Proses Upload File Impor"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

3. Maka akan tampil dialog **[Impor Excel & CSV]**. Lengkapi isian kanal, list, negara, kategori, dan tag sesuai kebutuhan.

<ImageZoom
  src="/images/id/hasil%20impor%20excel&#x26;csv.png"
  alt="Dialog Pengisian Data Impor"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

4. Kemudian lakukan mapping data Excel dengan field di NoBox.AI. Mapping dilakukan dengan cara memilih nama field yang sesuai dengan nama kolom pada data Excel Anda.

Pastikan Anda memilih field **[ID Eksternal]** untuk semua data seperti nomor telepon, email, dan lainnya. Setelah itu, klik tombol **[Simpan]** untuk menyimpan data.

<ImageZoom
  src="/images/id/mapping%20data%20excel.png"
  alt="Mapping Data Excel ke Field NoBox"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

## **Cara Ekspor ke Google Kontak**

Fitur ini digunakan untuk mengunduh data kontak dalam format CSV yang dapat langsung diimpor ke Google Kontak.

Untuk mengekspor data ke Google Kontak, ikuti langkah-langkah berikut:

1. Klik fitur **[Google Contacts]** pada halaman Kontak, maka sistem akan otomatis mengunduh data kontak Anda dalam format CSV.

<ImageZoom
  src="/images/id/google%20contact.png"
  alt="Tombol Ekspor ke Google Contact"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

2. Buka Google Kontak di akun Google Anda, lalu klik **[Impor kontak]**.

<ImageZoom
  src="/images/id/google%20kontak%20(impor).png"
  alt="Menu Impor di Google Kontak"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

3. Pilih file CSV yang sudah diunduh dari NoBox.AI, lalu klik **[Impor]**.

<ImageZoom
  src="/images/id/impor%20kontak.png"
  alt="Konfirmasi Impor File CSV"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

4. Maka data kontak akan tampil di Google Kontak. Pilih data dan klik ikon [ <img src="/images/id/Tambah.png" alt="Tambah" className="inline h-5 w-auto mx-1" /> ] untuk menambahkan kontak pada akun Google Anda.

<ImageZoom
  src="/images/id/hasil%20impor%20kontak.png"
  alt="Hasil Akhir Impor Kontak di Google"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

### Edit dan Hapus Data Link

#### Edit Data Link

Untuk mengedit suatu data link, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

#### Hapus Data Link

Untuk menghapus suatu data link adalah dengan pilih data link yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.

<Callout type="info" title="Informasi Penting">
Penghapusan data link bersifat permanen dan tidak dapat dibatalkan. Pastikan data yang akan dihapus benar-benar tidak diperlukan lagi.
</Callout>

## **vCard**

vCard adalah fitur yang memungkinkan Anda mengirim informasi kontak dalam format standar yang dapat dibaca oleh program email dan aplikasi kontak lainnya.

File vCard disimpan dalam format .vcf dan dapat digunakan untuk berbagi data kontak secara cepat dan mudah ke perangkat atau aplikasi lain.

<ImageZoom
  src="/images/id/VCard.png"
  alt="Fitur vCard"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

Klik tombol **[vCard]** pada halaman detail kontak untuk mengunduh file vCard dari kontak yang dipilih.

## **Gabung Kontak**

Gabung Kontak adalah fitur yang digunakan untuk menggabungkan dua atau lebih data kontak yang duplikat menjadi satu data kontak yang lengkap.

Fitur ini sangat berguna dalam manajemen kontak untuk menghindari duplikasi dan memastikan data yang bersih serta terorganisir, sehingga meminimalkan kekacauan dan meningkatkan efisiensi dalam mengelola daftar kontak.

<ImageZoom
  src="/images/id/gabung%20kontak.png"
  alt="Fitur Gabung Kontak"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

Untuk menggabungkan kontak, pilih kontak utama yang akan dipertahankan, kemudian pilih kontak duplikat yang ingin digabungkan. Data dari kontak duplikat akan disatukan ke dalam kontak utama.

## **Tips Penggunaan**
 
Beberapa tips untuk mengelola kontak secara lebih efisien:
 
- Gunakan fitur **[Impor Excel/CSV]** untuk memindahkan data kontak secara massal dengan cepat. Pastikan Anda mengunduh model terlebih dahulu agar format file sesuai.
- Mapping field **[ID Eksternal]** dengan benar (seperti nomor telepon atau email) agar data tidak duplikat dan pesan dapat terkirim dengan baik.
- Manfaatkan fitur **[vCard]** untuk berbagi informasi kontak ke aplikasi lain secara cepat.
- Gunakan fitur **[Gabung Kontak]** secara berkala untuk membersihkan data duplikat dan menjaga database tetap rapi.
- Manfaatkan tab **[Tugas]** dan **[Penawaran]** untuk melacak aktivitas dan progres penjualan yang terkait dengan setiap kontak.
- Tambahkan catatan di setiap interaksi penting agar histori komunikasi dengan kontak tetap terekam dengan baik.
- Gunakan fitur ekspor ke **[Google Kontak]** atau **[Fb Audiences]** untuk memaksimalkan jangkauan promosi di platform lain.
 
</Tips>

## **Penanganan Kendala**
 
Jika mengalami kendala saat mengelola data Kontak, lakukan langkah berikut:
 
- Jika impor Excel/CSV gagal, pastikan format file sudah sesuai dengan model yang diunduh dari sistem.
- Periksa field ID Eksternal jika data kontak yang diimpor terus mengalami duplikasi.
- Refresh halaman jika kontak yang baru ditambahkan secara manual tidak muncul di daftar.
- Periksa hak akses akun Anda jika tidak dapat melakukan edit atau hapus pada data tertentu.
- Pastikan koneksi internet stabil saat melakukan sinkronisasi dengan Google Kontak atau Facebook Audiences.
- Logout dan login kembali jika fitur filter atau pencarian tidak berfungsi normal.
- Bersihkan cache browser jika tampilan tidak sesuai.
- Hubungi tim support jika tautan (link) eksternal seperti WhatsApp atau Email tidak berfungsi atau mengalami kendala teknis.
