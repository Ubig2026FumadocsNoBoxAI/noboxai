---
 
<Tips>
## **Tips Penggunaan**
 
Agar fitur Dasbor Kontak dapat digunakan secara maksimal, perhatikan beberapa tips berikut:
 
- Gunakan grafik untuk melihat ringkasan data kontak secara cepat.
- Klik pada grafik untuk melihat detail data dari masing-masing kategori.
- Manfaatkan informasi sumber teratas untuk mengetahui asal kontak yang paling banyak.
- Periksa sebaran negara kontak untuk memetakan jangkauan audiens Anda.
 
</Tips>
 
---
 
<Kendala>
## **Penanganan Kendala**
 
Jika mengalami kendala saat menggunakan Dasbor Kontak, lakukan langkah berikut:
 
- Refresh halaman jika grafik tidak muncul atau data tidak terupdate.
- Pastikan koneksi internet stabil saat memuat data dasbor.
- Periksa kembali filter periode waktu jika data yang dicari tidak tampil.
- Logout dan login kembali jika terdapat error teknis pada tampilan.
 
</Kendala>
