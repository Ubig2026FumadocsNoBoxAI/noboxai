Menu Berlangganan (Subscription) adalah pusat informasi biaya dan paket layanan yang Anda gunakan di NoBox.AI. Melalui halaman ini, Anda dapat memantau kuota yang tersedia, melacak penggunaan penyimpanan, hingga melihat riwayat transaksi produk yang telah dibeli.

Halaman ini sangat penting bagi pemilik bisnis untuk memastikan seluruh operasional sistem tetap berjalan lancar dengan memantau kapasitas akun secara rutin.

Tampilan di atas menunjukkan ringkasan paket langganan Anda. Halaman ini dibagi menjadi beberapa modul informasi utama:

| Komponen            | Fungsi Utama                                                          |
| :------------------ | :-------------------------------------------------------------------- |
| **Anggota**         | Menampilkan total anggota tim atau agen yang terdaftar.               |
| **Akun**            | Menampilkan jumlah akun media sosial/marketplace yang terhubung.      |
| **Dukungan**        | Informasi mengenai level layanan bantuan (_support plan_) Anda.       |
| **Penyimpanan**     | Persentase penggunaan memori penyimpanan untuk lampiran dan database. |
| **Riwayat Pesanan** | Daftar seluruh transaksi atau _invoice_ pembelian produk NoBox.       |
| **Kuota**           | Detail jatah ekstraksi data dan limit pesan siaran (broadcast).       |

## **Monitoring Kuota dan Penggunaan**

Anda dapat melakukan pemantauan mendalam terhadap aset digital Anda melalui filter yang tersedia di bagian bawah halaman.

### Filter Berdasarkan Kanal

<Tips>
## **Tips Penggunaan**

Beberapa tips untuk mengelola langganan Anda secara efisien:

- Periksa kuota secara berkala, terutama sebelum menjalankan kampanye promosi besar.
- Pantau bar penyimpanan; jika sudah mendekati limit, lakukan pembersihan file lama atau upgrade paket.
- Gunakan riwayat pesanan (Order History) sebagai bahan laporan keuangan internal perusahaan Anda.
- Pastikan level dukungan (Support) sesuai dengan kebutuhan bisnis Anda agar mendapat respon prioritas saat terjadi kendala.

</Tips>

<Kendala>
## **Penanganan Kendala**

Jika menemui kejanggalan pada informasi langganan, pertimbangkan langkah berikut:

- **Kuota Tidak Bertambah Setelah Top-up**: Muat ulang (_refresh_) halaman atau tunggu 5-10 menit hingga sistem sinkronisasi selesai.
- **Penyimpanan Penuh**: Hapus beberapa lampiran file besar di File Manager atau hubungi kami untuk menambah kapasitas.
- **Riwayat Pesanan Kosong**: Pastikan Anda login dengan akun yang sama saat melakukan pembelian.
- **Data Tidak Muncul**: Periksa koneksi internet Anda atau pastikan tidak ada pemblokir iklan (ad-blocker) yang mengganggu tampilan sistem.

</Kendala>
