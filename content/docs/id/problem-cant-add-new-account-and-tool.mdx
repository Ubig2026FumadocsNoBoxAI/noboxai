Salah satu kendala yang terkadang dialami pengguna adalah ketidakmampuan untuk menambahkan akun media sosial baru (seperti WhatsApp atau Telegram) atau tool extractor ke dalam sistem. Masalah ini biasanya berkaitan dengan validasi data formulir atau sinkronisasi antara NoBox Desktop dengan server cloud.

Halaman ini akan memandu Anda melakukan langkah-langkah pemeriksaan mandiri untuk mengatasi kendala tersebut secara cepat.

---

## **Langkah Perbaikan Mandiri**

Jika tombol simpan tidak berfungsi atau akun tidak muncul setelah ditambahkan, silakan lakukan pemeriksaan berikut:

### 1. Verifikasi Status Akun

Pastikan Anda sedang login menggunakan akun yang telah terdaftar secara resmi di website [id.NoBox.AI](https://NoBox.AI/). Akun yang belum terverifikasi emailnya mungkin memiliki batasan dalam menambah sumber daya baru.

### 2. Validasi Isian Formulir

Periksa kembali apakah seluruh kolom pada dialog **[Tambah Akun]** atau **[Tambah Tool]** telah diisi dengan benar.

- Pastikan tidak ada spasi kosong di awal atau akhir nama akun.
- Pilih kanal (channel) yang sesuai dengan layanan yang ingin Anda gunakan.
- Pastikan format nomor telepon (jika diminta) menggunakan kode negara yang benar.

### 3. Cek Ketersediaan di Dashboard Web

Setelah menekan tombol simpan di aplikasi desktop, silakan buka menu **[Akun]** pada dashboard website [id.NoBox.AI](https://NoBox.AI/). Jika data muncul di sana, berarti proses penambahan berhasil dan hanya membutuhkan waktu beberapa saat untuk sinkronisasi ke tampilan desktop Anda.

## **Integrasi API Telegram**

Khusus untuk akun Telegram, Anda tidak dapat langsung menghubungkannya tanpa melakukan konfigurasi API terlebih dahulu.

<Callout type="info" title="Penting untuk Pengguna Telegram">
  Formulir penambahan akun Telegram memerlukan informasi API ID dan API Hash.
  Jika Anda belum memiliki data tersebut, silakan ikuti [Panduan Konfigurasi API
  Telegram](configuration-telegram-api) sebelum mencoba menambahkan akun
  kembali.
</Callout>

<Kendala>
## **Penanganan Kendala Lanjutan**

Jika masalah tetap berlanjut, silakan coba langkah teknis berikut:

- **Logout & Login Ulang**: Keluar dari aplikasi NoBox Desktop dan masuk kembali untuk memaksa sistem menarik data terbaru dari server.
- **Refresh Browser**: Jika Anda menggunakan dashboard web, tekan tombol F5 untuk menyegarkan tampilan data.
- **Cek Kuota Subscription**: Pastikan paket langganan Anda masih memiliki slot yang cukup untuk menambah akun baru.
- **Whitelist Firewall**: Pastikan aplikasi NoBox diizinkan oleh firewall windows untuk melakukan komunikasi data dua arah.

</Kendala>
