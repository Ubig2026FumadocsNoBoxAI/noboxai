OLX Sites Extractor adalah alat pada NoBox Desktop yang dirancang untuk mengumpulkan data iklan dari platform marketplace OLX secara otomatis. Fitur ini memungkinkan Anda mengekstrak informasi detail produk, harga, lokasi penjual, hingga nomor telepon untuk kebutuhan riset pasar atau database prospek bisnis.

Dengan antarmuka yang ramah pengguna, Anda dapat melakukan pencarian berbasis kategori dan lokasi di seluruh wilayah Indonesia dengan sangat efisien.

## **Fitur Utama OLX Extractor**

Software ini dilengkapi dengan berbagai fitur cerdas untuk meningkatkan kecepatan dan keamanan saat melakukan ekstraksi data.

### **1. Auto Login**

OLX Extractor mendukung fitur login otomatis. Dengan menyimpan kredensial akun OLX Anda, sistem dapat masuk ke platform tanpa perlu mengetik ulang password secara berkala, mempermudah akses ke data iklan yang memerlukan autentikasi.

## **Manajemen Konfigurasi (Page Process)**

Sebelum memulai proses ekstraksi, Anda perlu mengatur parameter pencarian pada tab **Process**.

| Fitur            | Uraian Fungsi                                                                          |
| :--------------- | :------------------------------------------------------------------------------------- |
| **Limit Result** | Menentukan jumlah maksimal data iklan yang akan diambil.                               |
| **Save To List** | Memilih kategori atau grup kontak untuk menyimpan hasil ekstraksi di website NoBox.AI. |
| **Option Debug** | Mengaktifkan catatan teknis (logs) secara mendalam untuk memantau proses kerja sistem. |
| **Load Images**  | Mengatur apakah gambar iklan perlu dimuat atau tidak guna menghemat resource.          |

<Tips>
## **Tips Penggunaan**
Optimalkan penggunaan OLX Extractor dengan langkah-langkah berikut:
- Selalu gunakan fitur **Auto Login** untuk mempermudah operasional harian.
- Gunakan fitur **Multi Tab** jika Anda perlu riset kompetitor di berbagai wilayah sekaligus (misal: Jakarta, Surabaya, Bandung).
- Pastikan untuk memilih file **Save To List** yang spesifik sesuai kategori barang agar database Anda tetap rapi.
- Aktifkan **Auto Refresh** pada website NoBox.AI untuk memantau penambahan data tanpa perlu refresh manual.
- Lakukan ekspor data secara berkala untuk analisis pasar mingguan atau bulanan.
</Tips>

<Kendala>
## **Penanganan Kendala**
Jika menemui masalah saat menggunakan extractor, silakan lakukan hal berikut:
- **Login Gagal**: Periksa kembali akurasi email dan password pada menu **Other Options** atau pastikan koneksi internet aktif.
- **Pencarian Terhenti**: Jika komputer mati mendadak, buka kembali aplikasi dan klik **[OK]** pada dialog pemulihan proses.
- **Data Tidak Muncul**: Periksa tab **Logs** untuk melihat apakah terjadi error pada saat akses halaman OLX.
- **Limit Tercapai**: Jika data berhenti diambil, periksa pengaturan **Limit Result** Anda atau kuota subscription Anda.
- **Sinkronisasi Terhambat**: Pastikan status akun extractor di NoBox Desktop dalam kondisi "Aktif".
</Kendala>
