Secara umum, NoBox.AI memiliki empat jenis tampilan utama, yaitu Dasbor, Kanban, List, dan Formulir. Keempat tampilan ini saling terintegrasi untuk mendukung proses kerja pengguna dari awal hingga akhir.

---

## **Dasbor**

Dasbor merupakan halaman utama yang menampilkan ringkasan aktivitas dan informasi penting secara real time. Halaman ini biasanya menjadi tampilan pertama yang muncul setelah pengguna berhasil masuk ke dalam sistem.

Melalui dasbor, pengguna dapat memantau performa layanan, perkembangan data, tren operasional, serta kondisi sistem secara menyeluruh dalam satu tampilan.

Informasi pada dasbor disajikan dalam bentuk grafik, statistik, dan ringkasan visual sehingga memudahkan pengguna dalam memahami data tanpa harus membuka menu lain.

Tampilan ini sangat membantu dalam proses pengambilan keputusan karena menyajikan data yang telah diolah secara otomatis oleh sistem.

### Fungsi Utama Dasbor

Beberapa fungsi utama dari tampilan dasbor antara lain:

- Menampilkan ringkasan performa sistem.
- Menyediakan laporan singkat aktivitas pengguna.
- Memantau status layanan secara real time.
- Menampilkan notifikasi penting.
- Mendukung evaluasi dan pengambilan keputusan.

### Fitur Umum pada Dasbor

| Fitur / Tombol   | Fungsi                                              |
| :--------------- | :-------------------------------------------------- |
| **Filter Tahun** | Menampilkan data berdasarkan periode waktu tertentu |
| **Auto Refresh** | Memperbarui data secara otomatis setiap 10 detik    |
| **Bantuan**      | Menampilkan panduan penggunaan dan knowledge base   |
| **Full Screen**  | Menampilkan grafik dalam mode layar penuh           |

## **Kanban**

Tampilan Kanban digunakan untuk mengelola alur kerja dan proses bisnis secara visual. Pada tampilan ini, data disusun dalam bentuk papan (board), kolom (column), dan kartu (card).

Setiap kartu biasanya merepresentasikan satu aktivitas, tugas, atau transaksi yang sedang diproses oleh sistem.

Dengan tampilan ini, pengguna dapat dengan mudah mengetahui status pekerjaan, memindahkan data antar tahap, serta memantau progres secara menyeluruh.

Metode Kanban sangat efektif untuk mengelola pipeline penjualan, proses penawaran, serta workflow internal perusahaan.

### Fungsi Utama Kanban

Fungsi utama tampilan Kanban meliputi:

- Mengelola proses kerja secara bertahap.
- Memantau progres setiap aktivitas.
- Mengurangi risiko keterlambatan pekerjaan.
- Meningkatkan koordinasi antar tim.
- Mempermudah identifikasi kendala proses.

### Fitur Umum pada Kanban

| Fitur / Tombol | Fungsi                                         |
| :------------- | :--------------------------------------------- |
| **Papan**      | Memilih proyek atau data yang akan ditampilkan |
| **Filter**     | Menyaring data berdasarkan kriteria tertentu   |
| **Cari**       | Mencari data menggunakan kata kunci            |
| **Kanban**     | Menampilkan alur kerja secara visual           |

## **List**

Tampilan List menyajikan data dalam bentuk tabel atau daftar yang tersusun secara sistematis. Tampilan ini sangat cocok digunakan untuk mengelola data dalam jumlah besar.

Pada halaman List, pengguna dapat melihat detail informasi setiap data secara lebih lengkap dalam bentuk baris dan kolom.

Selain itu, tampilan ini juga mendukung proses pencarian, pengurutan, dan pengolahan data secara langsung.

Pengguna dapat mengelola data seperti menambah, mengedit, menghapus, maupun mengekspor data melalui halaman ini.

### Fungsi Utama List

Beberapa fungsi utama tampilan List yaitu:

- Menampilkan data secara terstruktur.
- Mempermudah pengelolaan data massal.
- Mendukung proses pencarian cepat.
- Menyediakan fitur ekspor data.
- Menyederhanakan proses administrasi.

### Fitur Umum pada List

| Fitur / Tombol      | Fungsi                                          |
| :------------------ | :---------------------------------------------- |
| **Cari**            | Mencari data berdasarkan kata kunci             |
| **Refresh**         | Memperbarui data secara manual                  |
| **Column Picker**   | Mengatur tampilan kolom pada tabel              |
| **Restore Default** | Mengembalikan pengaturan kolom ke kondisi awal  |
| **Export**          | Mengunduh data dalam format Excel atau PDF      |
| **Auto Refresh**    | Memperbarui data otomatis setiap 10 detik       |
| **Filter Status**   | Menyaring data berdasarkan status atau kategori |

## **Formulir**

Tampilan Formulir digunakan untuk menambahkan data baru, mengedit data yang sudah ada, serta melakukan validasi input sebelum data disimpan ke dalam sistem.

Formulir dirancang dengan struktur yang jelas agar pengguna dapat mengisi data dengan mudah dan benar.

Setiap isian biasanya telah disesuaikan dengan kebutuhan sistem dan dilengkapi dengan validasi otomatis.

Hal ini bertujuan untuk meminimalkan kesalahan input dan menjaga kualitas data.

### Fungsi Utama Formulir

Fungsi utama tampilan Formulir antara lain:

- Menambahkan data baru ke sistem.
- Memperbarui data yang sudah ada.
- Memastikan kelengkapan informasi.
- Melakukan validasi otomatis.
- Menjaga konsistensi data.

### Fitur Umum pada Formulir

| Fitur / Tombol      | Fungsi                                               |
| :------------------ | :--------------------------------------------------- |
| **Grup / Kategori** | Mengelompokkan isian agar lebih rapi dan terstruktur |
| **Simpan**          | Menyimpan data baru atau perubahan data              |
| **Hapus**           | Menghapus data yang dipilih dari sistem              |

<Tips>
  ## **Tips Penggunaan** Untuk memaksimalkan penggunaan tampilan di NoBox.AI,
  perhatikan beberapa tips berikut: - Gunakan filter dan pencarian untuk
  mempercepat proses pencarian data. - Aktifkan auto refresh saat memantau data
  real time. - Manfaatkan fitur export untuk kebutuhan laporan. - Sesuaikan
  tampilan kolom sesuai kebutuhan pekerjaan. - Periksa kembali data sebelum
  menyimpan melalui formulir. - Gunakan tampilan Kanban untuk memantau progres
  kerja. - Manfaatkan dasbor sebagai pusat monitoring.
</Tips>

<Kendala>
  ## **Penanganan Kendala** Jika mengalami kendala saat menggunakan antarmuka
  NoBox.AI, lakukan langkah-langkah berikut: - Refresh halaman sistem jika data
  tidak muncul dengan sempurna. - Pastikan koneksi internet dalam kondisi stabil
  dan lancar. - Periksa kembali filter dan kolom pencarian yang sedang aktif. -
  Hapus cache browser jika terdapat elemen tampilan yang tidak muncul. - Logout
  dan login kembali jika fitur sistem tidak merespons. - Hubungi tim support
  jika kendala teknis masih ditemukan.
</Kendala>
