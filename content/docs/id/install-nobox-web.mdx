Halaman ini berisi panduan lengkap untuk melakukan instalasi dan konfigurasi NoBox Web, termasuk pengaturan NoBox Chat Manager dan NoBox Chat Session.

Panduan ini disusun untuk membantu pengguna memahami setiap tahapan instalasi, konfigurasi, hingga penggunaan sistem secara optimal.

Dengan mengikuti panduan ini, Anda dapat memastikan sistem berjalan dengan optimal dan siap digunakan untuk mendukung aktivitas bisnis.

---

## **Persiapan Sebelum Instalasi**

Sebelum memulai proses instalasi, pastikan hal-hal berikut telah dipenuhi agar proses berjalan lancar:

* Akun NoBox.AI sudah terdaftar dan aktif
* Paket berlangganan masih berlaku
* Perangkat terhubung ke internet
* Ruang penyimpanan mencukupi
* Data login akun tersedia
* Hak akses administrator tersedia jika diperlukan

---

## **Mengunduh Installer NoBox**

Bagian ini menjelaskan proses pengunduhan installer dari website resmi NoBox.

### Langkah-Langkah Mengunduh Installer

1. Kunjungi website resmi NoBox dan login menggunakan akun Anda.
2. Masuk ke menu **Berlangganan**.
3. Pilih paket layanan yang aktif.
4. Unduh installer sesuai dengan sistem operasi komputer Anda.

## **Proses Instalasi Aplikasi**

Setelah file installer berhasil diunduh, lakukan proses instalasi sesuai panduan berikut.

### Langkah-Langkah Instalasi

1. Jalankan file installer yang telah diunduh.
2. Pilih server yang akan digunakan.
3. Masukkan username dan password akun NoBox.AI.
4. Ikuti petunjuk instalasi yang muncul di layar.
5. Tentukan lokasi instalasi sesuai kebutuhan.

Untuk memastikan instalasi berhasil:

## **Struktur Folder Instalasi**

Jika instalasi berhasil, maka akan terbentuk dua folder berikut:

* **NoboxChatManager**
  Digunakan untuk mengelola integrasi chat, agent, dan automation.

* **NoboxChatSession**
  Digunakan untuk mengelola sesi login dan autentikasi akun.

Pastikan kedua folder ini tersedia agar aplikasi dapat berjalan normal.

### Fungsi Folder Sistem

Folder-folder ini berfungsi untuk:

* Menyimpan konfigurasi sistem.
* Menyimpan data sesi.
* Menjaga kestabilan aplikasi.
* Mendukung proses integrasi automation.

## **NoBox Chat Manager**

NoBox Chat Manager digunakan untuk mengintegrasikan chat web, media sosial, dan pesan instan dalam satu sistem terpusat.

Aplikasi ini mempermudah pengelolaan interaksi pelanggan dan prospek bisnis.

### Fungsi Utama Chat Manager

Beberapa fungsi utama Chat Manager antara lain:

* Mengelola agent customer service.
* Menghubungkan akun automation.
* Memantau aktivitas chat.
* Mengelola pesan masuk dan keluar.

### Cara Menjalankan NoBox Chat Manager

1. Buka folder instalasi NoBox.
2. Masuk ke folder **NoboxChatManager**.
3. Buka folder **Bin**.
4. Jalankan file **NoboxChatManager.exe**.

## **NoBox Chat Session**

NoBox Chat Session digunakan untuk mengelola proses login berbagai platform, seperti WhatsApp, Telegram, Shopee, Tokopedia, dan lainnya.

Aplikasi ini berfungsi untuk menjaga kestabilan sesi akun agar tetap aktif.

### Fungsi Utama Chat Session

Fungsi utama Chat Session meliputi:

* Menjaga sesi login tetap aktif.
* Menghindari logout otomatis.
* Menstabilkan koneksi akun.
* Mengelola autentikasi platform.

### Cara Menjalankan NoBox Chat Session

1. Buka folder instalasi NoBox.
2. Masuk ke folder **NoboxChatSession**.
3. Buka folder **Bin**.
4. Jalankan file **NoboxChatSession.exe**.

## **Mengatasi Kendala Instalasi**

Jika mengalami kendala selama instalasi atau penggunaan, lakukan langkah berikut:

**Pemeriksaan Dasar**

* Periksa koneksi internet
* Pastikan antivirus tidak memblokir aplikasi
* Pastikan server yang dipilih benar

**Pemeriksaan Sistem**

* Jalankan aplikasi sebagai administrator
* Periksa kapasitas penyimpanan
* Pastikan sistem operasi kompatibel

**Solusi Tambahan**

* Restart komputer
* Instal ulang aplikasi
* Perbarui versi aplikasi
* Hubungi tim support jika error berulang
