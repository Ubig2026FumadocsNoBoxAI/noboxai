## **Persiapan Sebelum Login**

Sebelum melakukan login, pastikan hal-hal berikut telah terpenuhi agar proses masuk ke sistem tidak mengalami kendala:

- Akun NoBox.AI sudah terdaftar dan terverifikasi melalui email
- Username dan password tersedia
- Perangkat terhubung dengan koneksi internet stabil
- Aplikasi NoBox Desktop sudah diunduh
- Sistem operasi komputer sesuai dengan spesifikasi aplikasi

---

## **Mengunduh Aplikasi NoBox Desktop**

Bagian ini menjelaskan cara mengunduh aplikasi NoBox Desktop dari website resmi.

### Langkah-Langkah Mengunduh Aplikasi

1. Kunjungi website resmi NoBox.AI dan login menggunakan akun Anda.
2. Setelah berhasil masuk, pilih menu **Berlangganan**.
3. Pilih paket layanan yang aktif.
4. Unduh installer sesuai dengan sistem operasi komputer Anda.

Pastikan file installer berhasil diunduh sebelum melanjutkan ke tahap berikutnya.

### Catatan Penting

- Gunakan koneksi internet yang stabil saat mengunduh file.
- Pastikan ruang penyimpanan mencukupi.
- Simpan file installer di lokasi yang mudah ditemukan.

---

## **Proses Instalasi Aplikasi**

Setelah file installer berhasil diunduh, lakukan proses instalasi sesuai panduan berikut.

### Langkah-Langkah Instalasi

1. Jalankan file installer yang telah diunduh.
2. Ikuti petunjuk instalasi yang muncul di layar.
3. Pilih lokasi instalasi sesuai kebutuhan.
4. Tunggu hingga proses instalasi selesai.

Setelah instalasi berhasil, aplikasi NoBox Desktop siap digunakan.

### Verifikasi Instalasi

Untuk memastikan instalasi berhasil:

- Periksa ikon aplikasi di desktop atau menu start.
- Jalankan aplikasi untuk memastikan tidak terjadi error.
- Pastikan folder sistem telah terbentuk.

---

## **Login ke Aplikasi NoBox.AI**

Bagian ini ditujukan bagi pengguna yang sudah memiliki akun terdaftar dan ingin masuk ke sistem NoBox.AI untuk mengakses layanan dan fitur yang tersedia.

### Langkah-Langkah Login ke Aplikasi

1. Buka aplikasi NoBox Desktop.
2. Pilih server yang ingin digunakan.
3. Masukkan username dan password yang sesuai dengan akun Anda.
4. Klik tombol **Login**.
5. Tunggu proses autentikasi dan sinkronisasi data.

Jika login berhasil, Anda akan diarahkan ke halaman utama aplikasi.

### Proses Setelah Login

Setelah berhasil login, sistem akan:

- Memuat data akun pengguna.
- Menyesuaikan tampilan sesuai hak akses.
- Menyinkronkan data dengan server.
- Menampilkan dashboard utama.

---

## **Struktur Folder Setelah Instalasi**

Setelah instalasi selesai, sistem akan membuat dua folder utama sebagai bagian dari konfigurasi aplikasi.

### Folder Sistem Utama

- **NoboxChatManager**
  Digunakan untuk mengelola sistem chat, agen, dan konfigurasi layanan.

- **NoboxChatSession**
  Digunakan untuk menyimpan data sesi, log aktivitas, dan riwayat komunikasi.

Pastikan kedua folder tersebut tersedia agar aplikasi dapat berjalan normal.

### Fungsi Folder

Keberadaan folder ini penting untuk:

- Menyimpan data sementara.
- Menjaga kestabilan sistem.
- Menyimpan konfigurasi pengguna.
- Menyimpan riwayat aktivitas.

<Callout type="warn" title="Pertanyaan Umum (FAQ)">
  - **Tidak bisa login meskipun data benar**
  Pastikan tidak ada kesalahan penulisan huruf besar dan kecil pada username dan password.
  - **Aplikasi tidak bisa dibuka**
  Coba instal ulang aplikasi dan pastikan sistem operasi kompatibel.
  - **Lupa password**
  Gunakan fitur lupa password pada website resmi NoBox.AI untuk melakukan reset.
</Callout>

<Tips>
## **Tips Penggunaan**

Jika mengalami masalah saat login, perhatikan hal-hal berikut: 
  **Pemeriksaan Akun**
  - Periksa kembali username dan password *
  - Pastikan akun masih aktif dan berlangganan * Pastikan email telah diverifikasi
  **Pemeriksaan Sistem** 
  - Pastikan koneksi internet stabil 
  - Gunakan versi aplikasi terbaru 
  - Tutup dan buka kembali aplikasi jika terjadi error
  - Restart komputer jika diperlukan 
  **Solusi Tambahan** 
  - Coba login kembali setelah beberapa menit 
  - Periksa status server NoBox.AI 
  - Pastikan tidak ada firewall yang memblokir aplikasi 
  - Jika masalah masih terjadi, silakan hubungi tim support.

</Tips>

## **Penanganan Kendala**

Jika mengalami kendala saat melakukan login, lakukan
  langkah-langkah berikut:

  Anda masukkan.

- Refresh
  halaman login atau gunakan mode samaran (incognito).
- Periksa email
  verifikasi di folder inbox atau spam.
- Pastikan akun Anda masih aktif dan
  memiliki hak akses.
- Hubungi administrator jika Anda lupa kata sandi atau
  akun terkunci.
