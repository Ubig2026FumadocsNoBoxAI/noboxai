Menu ini berperan penting dalam proses manajemen inventori, pengiriman, serta pengaturan distribusi barang ke pelanggan.

## **Tambah Form Warehouse**

1. Klik menu **[Form Warehouse]** ➔ **[Tambah Form Warehouse]**.
2. Maka Anda akan melihat dialog **[Tambah Form Warehouse]** untuk melengkapi data gudang.

## **Edit dan Hapus Data Form Warehouse**

### Edit Data Form Warehouse

Untuk mengedit suatu data warehouse, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

### Hapus Data Form Warehouse

Untuk menghapus suatu data warehouse adalah dengan pilih data warehouse yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.

<Callout type="info" title="Informasi Penting">
Penghapusan data warehouse bersifat permanen dan tidak dapat dibatalkan. Pastikan data yang akan dihapus benar-benar tidak diperlukan lagi.
</Callout>

## **Tips Penggunaan**
 
Jika mengalami kendala dalam penggunaan menu Form Warehouse, lakukan langkah berikut:

- Gunakan nama gudang yang mudah dikenali dan sesuai dengan lokasi aslinya.
- Pastikan data alamat dan kode pos diisi dengan benar untuk mendukung proses pengiriman.
- Cantumkan nomor telepon aktif agar tim operasional atau kurir dapat menghubungi pihak gudang.
- Perbarui data gudang jika terjadi perubahan alamat atau kontak.
- Hapus data gudang yang sudah tidak digunakan agar data tetap rapi dan relevan.

</Tips>
