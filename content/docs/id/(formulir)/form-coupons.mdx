## **Tambah Coupons Formulir**

Untuk membuat data model formulir silahkan klik menu **[Form Coupons] ➔ [Tambah Form Coupons]**. Maka akan tampil dialog **[Tambah Form Coupons]**.

<ImageZoom
  src="/images/id/Tambah%20Form%20Coupons.png"
  alt="Tampilan Dasbor NoBox"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

Lengkapi isian yang diperlukan, sebagai contoh seperti berikut :

| **Isian**         | Uraian               | **Contoh**                |
| ----------------- | -------------------- | ------------------------- |
| Nama              | Nama diskon formulir | Diskon Tahun Baru         |
| Code              | Kode diskon          | TahunBaru2026             |
| Status            | Status               | Aktif                     |
| Disc For          | Diskon untuk         | Shipping                  |
| Disc Type         | Tipe diskon          | Percentage                |
| Disc Value        | Jumlah diskon        | 100%                      |
| Has Zone          | Khusus daerah        |                           |
| Origin Type       | Daerah asal          | State                     |
| Origin State      | Provinsi asal        | Jawa Timur                |
| Destination Type  | Daerah tujuan        | State                     |
| Destination State | Provinsi tujuan      | Jawa Timur                |
| Max Use           | Batas penggunaan     | 100                       |
| Products          | Nama produk          | Pashmina Voal             |
| Payments          | Metode pembayaran    | Bank Central Asia, Paypal |
| New Customer      | Pelanggan baru       |                           |

Jika Anda sudah melengkapi isian yang ada, klik **[Simpan]** untuk menyimpan data model formulir yang baru saja Anda buat.

## **Edit Dan Hapus Warehouse Formulir**

Untuk mengedit suatu data model formulir, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Untuk menghapus suatu data model formulir adalah dengan pilih data mana yang akan dihapus, kemudian klik **[Hapus],** maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut. Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.
