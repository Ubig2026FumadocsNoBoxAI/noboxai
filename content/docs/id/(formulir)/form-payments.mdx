Pada halaman ini, pengguna dapat melihat nama metode pembayaran, status aktif/nonaktif, instruksi pembayaran, QR Code (jika ada), serta melakukan proses tambah, edit, dan hapus metode pembayaran.

## **Tambah Metode Pembayaran**

Untuk membuat data metode pembayaran baru, silahkan klik menu:

**[Metode Pembayaran] ➔ [Tambah Metode Pembayaran]**  

Maka akan tampil dialog **[Tambah Metode Pembayaran]**, kemudian lengkapi isian-isian yang terdapat pada dialog tersebut.

### **Isian dan Penjelasannya**

| **Isian** | **Uraian** | **Contoh** |
| ---------- | --------- | ---------- |
| Tipe       | Metode pembayaran | Bank Transfer |
| Nama       | Nama pembayaran | Bank Mandiri |
| Aktif      | Pilih status metode pembayaran | Aktif |
| Catatan    | Tambahan keterangan | - |
| Instruksi  | Instruksi cara bayar | Pembayaran ditransfer ke rekening Bank Mandiri xxxx.xxxx.xxxx atas nama Jasmine Aulya. Setelah transfer silahkan kirimkan buktinya ke WhatsApp di 62xxxxxxxxxx. |
| QR Code    | Upload QR Code pembayaran | - |

Setelah semua isian lengkap, klik **[Simpan]** untuk menyimpan data metode pembayaran baru.

## **Edit dan Hapus Data Pembayaran Formulir**

### Edit Data Pembayaran

Untuk mengedit suatu data pembayaran, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

### Hapus Data Pembayaran

Untuk menghapus suatu data pembayaran adalah dengan pilih data pembayaran yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.

## **Cara Menggunakan Metode Pembayaran**

Setelah data metode pembayaran tersimpan, metode ini bisa digunakan di form atau transaksi yang relevan.

1. Pilih salah satu model formulir.  
2. Pilih metode pembayaran yang tersedia dari daftar.  
3. Klik **[Simpan]**.

## **Tips Penggunaan Metode Pembayaran**

- Pastikan metode pembayaran yang diaktifkan sesuai dengan metode yang benar-benar tersedia.  
- Gunakan instruksi pembayaran yang jelas agar pelanggan tidak bingung.  
- Lengkapi QR Code jika ingin mempermudah pelanggan melakukan scan pembayaran.  
- Periksa kembali data pembayaran sebelum disimpan agar tidak terjadi kesalahan.

---

## **Penanganan Kendala**

Jika terdapat kendala atau masalah terkait metode pembayaran, langkah yang dapat dilakukan:

- Pastikan data metode pembayaran sudah disimpan dengan status **Aktif**.  
- Periksa kembali kolom **Instruksi** agar instruksi pembayaran terlihat pada formulir.  
- Jika QR Code tidak muncul, pastikan file sudah diunggah dengan benar.  
- Refresh halaman atau logout-login kembali jika perubahan tidak langsung terlihat.
