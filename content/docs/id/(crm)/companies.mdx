## **Tambah Perusahaan**

### Akses Menu Tambah Perusahaan

Untuk membuat data perusahaan, silahkan Anda klik modul **[CRM] ➔** klik menu **[Perusahaan] ➔ [Perusahaan] ➔ [Tambah Perusahaan]**.

Maka akan tampil dialog **[Tambah Perusahaan]** kemudian lengkapi isian-isian yang terdapat dalam dialog tersebut.

## **Edit dan Hapus Data Perusahaan**

### Edit Data Perusahaan

Untuk mengedit suatu data perusahaan, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

### Hapus Data Perusahaan

Untuk menghapus suatu data perusahaan adalah dengan pilih data perusahaan yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.

## **Tips Penggunaan Menu Perusahaan**

Beberapa tips untuk memaksimalkan penggunaan menu Perusahaan:

* Pastikan data perusahaan diisi secara lengkap dan akurat.
* Gunakan nama perusahaan sesuai dengan dokumen resmi.
* Perbarui informasi kontak secara berkala.
* Tambahkan catatan untuk mencatat hasil komunikasi dengan perusahaan.
* Gunakan fitur tugas untuk mengatur aktivitas tindak lanjut.
* Manfaatkan log aktivitas untuk memantau histori interaksi.
* Lampirkan dokumen pendukung jika diperlukan.
* Kelola data perusahaan secara rutin agar tetap terorganisir.

---

## **Penanganan Kendala**

Jika terjadi kendala dalam penggunaan menu Perusahaan, lakukan langkah berikut:

* Pastikan koneksi internet dalam kondisi stabil.
* Refresh halaman jika data tidak muncul.
* Periksa kembali hak akses akun Anda.
* Pastikan semua kolom wajib telah diisi saat menyimpan data.
* Coba logout dan login kembali jika fitur tidak berfungsi.
* Bersihkan cache browser jika tampilan bermasalah.
* Pastikan browser menggunakan versi terbaru.
* Hubungi tim support jika kendala masih berlanjut.
