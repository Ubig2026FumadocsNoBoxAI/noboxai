## **Manajemen Papan Tugas**

### Tambah Papan

Untuk tambah data papan, silahkan klik tombol [ <img src="/images/id/Tombol%20Papan.png" alt="Tambah Papan" className="inline h-5 w-auto mx-1" /> **],** maka akan tampil dialog **[Tambah Papan]\*\*.

### Tampilan Dialog Tambah Papan

<ImageZoom
  src="/images/id/Tampilan%20Tambah%20Papan.png"
  alt="Form Tambah Papan"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

### Isian Dialog Tambah Papan

Lengkapi isian yang diperlukan, sebagai contoh seperti berikut:

| Isian     | Uraian          | Contoh                                        |
| --------- | --------------- | --------------------------------------------- |
| Nama      | Nama papan      | Makmur Dev                                    |
| Deskripsi | Deskripsi papan | Informasi tugas tim developer yang dikerjakan |
| List      | Tahapan tugas   | Baru, Dikerjakan, Siap Tes, Selesai           |

Jika Anda sudah melengkapi isian yang ada, klik **[Simpan]** untuk menyimpan data papan yang baru saja Anda buat.

---

### Edit dan Hapus Data Papan

Untuk mengedit suatu data papan, silahkan pilih terlebih dahulu data mana yang akan diedit.

Klik icon [ <img src="/images/id/Edit%20Delete2.png" alt="Edit Delete" className="inline h-5 w-auto mx-1" /> ] maka akan tampil pop up, pilih tombol **[Edit]**, lalu editlah data Anda dan klik **[Simpan]**.

Untuk menghapus data papan, klik icon yang sama, pilih **[Hapus]**, lalu konfirmasi dengan klik **[Ya]**.

Jika Anda sudah melengkapi isian yang ada, klik **[Simpan]** untuk menyimpan data tugas.

### **Edit dan Hapus Data Tugas**

**Edit Data Tugas**

Untuk mengedit suatu data tugas, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

**Hapus Data Tugas**

Untuk menghapus suatu data tugas adalah dengan pilih data tugas yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.

## **Manajemen Komentar**

Komentar merupakan salah satu fitur Nobox.Ai yang digunakan untuk memberikan pandangan, tanggapan, atau feedback terhadap tugas tersebut.

### Tambah Komentar

Untuk tambah komentar silahkan klik icon [ <img src="/images/id/Comment.png" alt="Komentar" className="inline h-5 w-auto mx-1" /> ] maka akan tampil dialog komentar.

Isi komentar dan klik tombol **[Send]** untuk mengirimkan komentar tersebut.

## **Tips Penggunaan Menu Tugas**

Agar penggunaan menu Tugas lebih efektif dan terorganisir, perhatikan beberapa tips berikut:

- Gunakan nama tugas yang jelas dan mudah dipahami.
- Lengkapi detail dan rencana tugas untuk menghindari kesalahan komunikasi.
- Tentukan prioritas tugas sesuai tingkat kepentingannya.
- Gunakan fitur tag untuk mempermudah pencarian tugas.
- Perbarui status tugas secara berkala sesuai progres pekerjaan.
- Manfaatkan fitur komentar untuk koordinasi antar anggota tim.
- Gunakan fitur salin tugas untuk mempercepat pembuatan tugas serupa.
- Pastikan tanggal mulai dan selesai diisi dengan benar.

Dengan menerapkan tips di atas, pengelolaan tugas akan menjadi lebih rapi dan efisien.

## **Penanganan Kendala**

Jika mengalami kendala dalam penggunaan menu Tugas, lakukan langkah-langkah berikut:

- Refresh halaman apabila tampilan tidak muncul dengan sempurna.
- Pastikan koneksi internet dalam kondisi stabil.
- Periksa kembali data yang diinput sebelum menyimpan.
- Pastikan akun memiliki hak akses yang sesuai.
- Logout dan login kembali jika terjadi error sistem.
- Hapus cache browser jika sistem terasa lambat.
- Coba akses kembali melalui browser lain jika diperlukan.
- Hubungi tim support apabila kendala masih berlanjut.

Langkah-langkah di atas dapat membantu mengatasi permasalahan umum saat menggunakan fitur Tugas.
