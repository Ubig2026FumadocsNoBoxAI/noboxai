Target Perusahaan merupakan salah satu menu pada Nobox.Ai yang digunakan untuk menentukan suatu target pemasaran dalam salah satu perusahaan disetiap bulannya.

Menu ini membantu perusahaan dalam menetapkan perencanaan pemasaran, memantau pencapaian penjualan, serta mengevaluasi performa bisnis secara berkala.

## **Tambah Data Target Perusahaan**

Untuk membuat data target perusahaan, silahkan klik modul **[CRM]**, klik menu **[Perusahaan]**, lalu klik **[Target Perusahaan] ➔ [Tambah Target Perusahaan]**.

Maka akan tampil dialog **[Tambah Target Perusahaan]**, kemudian lengkapi isian-isian yang terdapat pada dialog tersebut sesuai kebutuhan perusahaan.

### Tampilan Dialog Tambah Target

Form ini digunakan untuk memasukkan data target pemasaran perusahaan secara detail.

### Isian Dialog Tambah Target

| Isian              | Uraian               | Contoh       |
| ------------------ | -------------------- | ------------ |
| Perusahaan         | Pilih perusahaan     | PT SEJAHTERA |
| Tahun              | Pilih tahun target   | 2024         |
| Mata Uang          | Pilih mata uang      | Rp           |
| Januari - Desember | Isi target pemasaran | 1.200.000    |

Setiap isian harus diisi dengan benar agar sistem dapat menghitung dan menampilkan data pencapaian secara akurat.

Pastikan nominal target telah disesuaikan dengan kondisi dan kapasitas perusahaan.

Jika Anda sudah melengkapi isian-isian yang ada, klik **[Simpan]** untuk menyimpan data target perusahaan yang baru saja Anda buat.

## **Edit dan Hapus Data Target Perusahaan**

### Edit Data Target Perusahaan

Untuk mengedit suatu data target perusahaan, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

### Hapus Data Target Perusahaan

Untuk menghapus suatu data target perusahaan adalah dengan pilih data target perusahaan yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.

<Callout type="info" title="Informasi Penting">
Penghapusan data target perusahaan bersifat permanen dan tidak dapat dibatalkan. Pastikan data yang akan dihapus benar-benar tidak diperlukan lagi.
</Callout>

## **Pendapatan Saat Ini**

### Pengertian Pendapatan Saat Ini

Pendapatan Saat Ini merupakan indikator yang menunjukkan jumlah pendapatan yang telah dicapai berdasarkan data penawaran.

Pendapatan saat ini akan ditampilkan ketika penawaran yang terkait dengan perusahaan tersebut sudah berstatus **[Won]**.

### Contoh Perhitungan Pendapatan

Contoh:  
Pada penawaran PT SEJAHTERA harga penawaran sebesar 18.000.000 sudah berstatus **Won**, maka pada target perusahaan nilai target akan tampil data sebesar 18.000.000.

Dengan fitur ini, pengguna dapat membandingkan antara target dan realisasi secara lebih mudah.

## **Tips Penggunaan**
 
Agar pengelolaan target perusahaan berjalan optimal, perhatikan beberapa tips berikut:
 
- Tentukan target berdasarkan data historis perusahaan.
- Lakukan evaluasi pencapaian setiap bulan.
- Sesuaikan target dengan kondisi pasar.
- Perbarui data secara berkala.
- Gunakan laporan sebagai bahan evaluasi manajemen.
- Libatkan tim dalam penentuan target.
 
</Tips>
