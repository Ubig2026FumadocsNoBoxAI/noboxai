Tag Perusahaan adalah salah satu menu pada NoBox.Ai yang digunakan untuk mengelompokkan data produk Anda.

Menu ini membantu pengguna dalam mengelola dan mengorganisir data perusahaan agar lebih terstruktur di dalam sistem.

## **Tambah Data Tag**

Untuk membuat data tag, silahkan klik menu **[Tag Perusahaan] ➔ [Tambah Perusahaan Tag]**.

Maka akan tampil dialog **[Tambah Perusahaan Tag]** kemudian lengkapi nama tag yang terdapat pada dialog tersebut.

### Tampilan Dialog Tambah Tag

<ImageZoom
  src="/images/id/Tag%20Perusahaan.PNG"

Form ini digunakan untuk memasukkan nama tag baru yang akan digunakan pada data perusahaan.

Jika Anda sudah melengkapi nama tag, klik **[Simpan]** untuk menyimpan data tag yang baru saja Anda buat.

Pastikan nama tag sudah sesuai dengan kategori perusahaan agar mudah digunakan kembali.

## **Edit dan Hapus Data Tag**

### Edit Data Tag

Untuk mengedit suatu data tag, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

### Hapus Data Tag

Untuk menghapus suatu data tag adalah dengan pilih data tag yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.

---

## **Cara Penggunaan Tag Perusahaan**

Berikut langkah-langkah penggunaan Tag Perusahaan pada data perusahaan:

1. Buka menu **[Perusahaan]**, pilih salah satu data yang sudah dibuat atau tambahkan produk baru.

2. Klik pada bagian isian di kolom Tag, maka akan muncul tag yang sudah dibuat sebelumnya.

### Tampilan Pilih Tag Perusahaan

3. Pilih satu atau lebih dari data tag tersebut dan klik **[Simpan]** untuk menyimpan kembali data produk.

Dengan menggunakan tag, data perusahaan akan lebih mudah dikategorikan sesuai kebutuhan.

<Tips>
## **Tips Penggunaan**
 
Untuk hasil yang optimal, perhatikan beberapa tips berikut:
 
- Gunakan nama tag yang jelas dan mudah dipahami.
- Hindari penggunaan tag yang terlalu banyak.
- Gunakan tag sesuai kategori perusahaan.
- Lakukan pengecekan tag secara berkala.
- Hapus tag yang sudah tidak digunakan.
 
</Tips>
