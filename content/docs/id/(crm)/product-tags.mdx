Tag Produk adalah salah satu menu pada NoBox.Ai yang digunakan untuk mengelompokkan data produk Anda.

Menu ini membantu pengguna dalam mengelola dan mengklasifikasikan produk berdasarkan kategori tertentu, sehingga memudahkan proses pencarian, filter data, serta pengelolaan produk secara lebih terstruktur dan efisien dalam sistem.

---

## **Tambah Data Tag**

Untuk membuat data tag, silahkan klik menu **[Tag Produk] ➔ \[Tambah Produk Tag]**. Maka akan tampil dialog **[Tambah Produk Tag]** kemudian lengkapi nama tag yang terdapat pada dialog tersebut.

### Tampilan Dialog Tambah Tag

<ImageZoom
  src="/images/id/Tambah%20Tag%20Produk.png"

Form ini digunakan untuk menambahkan data tag baru ke dalam sistem.

Jika Anda sudah melengkapi nama tag, klik **[Simpan]** untuk menyimpan data tag yang baru saja Anda buat.

## **Edit dan Hapus Data Tag**

### Edit Data Tag

Untuk mengedit suatu data tag, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

### Hapus Data Tag

Untuk menghapus suatu data tag adalah dengan pilih data tag yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.

---

## **Cara Penggunaan Tag Produk**

### Penerapan Tag pada Produk

1. Buka menu **[Produk]**, pilih salah satu data yang sudah dibuat atau tambahkan produk baru.

2. Klik pada bagian isian di kolom Tag, maka akan muncul tag yang sudah dibuat sebelumnya.

### Tampilan Pilih Tag Produk

3. Pilih satu atau lebih dari data tag tersebut dan klik **[Simpan]** untuk menyimpan kembali data produk.

---

## **Filter Tag Produk**

### Fungsi Filter Tag

Filter produk tag digunakan untuk memilah data produk dan menampilkan data berdasarkan tag yang dipilih.

Fitur ini memudahkan pengguna dalam menemukan produk dengan cepat sesuai kategori yang diinginkan.

### Cara Menggunakan Filter Tag

Buka menu **[Produk]**, klik filter Tag dan pilihlah data tag mana yang ingin Anda cari. Maka akan tampil data yang telah terfilter.

## **Tips Penggunaan**
 
Agar penggunaan tag lebih optimal, perhatikan hal berikut:
 
- Gunakan nama tag yang jelas.
- Hindari duplikasi tag.
- Kelompokkan tag sesuai kategori.
- Perbarui tag secara berkala.
- Gunakan tag secara konsisten.
 
</Tips>
