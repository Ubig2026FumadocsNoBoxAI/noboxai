## **Tambah Papan**

### Akses Menu Tambah Papan

Untuk tambah data papan, silahkan klik tombol  [ <img src="/images/id/Tombol%20Papan%20(1).png" alt="Tambah Papan" className="inline h-5 w-auto mx-1" /> ],  maka akan tampil dialog **[Add Pipeline]**.

<ImageZoom
  src="/images/id/tambah%20papan%20penawaran.png"
  alt="Form Tambah Papan Penawaran"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

### Isian Form Tambah Papan

| Isian     | Uraian          | Contoh                                                 |
| --------- | --------------- | ------------------------------------------------------ |
| Nama      | Nama papan      | Marketing                                              |
| Deskripsi | Deskripsi papan | Informasi manajemen penjualan                          |
| Stage     | Tahap manajemen | New, Qualified, Advance, Payment In Process, Won, Lost |

Klik **[Simpan]** setelah selesai.

## **Edit Dan Hapus Data Papan**

### Edit Data Papan

Gunakan icon  [ <img src="/images/id/titik%20tiga.png" alt="Opsi" className="inline h-5 w-auto mx-1" /> ] → **Edit** untuk mengubah data.

### Hapus Data Papan

Gunakan icon yang sama → **Hapus** untuk menghapus data.

Sistem akan menampilkan konfirmasi sebelum data dihapus.

## **Tips Penggunaan Tag Tugas**

Untuk penggunaan yang lebih optimal, perhatikan beberapa tips berikut:

- Gunakan nama tag yang konsisten.
- Hindari penggunaan singkatan yang membingungkan.
- Kelompokkan tugas dengan tag sesuai kategori.
- Perbarui tag secara berkala jika ada perubahan kebutuhan.
- Hapus tag yang sudah tidak digunakan.

Dengan penerapan tips ini, pengelolaan tugas akan menjadi lebih rapi dan terstruktur.

## **Penanganan Kendala**

Jika mengalami kendala dalam penggunaan menu Tag Tugas, lakukan langkah-langkah berikut:

- Refresh halaman sistem.
- Periksa koneksi internet.
- Pastikan Anda memiliki hak akses yang sesuai.
- Logout lalu login kembali ke sistem.
- Periksa kembali data tag yang dimasukkan.
- Hubungi tim support jika kendala masih terjadi.
