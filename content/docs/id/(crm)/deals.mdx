Penawaran merupakan salah satu menu pada NoBox.AI yang digunakan untuk melakukan manajemen penawaran, sehingga mempermudah Anda dalam mengelola seluruh informasi dan aktivitas progress penjualan.

Menu ini memungkinkan pengguna untuk membuat, melacak, dan mengatur penawaran kepada pelanggan, memantau statusnya, serta mempermudah proses tindak lanjut hingga penutupan penjualan.

1. Untuk tambah data papan, silahkan klik tombol [ <img src="/images/id/Tombol%20Papan%20(1).png" alt="Tambah Papan" className="inline h-5 w-auto mx-1" /> ].
2. Maka akan tampil dialog **[Add Pipeline]**. Isi Nama, Deskripsi, dan Stage penawaran Anda.

### Tampilan Dialog Tambah Papan Penawaran

<ImageZoom
  src="/images/id/tambah%20papan%20penawaran.png"

### Isian Tambah Papan

| Isian     | Uraian          | Contoh                                                 |
| --------- | --------------- | ------------------------------------------------------ |
| Nama      | Nama papan      | Marketing                                              |
| Deskripsi | Deskripsi papan | Informasi manajemen penjualan                          |
| Stage     | Tahap manajemen | New, Qualified, Advance, Payment In Process, Won, Lost |

Klik **[Simpan]** setelah selesai.

### Edit Data Papan

Untuk mengedit suatu data papan, gunakan icon [ <img src="/images/id/titik%20tiga.png" alt="Opsi" className="inline h-5 w-auto mx-1" /> ] → **Edit** untuk mengubah data, kemudian klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

### Hapus Data Papan

Untuk menghapus suatu data papan, gunakan icon yang sama → **Hapus**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.

<Callout type="info" title="Informasi Penting">
  Penghapusan data papan bersifat permanen dan tidak dapat dibatalkan. Pastikan
  data yang akan dihapus benar-benar tidak diperlukan lagi.
</Callout>

## **Tips Penggunaan**
Agar penggunaan fitur Penawaran lebih optimal, perhatikan beberapa tips berikut:

**Pengelolaan Data**

- Gunakan nama penawaran yang jelas dan mudah dikenali.
- Lengkapi seluruh data pada form penawaran untuk memudahkan monitoring.
- Perbarui status tahapan penawaran secara berkala.
- Gunakan fitur komentar untuk mencatat hasil diskusi atau keputusan.

**Manajemen Aktivitas**
- Tambahkan tugas untuk setiap penawaran agar progres lebih terkontrol.
- Tentukan prioritas pada penawaran penting.
- Gunakan fitur transaksi dan pengeluaran untuk pencatatan keuangan.
- Manfaatkan log aktivitas untuk memantau perubahan data.
</Tips>

## **Penanganan Kendala**
Jika mengalami kendala saat menggunakan menu Penawaran, perhatikan hal-hal berikut:
- Pastikan akun memiliki hak akses yang sesuai.
- Periksa apakah akun masih aktif dan tidak dibatasi.
- Pastikan sudah login dengan akun yang benar.
- Pastikan koneksi internet stabil.
- Gunakan browser versi terbaru.
- Refresh halaman jika data tidak muncul.
- Logout dan login kembali jika terjadi error.
- Restart perangkat jika sistem tidak merespons.
- Coba ulangi proses input data jika gagal tersimpan.
- Periksa apakah server NoBox.AI sedang maintenance.
- Pastikan tidak ada ekstensi browser yang mengganggu sistem.
- Hapus cache browser jika tampilan bermasalah.
</Kendala>
