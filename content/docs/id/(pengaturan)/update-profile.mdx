<Tips>
## **Tips Penggunaan**
 
Tips untuk memastikan profil akun dan perusahaan Anda tetap akurat:
 
- Gunakan foto profil yang jelas agar rekan tim Anda dapat mengenali akun Anda dengan mudah di sistem.
- Pastikan alamat email yang terdaftar adalah email aktif untuk menerima notifikasi penting terkait aktivitas akun.
- Selalu simpan MyNoBox Tenant Id Anda; Id ini sangat krusial untuk proses berlangganan dan aktivasi fitur.
- Pilih 'Default Language' yang paling sesuai dengan preferensi tim Anda untuk mempermudah navigasi.
- Perbarui informasi nomor telepon perusahaan secara berkala agar pelanggan selalu diarahkan ke kontak yang benar.
 
</Tips>
 
---
 
<Kendala>
## **Penanganan Kendala**
 
Apabila menemui masalah saat memperbarui profil:
 
- Jika tombol Update tidak merespons, pastikan semua field wajib pada tabel profil sudah terisi dengan benar.
- Apabila foto profil tidak terunggah (upload), periksa ukuran file dan pastikan dalam format gambar yang didukung (JPG/PNG).
- Jika Anda lupa Tenant Id, hubungi administrator atau cari Id tersebut pada email konfirmasi pendaftaran awal.
- Periksa kembali penulisan password baru jika sistem memberikan peringatan 'Password mismatch' saat proses pembaruan.
- Logout dan login kembali jika perubahan pada profil tidak langsung terlihat pada tampilan dashboard.
 
</Kendala>
