Perbarui Profil adalah salah satu menu pada NoBox.AI yang digunakan untuk memperbarui informasi akun pengguna (User Profile) dan data perusahaan (Company Details).

Menu ini membantu pengguna dalam mengelola dan memperbarui data pribadi, pengaturan preferensi akun, serta informasi resmi perusahaan yang terdaftar di dalam sistem. Dengan memastikan data profil selalu akurat dan terkini, pengguna dapat meningkatkan efisiensi koordinasi tim, kualitas komunikasi, serta kepatuhan administratif perusahaan.

## **User Profile**

User Profile berisi informasi data diri pengguna yang digunakan untuk mengidentifikasi akun di dalam sistem NoBox.AI.

Melalui halaman ini, pengguna dapat mengubah foto profil, memperbarui informasi pribadi, mengaktifkan notifikasi email, mengatur tampilan interface, serta mengganti password akun.

### Cara Memperbarui User Profile

Untuk memperbarui profil pengguna, ikuti langkah-langkah berikut:

1. Klik ikon user di pojok kanan atas halaman.
2. Klik menu **[Profile]** untuk membuka halaman pengaturan profil.
3. Editlah data yang ingin diubah sesuai kebutuhan.
4. Klik tombol **[Update]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan berlaku di seluruh sistem.

### Cara Memperbarui Company Details

Untuk memperbarui informasi perusahaan, ikuti langkah-langkah berikut:

1. Klik ikon user di pojok kanan atas halaman.
2. Klik menu **[Company Details]** untuk membuka halaman pengaturan perusahaan.
3. Edit data perusahaan yang ingin diubah sesuai kebutuhan.
4. Klik tombol **[Update]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan berlaku di seluruh sistem.

<Tips>
## **Tips Penggunaan**
 
Tips untuk memastikan profil akun dan perusahaan Anda tetap akurat:
 
- Gunakan foto profil yang jelas dan profesional agar rekan tim dapat mengenali akun Anda dengan mudah di sistem.
- Pastikan alamat email yang terdaftar adalah email aktif untuk menerima notifikasi penting terkait aktivitas akun.
- Selalu simpan **MyNoBox Tenant Id** Anda dengan aman; ID ini sangat krusial untuk proses berlangganan dan aktivasi fitur.
- Pilih **Default Language** yang paling sesuai dengan preferensi tim Anda untuk mempermudah navigasi.
- Perbarui informasi nomor telepon perusahaan secara berkala agar pelanggan selalu diarahkan ke kontak yang benar.
- Gunakan password yang kuat dan berbeda dari akun lainnya untuk menjaga keamanan akun.
 
</Tips>
 
---
 
<Kendala>
## **Penanganan Kendala**
 
Apabila menemui masalah saat memperbarui profil, lakukan langkah berikut:
 
- Jika tombol **[Update]** tidak merespons, pastikan semua field wajib pada formulir sudah terisi dengan benar.
- Apabila foto profil tidak terunggah, periksa ukuran file dan pastikan dalam format gambar yang didukung (JPG/PNG).
- Jika Anda lupa Tenant Id, hubungi administrator atau cari ID tersebut pada email konfirmasi pendaftaran awal.
- Periksa kembali penulisan password baru jika sistem memberikan peringatan 'Password mismatch' saat proses pembaruan.
- Pastikan koneksi internet stabil saat proses pembaruan data berlangsung.
- Logout dan login kembali jika perubahan pada profil tidak langsung terlihat pada tampilan dashboard.
- Bersihkan cache browser jika tampilan tidak sesuai.
- Hubungi tim support jika kendala masih berlanjut.
 
</Kendala>
