Inputan Khusus adalah salah satu menu pada NoBox.AI yang memungkinkan pengguna untuk menambahkan kolom data tambahan sesuai dengan kebutuhan bisnis di luar kolom bawaan sistem.

Menu ini membantu pengguna dalam menyesuaikan struktur data kontak atau produk agar lebih lengkap dan relevan dengan operasional perusahaan. Dengan fitur ini, setiap informasi penting yang spesifik terhadap bisnis Anda dapat dicatat dan dikelola secara terstruktur di dalam platform NoBox.AI.

## **Tambah Inputan Khusus**

1. Untuk membuat data inputan khusus baru, silahkan klik menu **[Inputan Khusus]**, kemudian pilih kategori **[Company]**, **[Kontak]** atau **[Produk]** sesuai kebutuhan.
2. Klik tombol **[Tambah Inputan Khusus]**, maka akan tampil dialog pengisian data inputan khusus.

## **Edit dan Hapus Data Inputan Khusus**

### Edit Data Inputan Khusus

Untuk mengedit suatu data inputan khusus, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan berlaku pada seluruh formulir yang menggunakan inputan khusus tersebut.

### Hapus Data Inputan Khusus

Untuk menghapus suatu data inputan khusus, silahkan pilih data inputan khusus yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]**. Jika tidak, klik **[Tidak]**.

<Callout type="info" title="Informasi Penting">
  Penghapusan inputan khusus akan menghapus kolom tersebut dari semua data
  kontak atau produk yang sudah menggunakannya. Pastikan data yang terkait sudah
  tidak diperlukan sebelum melakukan penghapusan.
</Callout>

## **Cara Penggunaan Inputan Khusus**

Fitur Inputan Khusus akan muncul secara otomatis pada formulir kontak atau produk setelah berhasil dibuat.

Untuk menggunakan inputan khusus, ikuti langkah-langkah berikut:

1. Buka menu **[Kontak]** atau **[Produk]**, pilih salah satu data yang sudah dibuat atau buat data baru.

2. Pada bagian formulir, kolom inputan khusus yang sebelumnya sudah dibuat akan tampil secara otomatis di bagian bawah formulir.

3. Isi data pada kolom inputan khusus sesuai kebutuhan, kemudian klik **[Simpan]** untuk menyimpan kembali data kontak atau produk beserta informasi tambahannya.

<Tips>
## **Tips Penggunaan**
Beberapa tips untuk mengoptimalkan penggunaan Inputan Khusus:
- Gunakan label yang singkat namun deskriptif agar mudah dipahami oleh pengguna lain.
- Manfaatkan fitur **[Wajib Diisi]** untuk data krusial yang harus selalu ada di setiap kontak atau produk.
- Pilih tipe input yang sesuai (misalnya 'Number' untuk nominal, 'Email' untuk alamat email) untuk menjaga validitas data.
- Atur urutan inputan khusus agar alur pengisian data terasa logis dan efisien.
- Gunakan placeholder sebagai petunjuk format data yang benar (contoh: 62xxxxxx untuk nomor telepon).
- Manfaatkan nilai bawaan untuk data yang umumnya memiliki nilai sama pada setiap entri.
</Tips>

<Kendala>
## **Penanganan Kendala**
Jika terjadi kendala pada menu Inputan Khusus, lakukan langkah berikut:
- Refresh halaman jika inputan khusus yang baru dibuat tidak langsung muncul di formulir kontak atau produk.
- Periksa kembali pengaturan tipe input jika data yang dimasukkan terus-menerus ditolak oleh sistem.
- Pastikan Anda memiliki hak akses yang cukup untuk menambah atau menghapus inputan khusus.
- Pastikan semua isian wajib sudah terisi sebelum menyimpan data.
- Logout dan login kembali jika fitur tidak berfungsi normal.
- Bersihkan cache browser jika tampilan tidak sesuai.
- Hubungi tim support jika penghapusan inputan khusus menyebabkan gangguan pada data yang sudah ada.
</Kendala>
