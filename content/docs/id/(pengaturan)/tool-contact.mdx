Tool Contact digunakan untuk mengelola data kontak pelanggan di Nobox.Ai.
Melalui menu ini, Anda dapat menambahkan, mengimpor, memfilter, dan mengelola kontak secara terpusat untuk kebutuhan CRM, pesan, dan promosi.

## Tambah Kontak

Gunakan fitur ini untuk menambahkan kontak baru secara manual.

Langkah-langkah:

1\. Klik [Tambah Kontak]

2\. Lengkapi informasi kontak

3\. Klik [Simpan]

## Field Kontak

| Bagian | Field           | Deskripsi                                                            |
| ------ | --------------- | -------------------------------------------------------------------- |
| Umum   | Nama            | Nama lengkap kontak dan digunakan sebagai identitas utama _(wajib)_  |
|        | Kategori        | Klasifikasi kontak untuk membantu pengelompokan data                 |
|        | List            | Daftar kontak yang digunakan untuk segmentasi dan kebutuhan kampanye |
|        | Tag             | Label khusus untuk memudahkan identifikasi dan penyaringan kontak    |
|        | Kontak Person   | Penanggung jawab atau perwakilan utama dari kontak                   |
|        | Catatan         | Informasi tambahan atau catatan penting terkait kontak               |
| Link   | Kanal           | Platform komunikasi eksternal yang terhubung dengan kontak           |
|        | Nama Eksternal  | Nama akun atau username pada platform yang dipilih                   |
|        | ID Eksternal    | Identitas unik kontak dari platform eksternal                        |
| Alamat | Alamat          | Alamat fisik lengkap dari kontak                                     |
|        | Negara          | Negara tempat kontak berada                                          |
|        | Provinsi        | Provinsi atau wilayah administratif kontak                           |
|        | Kota            | Kota atau kabupaten tempat kontak berada                             |
|        | Kode Pos        | Kode pos sebagai referensi lokasi                                    |
|        | Lintang / Bujur | Koordinat geografis untuk penentuan lokasi peta                      |
|        | Lokasi          | Nama area atau penanda lokasi                                        |
|        | Diblokir        | Status untuk menandai apakah kontak diblokir atau aktif              |

## Import Kontak

Tambahkan kontak secara massal melalui integrasi berikut:

| Metode             | Deskripsi                    |
| ------------------ | ---------------------------- |
| Excel & CSV        | Unggah file kontak           |
| Google Contacts    | Sinkronisasi kontak Google   |
| Facebook Audiences | Import data audiens Facebook |
| Import WhatsApp    | Import kontak dari WhatsApp  |

## Filter & Pencarian

Memudahkan pencarian kontak berdasarkan:

- Kategori, Negara, Provinsi, Kota
- Sumber, List, Tag
- Tanggal dibuat

Klik [Filter] untuk menerapkan kriteria pencarian.

## Hapus Kontak

- Pilih satu atau beberapa kontak
- Klik [Hapus yang Terpilih]
- Konfirmasi tindakan

## Catatan

- Pastikan field wajib telah diisi sebelum menyimpan
- Data kontak dapat digunakan untuk CRM, Pesan, dan Promosi

<Tips>
## **Tips Penggunaan**
 
Agar pengelolaan kontak melalui Tool Contact lebih optimal, perhatikan beberapa tips berikut:
 
- Gunakan fitur 'Import WhatsApp' untuk memindahkan data percakapan pelanggan ke dalam sistem NoBox secara cepat.
- Berikan Tag yang spesifik pada setiap kontak (misal: 'Hot Prospect', 'VIP Customer') untuk mempermudah segmentasi promosi.
- Mapping field ID Eksternal dengan benar saat melakukan impor Excel agar tidak terjadi duplikasi data kontak.
- Gunakan fitur Filter berdasarkan 'Tanggal Dibuat' untuk memantau pertumbuhan database kontak Anda setiap bulannya.
- Aktifkan status 'Diblokir' hanya untuk kontak yang mengganggu atau spam agar laporan performa tim tetap akurat.
 
</Tips>

<Kendala>
## **Penanganan Kendala**
 
Jika mengalami kendala pada menu Tool Contact, lakukan langkah berikut:
 
- Jika proses impor Excel gagal, pastikan format file sudah mengikuti template yang disediakan oleh sistem.
- Refresh halaman jika kontak yang baru ditambahkan secara manual tidak muncul di daftar pencarian.
- Periksa koneksi internet jika sinkronisasi dengan Google Contacts atau Facebook Audiences berjalan lambat.
- Pastikan semua field wajib (Nama dan ID Eksternal) sudah terisi dengan benar saat mencoba menyimpan kontak baru.
- Hubungi tim support jika fitur hapus massal (Hapus yang Terpilih) mengalami gangguan teknis.
 
</Kendala>
