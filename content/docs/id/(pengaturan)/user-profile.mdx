Profil Pengguna adalah salah satu menu pada NoBox.AI yang berisi informasi tentang data diri pengguna dan digunakan untuk mengelola pengaturan format tampilan data di dalam sistem.

Menu ini membantu pengguna dalam menyesuaikan preferensi format tanggal, waktu, dan angka sesuai dengan standar yang digunakan di wilayah kerja masing-masing. Dengan pengaturan yang tepat, data yang ditampilkan di seluruh halaman sistem akan seragam dan mudah dipahami, sehingga meminimalkan risiko kesalahan interpretasi data.

## **Penjelasan Detail Setiap Komponen Profil Pengguna**

- **Umum**

Bagian ini berisi informasi dasar pengguna yang digunakan sebagai identitas akun di dalam sistem.

Data yang termasuk dalam bagian ini antara lain nama lengkap, alamat email, nomor telepon, dan informasi kontak lainnya. Pastikan data ini selalu diperbarui agar rekan tim tidak kesulitan saat perlu menghubungi Anda.

- **Alamat**

Menampilkan data alamat lengkap pengguna, termasuk alamat, kota, provinsi, negara, dan kode pos.

Data alamat ini penting untuk keperluan dokumentasi internal dan referensi lokasi pengguna di dalam sistem.

- **Format Tanggal**

Pengaturan ini digunakan untuk menentukan bagaimana tanggal ditampilkan di seluruh halaman sistem.

Pilih format yang sesuai dengan standar wilayah kerja Anda (contoh: DD/MM/YYYY untuk format Indonesia, atau MM/DD/YYYY untuk format internasional) agar tidak terjadi kesalahan interpretasi data waktu.

- **Format Waktu**

Pengaturan ini digunakan untuk menentukan bagaimana waktu ditampilkan di dalam sistem.

Pengguna dapat memilih format 24 jam atau 12 jam (AM/PM) sesuai preferensi masing-masing.

- **Format Angka**

Pengaturan ini digunakan untuk menentukan bagaimana angka dan desimal ditampilkan di dalam sistem.

Pilih format yang biasa digunakan di wilayah Anda (contoh: penggunaan titik sebagai pemisah ribuan dan koma sebagai pemisah desimal untuk format Indonesia) agar data laporan lebih mudah dibaca dan dipahami.

## **Cara Memperbarui Profil Pengguna**

Untuk memperbarui data pada halaman Profil Pengguna, ikuti langkah-langkah berikut:

1. Buka menu **[Profil Pengguna]** pada modul Pengaturan.
2. Edit data yang ingin diubah pada bagian yang sesuai (Umum, Alamat, Format Tanggal, Format Waktu, atau Format Angka).
3. Setelah selesai mengedit data, klik tombol **[Simpan]** untuk menyimpan perubahan.

Perubahan data akan langsung tersimpan dan berlaku di seluruh halaman sistem.

<Callout type="info" title="Informasi Penting">
  Perubahan pada format tanggal, waktu, atau angka akan mempengaruhi tampilan
  data di seluruh halaman sistem. Pastikan format yang dipilih sudah sesuai
  dengan kebutuhan sebelum menyimpan. Jika perubahan tidak langsung terlihat,
  coba logout dan login kembali.
</Callout>

<Tips>
## **Tips Penggunaan**
Agar profil pengguna Anda tetap optimal dan informatif, perhatikan tips berikut:
- Pastikan format tanggal dan waktu sudah sesuai dengan standar wilayah kerja Anda untuk menghindari kerancuan jadwal. 
- Gunakan format angka yang biasa Anda gunakan (misal: penggunaan titik atau koma sebagai desimal) agar data laporan lebih mudah dibaca.
- Selalu perbarui informasi kontak di bagian Umum agar rekan tim Anda tidak kesulitan saat perlu menghubungi Anda.
- Simpan perubahan secara berkala setiap kali Anda melakukan update pada bagian pengaturan format.
- Periksa kembali pengaturan format setelah menyimpan untuk memastikan tampilannya sudah sesuai yang diinginkan.
</Tips>

---

<Kendala>
## **Penanganan Kendala**
Jika mengalami kendala saat memperbarui Profil Pengguna, lakukan langkah berikut:
- Apabila tombol simpan tidak aktif, periksa kembali apakah ada field wajib yang belum terisi dengan benar.
- Jika perubahan format tanggal tidak langsung terlihat di menu lain, coba logout dan login kembali ke sistem.
- Pastikan koneksi internet stabil saat melakukan pembaruan data alamat agar proses penyimpanan tidak terputus.
- Periksa apakah format yang dipilih sudah sesuai jika data yang ditampilkan terlihat tidak benar.
- Bersihkan cache browser jika tampilan tidak sesuai setelah perubahan.
- Hubungi administrator jika fitur pengaturan format angka tidak berfungsi sebagaimana mestinya.
</Kendala>
