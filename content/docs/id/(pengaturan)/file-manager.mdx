File Manager adalah salah satu menu pada NoBox.AI yang berfungsi sebagai pusat penyimpanan terpusat untuk mengelola seluruh aset digital yang digunakan di dalam platform.

Menu ini membantu pengguna dalam mengorganisir, mencari, menyaring, dan menghapus file secara efisien. Dengan File Manager, semua file yang pernah diunggah atau digunakan dalam percakapan, promosi, maupun fitur lainnya akan tersimpan rapi di satu tempat, sehingga dapat diakses dan digunakan kembali kapan saja tanpa perlu mengunggah ulang.

## **Mengelola File di File Manager**

### Pencarian dan Penyaringan File

Untuk mempermudah pengelolaan, pengguna dapat memanfaatkan fitur pencarian dan penyaringan untuk menemukan file yang dibutuhkan dengan cepat.

1. Gunakan kolom pencarian untuk mencari file berdasarkan nama.
2. Gunakan fitur filter untuk menyaring file berdasarkan tipe atau tanggal unggah.
3. Hasil pencarian dan penyaringan akan ditampilkan secara real-time pada daftar file.

### Melihat Total Ukuran Penyimpanan

Jika Anda ingin mengetahui seberapa banyak data yang telah terpakai, klik pada opsi **[Size All Files]** yang tersedia.

Opsi ini akan menampilkan total ukuran seluruh file yang ada, sehingga memudahkan pengguna dalam memantau penggunaan penyimpanan dan mengambil tindakan jika kuota sudah mendekati batas.

## **Hapus File**

Untuk menghapus file yang tidak diperlukan lagi, ikuti langkah-langkah berikut:

1. Pilih terlebih dahulu file yang akan dihapus dari daftar file.
2. Klik tombol **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus file tersebut.
3. Jika ya, klik **[Ya]**. Jika tidak, klik **[Tidak]**.

<Callout type="info" title="Informasi Penting">
  Penghapusan file bersifat permanen dan tidak dapat dibatalkan. Pastikan file
  yang akan dihapus benar-benar tidak diperlukan lagi sebelum melakukan
  penghapusan.
</Callout>

<Tips>
  ## **Tips Penggunaan** Agar File Manager tetap efisien dan mudah digunakan,
  perhatikan beberapa tips berikut: - Berikan nama file yang jelas dan
  deskriptif sebelum diunggah agar mudah dicari di kemudian hari. - Gunakan
  fitur saring (filter) untuk mempercepat pencarian file berdasarkan tipe atau
  tanggal unggah. - Cek kapasitas penyimpanan secara berkala melalui menu
  **[Size All Files]** untuk memantau penggunaan kuota. - Hapus file atau aset
  digital yang sudah tidak digunakan lagi untuk menjaga ruang penyimpanan tetap
  bersih. - Organisir file dengan penamaan yang konsisten agar tidak terjadi
  duplikasi atau kebingungan saat mencari file. - Lakukan pencadangan (backup)
  file penting secara mandiri sebelum menghapusnya dari sistem.
</Tips>
---
<Kendala>
  ## **Penanganan Kendala** Jika menemui kendala saat menggunakan File Manager,
  lakukan langkah berikut: - Refresh halaman jika file yang baru diunggah tidak
  langsung muncul di daftar. - Pastikan ukuran file tidak melebihi batas
  maksimal yang diizinkan oleh sistem NoBox. - Periksa format file yang diunggah
  apakah sudah didukung oleh sistem. - Jika tombol hapus tidak berfungsi,
  periksa kembali hak akses akun Anda (apakah sebagai Owner/Manager). - Pastikan
  koneksi internet stabil saat proses pengunggahan atau penghapusan file
  berlangsung. - Logout dan login kembali jika fitur tidak berfungsi normal. -
  Bersihkan cache browser jika tampilan tidak sesuai. - Hubungi tim support jika
  Anda mengalami kegagalan saat mencoba mengunduh atau mengunggah file.
</Kendala>
