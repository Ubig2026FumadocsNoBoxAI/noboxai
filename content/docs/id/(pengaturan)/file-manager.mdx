## Mengelola File di File Manager

Untuk mempermudah pengelolaan, kalian bisa memanfaatkan fitur pencarian dan penyaringan untuk menemukan file yang dibutuhkan. Jika kalian ingin mengetahui seberapa banyak data yang telah terpakai, **klik pada opsi \[Size All Files]** yang tersedia. Opsi ini akan menampilkan total ukuran semua file yang ada, sehingga memudahkan dalam memantau penggunaan penyimpanan.

## Hapus File

Untuk mengedit suatu File, silahkan pilih terlebih dahulu data mana yang akan dihapus, kemudian klik **\[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut. Jika ya, klik **\[Ya]** jika tidak maka klik **\[Tidak]**.

<Tips>
## **Tips Penggunaan**
 
Agar File Manager tetap efisien dan mudah digunakan:
 
- Berikan nama file yang jelas dan deskriptif sebelum diunggah agar mudah dicari di kemudian hari.
- Gunakan fitur saring (filter) untuk mempercepat pencarian file berdasarkan tipe atau tanggal unggah.
- Cek kapasitas penyimpanan secara berkala melalui menu **[Size All Files]** untuk memantau penggunaan kuota.
- Hapus file atau aset digital yang sudah tidak digunakan lagi untuk menjaga ruang penyimpanan tetap bersih.
 
</Tips>

<Kendala>
## **Penanganan Kendala**
 
Jika menemui kendala saat menggunakan File Manager:
 
- Apabila file yang baru diunggah tidak muncul, coba segarkan (refresh) halaman browser Anda.
- Pastikan ukuran file tidak melebihi batas maksimal yang diizinkan oleh sistem NoBox.
- Jika tombol hapus tidak berfungsi, periksa kembali hak akses akun Anda (apakah sebagai Owner/Manager).
- Hubungi tim support jika Anda mengalami kegagalan saat mencoba mengunduh file yang tersimpan.
 
</Kendala>
