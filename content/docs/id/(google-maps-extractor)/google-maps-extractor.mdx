Google Maps Extractor adalah fitur unggulan pada NoBox Desktop yang memungkinkan Anda mengumpulkan data bisnis secara massal langsung dari profil Google Maps. Tool ini sangat efektif untuk tim sales dan marketing dalam membangun database calon pelanggan (_lead generation_) secara cepat dan akurat.

Dengan sistem ekstraksi otomatis, Anda bisa mendapatkan informasi seperti nama bisnis, alamat, nomor telepon, rating, hingga kategori industri hanya dalam hitungan menit.

## **Fitur Utama Web Extractor**

NoBox menyediakan berbagai kontrol untuk memastikan proses ekstraksi berjalan aman dan sesuai dengan kebutuhan target pasar Anda.

### **1. Set Karakter Unicode**

NoBox mendukung penyimpanan hasil pencarian dalam format karakter internasional (Unicode) seperti bahasa Arab, Cina, Jepang, dan lainnya. Hal ini memungkinkan Anda melakukan riset pasar global tanpa kendala bahasa.

## **Manajemen Proses Ekstraksi**

### Halaman Konfigurasi (Page Process)

Di halaman ini, Anda dapat menentukan parameter pencarian sebelum memulai ekstraksi data.

| Fitur            | Uraian Fungsi                                                         |
| :--------------- | :-------------------------------------------------------------------- |
| **Limit Result** | Membatasi jumlah data yang ingin diambil agar sesuai kuota.           |
| **Save To List** | Mengarahkan hasil ekstraksi ke grup kontak/list tertentu di NoBox.AI. |
| **Option Debug** | Menampilkan log aktivitas sistem secara sangat detail untuk audit.    |
| **Load Images**  | Opsi untuk mengunduh gambar profil bisnis jika tersedia.              |

<Tips>
## **Tips Penggunaan**

Optimalkan ekstraksi data Anda dengan tips berikut:

- Gunakan fitur **Multi Tab** untuk menjalankan beberapa kata kunci pencarian berbeda secara bersamaan.
- Atur interval jeda yang bervariasi (random) agar aktivitas terlihat lebih natural di mata sistem Google.
- Pastikan Anda memilih list yang tepat di bagian **Save To List** agar data tidak tercampur dengan database lama.
- Aktifkan fitur **Auto Refresh** pada website NoBox.AI untuk memantau trafik data yang masuk secara langsung.

</Tips>

<Kendala>
## **Penanganan Kendala**

Jika menemui masalah teknik, silakan lakukan langkah berikut:

- **Data Gagal Ditarik**: Periksa koneksi internet Anda atau pastikan akun Google Anda tidak sedang memerlukan verifikasi captcha.
- **Hasil Sedikit**: Coba gunakan kata kunci yang lebih umum atau perluas radius pencarian pada peta Google Maps.
- **Aplikasi Close Sendiri**: Periksa ketersediaan RAM komputer Anda dan tutup aplikasi lain yang tidak diperlukan.
- **Data Tidak Masuk ke Web**: Pastikan status akun di NoBox Desktop adalah "Aktif" dan sinkronisasi cloud tidak terhambat firewall.

</Kendala>
