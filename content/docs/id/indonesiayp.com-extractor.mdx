Indonesiayp.com Extractor adalah fitur pada NoBox Desktop yang memungkinkan Anda mengambil data profil bisnis (Yellow Pages) dari situs Indonesiayp.com secara massal. Dengan tool ini, Anda dapat mengumpulkan informasi perusahaan seperti kategori bisnis, alamat, nomor telepon, dan lokasi di seluruh Indonesia dengan cepat.

Sangat ideal bagi tim pemasaran B2B untuk menyusun database prospek bisnis yang terverifikasi langsung dari sumber direktori bisnis terbesar di Indonesia.

## **Menghubungkan Akun Extractor**

Untuk mulai melakukan ekstraksi, Anda harus mendaftarkan akun extractor baru di sistem NoBox Desktop.

### Langkah Penambahan Akun:

1. Buka aplikasi **NoBox Desktop** ➔ modul **[Extractor]** ➔ klik **[Tambah Akun]**.
2. Pilih kanal **Indonesiayp.com** pada dialog yang muncul.
3. Lengkapi rincian nama akun dan kategori sebagai identitas internal Anda.
4. Klik **[OK]** untuk menyimpan.

### Konfigurasi Awal
Buka akun yang baru saja dibuat. Pastikan pada tab **[Logs]** muncul status **"Ready to start"**. Hal ini menandakan sistem siap melakukan navigasi ke situs Indonesiayp secara otomatis.

## **Manajemen Proses Ekstraksi**

### Halaman Pengaturan (Page Process)
Sebelum mulai menarik data, tentukan parameter pencarian Anda agar hasilnya lebih spesifik.

| Isian | Fungsi Detail |
| :--- | :--- |
| **Searching** | Kata kunci bisnis (misal: "Bags", "Restaurant"). |
| **Category** | Memilih kategori industri yang relevan. |
| **Limit Result** | Membatasi jumlah data perusahaan yang akan ditarik. |
| **Save To List** | Menentukan folder penyimpanan kontak di cloud NoBox.AI. |
| **Options** | Mengaktifkan mode debug atau pemuatan gambar profil. |

## **Sinkronisasi dan Ekspor Data**

### Hasil Real-Time (Page Result)
Setiap data yang berhasil ditarik akan muncul di tab Result. Jika jumlah data akhir lebih sedikit dari **Limit Result**, berarti sistem telah menyaring dan membuang data duplikat secara otomatis.

### Sinkronisasi ke Website NoBox.AI
Seluruh data hasil ekstraksi otomatis tersimpan di menu **Kontak** dashboard NoBox.AI. Gunakan filter **Source ➔ Indonesiayp.com** untuk melihat data tersebut.

Sistem menyediakan opsi download data dalam format `.xlsx`, `.csv`, atau `.json` melalui tombol **Action** di halaman **Results**.

## **Fitur Lanjutan dan Pemulihan**

### Multi Tab (Banyak Pencarian)
Anda dapat menjalankan beberapa pencarian di lokasi berbeda secara bersamaan dengan menggunakan fitur Multi Tab pada NoBox Desktop.

### Pemulihan Sesi (Auto Recovery)
Jika aplikasi tertutup secara tiba-tiba (komputer mati), NoBox akan mendeteksi proses yang terhenti. Saat dibuka kembali, akan muncul notifikasi untuk melanjutkan atau membatalkan proses ekstraksi sebelumnya.

<Tips>
## **Tips Penggunaan**

Optimalkan penggunaan Indonesiayp Extractor dengan panduan berikut:

- Gunakan kata kunci pencarian yang spesifik namun umum (misal: "Laundry" daripada "Laundry Cuci Sepatu") untuk mendapatkan lebih banyak hasil.
- Manfaatkan fitur **Save to List** dengan nama folder yang mencantumkan tanggal dan kategori agar database Anda terorganisir.
- Aktifkan **Auto Refresh** pada dashboard web NoBox.AI untuk memantau trafik data masuk tanpa harus menekan tombol refresh.
- Lakukan ekspor ke Excel secara berkala untuk kebutuhan monitoring offline atau backup data lokal.

</Tips>

<Kendala>
## **Penanganan Kendala**

Jika menemui masalah pada layanan extractor, silakan ikuti langkah berikut:

- **Logs Menunjukkan Pesan Error**: Periksa kembali koneksi internet Anda atau pastikan website Indonesiayp.com dapat diakses secara manual melalui browser biasa.
- **Data Tidak Masuk ke Web**: Pastikan akun extractor Anda dalam status "Aktif" (centang status pada manajemen akun).
- **Proses Berhenti Sendiri**: Periksa ketersediaan RAM komputer; menonaktifkan akun lain yang tidak digunakan dapat membantu mempercepat proses.
- **Hasil Data Kosong**: Coba ubah kategori atau perluas jangkauan lokasi pencarian Anda di peta.
- **Aplikasi Lambat**: Hapus log aktivitas lama atau bersihkan file sampah (temporary files) di sistem operasi Anda.

</Kendala>
