---

## **Fitur Filter Data Menu Percakapan**

Pada menu Percakapan, pengguna dapat memfilter data percakapan berdasarkan beberapa kategori berikut:

### 1. Filter Status

Digunakan untuk menyaring data percakapan berdasarkan status penanganan, seperti aktif, diproses, atau selesai.

### 2. Filter Kanal

Digunakan untuk menampilkan data percakapan berdasarkan channel komunikasi yang digunakan, seperti WhatsApp, Telegram, atau platform lainnya.

### 3. Filter Akun

Digunakan untuk memfilter percakapan berdasarkan akun atau nomor yang terhubung dalam sistem.

### 4. Filter Tambahan

Selain filter utama, tersedia juga filter lain yang dapat digunakan sesuai kebutuhan monitoring data.

Pengguna dapat mengombinasikan beberapa filter sekaligus untuk mendapatkan hasil data yang lebih spesifik dan akurat.

---

## **Melihat Detail Percakapan**

Untuk melihat detail data percakapan, Anda dapat melakukan langkah berikut:

1. Pilih salah satu data percakapan pada daftar.
2. Klik ikon [ <img src="/images/id/Go%20to%20message.png" alt="Buka Pesan" className="inline h-5 w-auto mx-1" /> ] **[Buka Percakapan]**.
3. Sistem akan mengarahkan Anda ke halaman percakapan yang dipilih.

Pada Tips Penggunaan menu percakapan, pengguna dapat melihat isi pesan, informasi kontak, serta riwayat komunikasi secara lengkap.

## **Tips Penggunaan**
 
Agar penggunaan menu Percakapan lebih optimal, perhatikan beberapa tips berikut:
 
- Gunakan filter secara rutin untuk memantau status percakapan.
- Lakukan pengecekan data percakapan secara berkala.
- Manfaatkan fitur pencarian dan filter untuk efisiensi waktu.
- Pastikan data akun dan channel selalu terhubung dengan baik.
- Gunakan menu ini sebagai bahan evaluasi performa layanan.
 
</Tips>

## **Penanganan Kendala**
 
Jika mengalami kendala saat menggunakan menu Percakapan, Anda dapat mencoba langkah berikut:
