## **Fungsi Menu Funnel**

Menu Funnel memiliki beberapa fungsi utama, antara lain:

- Mengelompokkan tahapan proses pemasaran.
- Mengelola status prospek dan pelanggan.
- Membantu monitoring proses layanan.
- Mempermudah evaluasi kinerja tim.
- Menyusun alur komunikasi yang lebih terstruktur.

---

## **Tambah Funnel**

Untuk membuat data funnel, silahkan klik menu **[Funnel] ➔ [Tambah Funnel]**. Maka akan tampil dialog **[Tambah Funnel]**, kemudian isi nama funnel pada dialog tersebut.

**Form Tambah Funnel**

Pada dialog Tambah Funnel, pengguna hanya perlu mengisi nama funnel sesuai dengan kebutuhan bisnis.

Jika Anda sudah mengisi nama funnel pada dialog di atas, klik **[Simpan]** untuk menyimpan data tersebut.

---

## **Edit dan Hapus Data Funnel**

### Edit Data Funnel

Untuk mengedit suatu data funnel, silahkan pilih data yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

### Hapus Data Funnel

Untuk menghapus suatu data funnel yaitu dengan pilih data funnel yang akan dihapus, kemudian klik **[Hapus]** maka akan muncul dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]**, jika tidak klik **[Tidak]**.

---

## **Cara Penggunaan Funnel**

Funnel dapat digunakan langsung pada halaman percakapan untuk mengelompokkan tahapan proses layanan dan pemasaran.

Berikut langkah-langkah penggunaan Funnel:

**Langkah 1: Membuka Percakapan**

Buka menu **[Kotak Masuk]**, kemudian pilih salah satu percakapan yang ada.

**Langkah 2: Membuka Informasi Percakapan**

Klik ikon  [ <img src="/images/id/information%20icon.png" alt="Informasi" className="inline h-5 w-auto mx-1" /> ]  maka akan tampil detail dari chat tersebut.

**Langkah 3: Memilih Funnel**

Klik pada bagian Funnel maka akan tampil data funnel yang telah dibuat sebelumnya atau tambahkan funnel baru.

Selanjutnya, pilih salah satu dari data funnel tersebut.

## **Manfaat Penggunaan Funnel**

Penggunaan fitur Funnel memberikan beberapa manfaat, antara lain:

- Memudahkan pengelompokan status pelanggan.
- Memperjelas tahapan komunikasi.
- Membantu analisis proses penjualan.
- Meningkatkan efektivitas layanan.
- Mendukung pengambilan keputusan.

---

## **Tips Penggunaan Funnel**

Agar penggunaan Funnel lebih optimal, perhatikan beberapa tips berikut:

- Buat funnel sesuai alur bisnis.
- Gunakan nama funnel yang mudah dipahami.
- Evaluasi funnel secara berkala.
- Sesuaikan tahapan dengan kebutuhan pelanggan.
- Hindari penggunaan funnel yang terlalu banyak.

---

## **Penanganan Kendala**

Jika terjadi kendala saat menggunakan fitur Funnel, lakukan langkah berikut:

- Refresh halaman sistem.
- Periksa koneksi internet.
- Pastikan akun memiliki hak akses.
- Logout lalu login kembali.
- Hubungi tim support jika masalah berlanjut.
