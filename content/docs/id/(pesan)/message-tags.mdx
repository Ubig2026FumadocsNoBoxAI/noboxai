Menu ini membantu pengguna dalam mengatur, mengelola, dan memfilter percakapan dengan lebih mudah, sehingga pesan dapat ditemukan dan ditangani secara lebih cepat dan terstruktur.

## **Tambah Tag**

Untuk membuat data tag, silahkan klik menu **[Tag Pesan] ➔ [Tambah Tag Pesan]**. Maka akan tampil dialog **[Tambah Tag Pesan]** kemudian isi nama tag pada dialog tersebut.

**Tampilan Tambah Tag Pesan**

Pada dialog Tambah Tag Pesan, pengguna dapat mengisi nama tag sesuai dengan kategori pesan yang diinginkan.

Jika Anda sudah melengkapi nama tag di atas, klik **[Simpan]** untuk meyimpan data tag tersebut.

## **Edit dan Hapus Data Tag Pesan**

### Edit Data Tag Pesan

Untuk mengedit suatu data tag, silahkan pilih terlebih dahulu data mana yang akan diedit. Editlah data Anda dan klik **[Simpan]** untuk menyimpan perubahan tersebut.

Perubahan data akan langsung tersimpan dan dapat digunakan kembali dalam sistem.

### Hapus Data Tag Pesan

Untuk menghapus suatu data tag adalah dengan pilih data tag yang akan dihapus, kemudian klik **[Hapus]**, maka akan tampil dialog konfirmasi apakah Anda ingin menghapus data tersebut.

Jika ya, klik **[Ya]** jika tidak maka klik **[Tidak]**.

---

## **Cara Penggunaan Tag Pesan**

Tag Pesan dapat digunakan langsung pada halaman percakapan untuk membantu pengelompokan dan monitoring pesan.

Berikut langkah-langkah penggunaan Tag Pesan:

1\. Buka menu **[Kotak Masuk]**, pilih salah satu percakapan yang ada.

2\. Klik ikon [ <img src="/images/id/information%20icon.png" alt="Informasi" className="inline h-5 w-auto mx-1" /> ] maka akan tampil detail dari percakapan tersebut.

3\. Klik pada bagian Tag Pesan maka akan tampil data tag yang telah dibuat sebelumnya atau tambahkan tag baru.

4\. Pilih satu atau lebih dari data tag tersebut.

<Tips>
## **Tips Penggunaan**
 
Agar penggunaan Tag Pesan lebih optimal, perhatikan beberapa tips berikut:
 
- Gunakan nama tag yang jelas dan konsisten.
- Hindari membuat tag yang terlalu banyak.
- Lakukan evaluasi tag secara berkala.
- Sesuaikan tag dengan kebutuhan operasional.
- Hapus tag yang sudah tidak digunakan.
 
</Tips>
