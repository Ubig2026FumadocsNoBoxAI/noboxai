Kendala login biasanya terjadi karena akun yang belum terverifikasi, kesalahan input kredensial, atau pemilihan server yang tidak tepat. Halaman ini akan membantu Anda mengidentifikasi penyebab masalah dan memberikan solusi langkah demi langkah untuk kembali mengakses layanan NoBox.AI.

Pastikan Anda memiliki akses ke email terdaftar untuk proses verifikasi dan pemulihan akun.

---

## **Pemeriksaan Pendaftaran Akun**

Sebelum mencoba login, pastikan Anda telah menyelesaikan seluruh proses pendaftaran akun di website resmi kami.

### Langkah Pendaftaran yang Benar:
1. Kunjungi [https://NoBox.AI/](https://NoBox.AI/) dan klik tombol **Login**.
2. Pilih lokasi server yang paling dekat dengan wilayah operasional Anda untuk kecepatan akses terbaik.
3. Gunakan menu **Sign Up** dan lengkapi formulir pendaftaran penyewa (*tenant*) baru.
4. **Penting:** Periksa kotak masuk (inbox) atau folder spam email Anda, lalu klik tautan verifikasi yang kami kirimkan. Akun Anda tidak akan bisa digunakan sebelum proses verifikasi ini selesai.

## **Solusi Masalah Login**

Jika Anda sudah memiliki akun tapi tetap tidak bisa masuk, silakan ikuti panduan berikut:

### 1. Login Dashboard Web
Gunakan email dan kata sandi yang telah didaftarkan pada alamat [id.NoBox.AI](https://NoBox.AI/). Di dashboard ini, Anda dapat memantau seluruh data dan status paket langganan Anda secara transparan.

### 2. Login Aplikasi NoBox Desktop
Aplikasi desktop memerlukan kredensial yang sama dengan akun website Anda. Jika Anda baru saja mendaftar, tunggu sekitar 1-2 menit agar database tersinkronisasi sebelum mencoba login di aplikasi desktop.

<Callout type="info" title="Informasi Penting">
Satu akun NoBox hanya dapat aktif di satu sesi login desktop utama. Jika Anda mencoba login di perangkat baru, sesi di perangkat sebelumnya mungkin akan terputus secara otomatis demi keamanan data.
</Callout>

<Tips>
## **Tips Login Aman & Lancar**

- Selalu simpan kata sandi Anda di pengelola kata sandi (*password manager*) yang aman untuk menghindari kesalahan ketik.
- Pastikan tombol **Caps Lock** tidak aktif saat memasukkan kata sandi.
- Gunakan koneksi internet pribadi; penggunaan VPN atau Proxy terkadang dapat menghambat proses autentikasi server.
- Lakukan pembersihan cache browser secara berkala jika halaman login terasa lambat atau tidak merespon.

</Tips>

<Kendala>
## **Penanganan Kendala Lanjutan**

Jika Anda tetap tidak bisa masuk ke akun, coba langkah-langkah berikut:

- **Lupa Kata Sandi**: Gunakan fitur [Lupa Password](forgot-password) untuk mengatur ulang kata sandi Anda melalui email.
- **Email Verifikasi Tidak Diterima**: Klik tombol "Kirim Ulang Email Verifikasi" pada halaman login atau hubungi dukungan admin jika email tetap tidak masuk dalam 10 menit.
- **Pesan Error "Tenant Not Found"**: Pastikan Anda masuk melalui URL server yang benar sesuai saat pendaftaran awal.
- **Aplikasi Desktop Blank**: Pastikan aplikasi memiliki izin akses penuh ke internet dan tidak diblokir oleh antivirus.

</Kendala>
