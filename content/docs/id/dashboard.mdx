## **Penjelasan Setiap Fitur**

### 1. Fitur Percakapan

Fitur ini digunakan untuk memantau aktivitas komunikasi dengan pelanggan dalam bentuk statistik dan grafik tahunan.

Data ini membantu pengguna dalam mengevaluasi performa layanan.

### 2. Fitur Kontak

Fitur Kontak menampilkan jumlah dan lokasi pelanggan dalam bentuk peta.

Informasi ini berguna untuk mengetahui sebaran pelanggan.

### 3. Fitur List

Fitur List berfungsi untuk menampilkan total data daftar yang tersimpan di dalam sistem.

Data ini dapat digunakan sebagai referensi pengelolaan informasi.

### 4. Fitur Kirim Pesan

Fitur Kirim Pesan menampilkan statistik pengiriman pesan dalam periode tertentu.

Pengguna dapat memantau efektivitas komunikasi melalui fitur ini.

### 5. Fitur Tugas

Fitur Tugas digunakan untuk mengelola aktivitas kerja harian.

Pengguna dapat menambahkan, menyelesaikan, maupun menghapus tugas sesuai kebutuhan.

---

## **Manfaat Penggunaan Dasbor**

Dengan memanfaatkan dasbor secara optimal, pengguna dapat:

* Memantau performa sistem secara real-time
* Mengelola data pelanggan dengan lebih efektif
* Mengatur tugas dan aktivitas kerja
* Membantu pengambilan keputusan berbasis data

---

## **Tips Menggunakan Dasbor**

Agar penggunaan dasbor lebih maksimal, perhatikan tips berikut:

* Periksa dasbor secara rutin
* Gunakan data sebagai dasar evaluasi
* Pastikan informasi selalu diperbarui
* Manfaatkan fitur tugas untuk manajemen waktu
