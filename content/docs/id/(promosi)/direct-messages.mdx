Menu ini sangat berguna untuk komunikasi mendadak, konfirmasi pesanan, atau tindak lanjut cepat kepada pelanggan tertentu.

## **Proses Kirim Pesan**

Untuk mengirim pesan langsung, Anda perlu melengkapi beberapa isian pada formulir yang tersedia.

### Isian Formulir Pesan Langsung

| Isian | Uraian | Contoh |
| :--- | :--- | :--- |
| **Kontak** | Pilih satu atau beberapa kontak tujuan. | Sheila Syandana |
| **Kanal** | Pilih kanal komunikasi (misal: WhatsApp). | WhatsApp |
| **Akun** | Pilih akun pengirim yang telah terintegrasi. | s_whatsapp tester |
| **Format Pesan** | Tentukan format pesan (Teks, Gambar, dll). | Teks |
| **Tautan Singkat** | Aktifkan fitur pemendek link jika diperlukan. | Tidak aktif |
| **Pesan** | Isi pesan yang ingin disampaikan. | Selamat sore, berikut informasi promo... |
| **Lampiran** | Tambahkan file pendukung jika diperlukan. | - |

Setelah melengkapi data tersebut, klik tombol **[Kirim Sekarang]** untuk memproses pengiriman.

<Tips>
## **Tips Penggunaan**

Beberapa tips untuk memudahkan penggunaan Kirim Pesan Langsung:

- Gunakan fitur pencarian pada kolom kontak untuk menemukan nama pelanggan dengan cepat.
- Pastikan akun pengirim dalam status aktif sebelum menekan tombol kirim.
- Gunakan bahasa yang personal agar pelanggan merasa lebih dihargai.
- Periksa kembali isi pesan sebelum dikirim untuk menghindari kesalahan ketik (typo).
- Manfaatkan lampiran untuk mengirimkan katalog atau bukti transaksi secara langsung.

</Tips>

<Kendala>
## **Penanganan Kendala**

Jika menemui kendala saat mengirim pesan langsung, silakan ikuti panduan berikut:

- **Pesan Gagal Terkirim**: Pastikan nomor tujuan valid dan akun pengirim terhubung dengan stabil.
- **Kanal Tidak Muncul**: Periksa kembali integrasi akun Anda pada menu profil usaha.
- **Loading Terlalu Lama**: Refresh halaman dan pastikan koneksi internet Anda stabil.
- **Kontak Tidak Ditemukan**: Pastikan kontak tersebut sudah terdaftar di database Kontak NoBox.AI.
- **Gagal Upload Lampiran**: Periksa ukuran file dan pastikan format file didukung oleh kanal terkait.

</Kendala>
