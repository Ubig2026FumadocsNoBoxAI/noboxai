This integration hub enables automatic synchronization of conversation data and activities between NoBox Desktop and the NoBox.AI cloud platform.

## **Adding a WhatsApp Account**

To add a WhatsApp account, click the **[Add Account]** button and select the WhatsApp channel.

### **WhatsApp Add Dialog Fields**

| Field | Description | Example |
| :--- | :--- | :--- |
| **Name** | Identity name for the WhatsApp account. | Primary CS #1 |
| **Category** | Account grouping (e.g., Sales). | Automation |
| **Channel** | Communication platform. | WhatsApp |
| **Status** | Account activation status. | Active |
| **Notes**| Optional description. | Pro Sales Account |

After saving, you are required to scan the QR code using the WhatsApp application on your phone to link the account.

## **Adding Social Media & Marketplace Accounts**

NoBox.AI supports various other platforms like Telegram, Instagram, Shopee, Tokopedia, and Bukalapak. The process for adding accounts on these platforms generally involves filling in credentials (username/password) or a phone number followed by OTP verification.

### **Marketplace Account Addition Examples**

| Platform | Primary Field | Verification Method |
| :--- | :--- | :--- |
| **Telegram** | Client ID (Phone Number) | OTP Code via Telegram |
| **Shopee** | Account Name & Category | OTP Code via SMS/App |
| **Tokopedia** | Username & Password | OTP Code |
| **Instagram** | Username & Password | Login Verification |

## **Editing and Deleting Account Data**

### **Edit Account Data**
Select the account you want to change, adjust the information, and then click **[Save]**. Changes will be immediately synchronized throughout the system.

### **Delete Account Data**
Select the account to be deleted, click the **[Delete]** button, and confirm your action.

<Callout type="info" title="Important Information">
Deleting account data is permanent and will disconnect all integrations related to that account. Ensure the account data being deleted is truly no longer needed.
</Callout>

<Tips>
## **Usage Tips**

Optimize your account management with these tips:

- Use clear account names (e.g., "WhatsApp_CS_Jakarta") for easier monitoring.
- Ensure a stable internet connection when scanning QR codes or entering OTPs.
- Turn off accounts that are not actively being used to save computer resources.
- Regularly verify accounts to ensure integrations remain smooth.
- Use appropriate categories to keep the account list neatly organized.

</Tips>

<Kendala>
## **Troubleshooting**

If you encounter issues in the Account menu, please check the following solutions:

- **WhatsApp QR Scan Failed**: Ensure the phone screen is clean and the WhatsApp camera is focused on the QR code.
- **OTP Not Received**: Check cellular signal or ensure the phone number entered is correct.
- **Account Logout Status**: Re-login or re-scan the QR code to reconnect the broken connection.
- **Slow Application**: Deactivate unnecessary accounts to reduce RAM load.
- **Failed to Save Account**: Ensure all mandatory fields are filled and the phone number format is correct (using country code).

</Kendala>
