## **Adding a Task Tag**

To define a new tag, navigate to the **[CRM]** module ➔ **[Tasks]** ➔ **[Task Tags]** ➔ **[Add Task Tag]**.

An **[Add Task Tag]** dialog will prompt you to enter the tag name.

### Add Tag Dialog View

<ImageZoom
  src="/images/id/tampilan%20Tambah%20tag%20tugas.png"
  alt="Adding a New Task Tag"

  className="rounded-lg border shadow-sm mb-4"

Once the name is entered, click **[Save]** to store the tag. Ensure the name is descriptive and aligned with your team's workflow naming conventions.

## **Editing and Deleting Tag Data**

### Editing Tags

To modify a tag, select the target record from the list. Update the name and click **[Save]**. Changes are updated immediately and will be reflected in all active tasks using that tag.

### Deleting Tags

To remove a tag, select it and click **[Delete]**. A confirmation prompt will appear to ensure the action is intentional.

Click **[Yes]** to confirm or **[No]** to cancel.

<Callout type="info" title="Important Information">
  Tag deletion is permanent and cannot be undone. Ensure that the tag is no
  longer needed for historical reporting before deleting it.
</Callout>

## **How to Apply Task Tags**

### Using Tags During Task Management

Tags can be assigned both during the initial creation of a task and when updating existing tasks.

Follow these steps:

1. Open the **[Tasks]** menu and select a task.
2. Click on the **Tags** input field; this will display a dropdown of all available tags.
3. Select one or more tags and click **[Save]**.

## **Filtering by Task Tags**

### The Utility of Tag Filters

The **Tag Filter** is designed to isolate specific action items, showing only the records that carry your selected labels.

This is particularly useful for quickly focusing on specific project phases or priority levels during busy work shifts.

### Applying Tag Filters

To filter your task list:

- Navigate to the **[Tasks]** menu.
- Click the **[Filter]** button.
- Select the **[Tags]** section.
- Choose your desired tags from the dropdown.
- Click **[Apply/Show]**.

The task list will immediately update to reflect your filter criteria.

<Tips>
## **Usage Tips**

To optimize your tagging workflow:

- Use short, intuitive, and consistent tag names.
- Avoid over-tagging a single task; focus on the most relevant labels.
- Periodically delete obsolete or duplicate tags to keep the dropdown menu clean.
- Combine tags with the **Search** feature for maximum retrieval speed.

Effective tag management directly translates to a more organized and productive team environment.

</Tips>

## **Troubleshooting**

If you encounter difficulties while using Task Tags, consider these troubleshooting steps:

**Account Permissions**
    - Verify that your user role has the authority to manage organizational tags. 
    - Ensure your session hasn't expired (try logging out and back in). 
    - Confirm you are modifying tags within the correct business tenant.

**System Performance**
    - Check your internet stability for real-time synchronization. 
    - Ensure you are using a modern, updated web browser. 
    - Refresh the page if the tag dropdown fails to load or update. 
    - Clear browser cache if UI elements appear broken or misaligned.
    
**Data Entry Errors**
    - Double-check for duplicate tag names (the system may prevent duplicates).
    - Ensure no special characters are used if they are unsupported by the
    system. 
    - Check if any browser extensions are interfering with the dropdown
    scripts. 
    - Verify if NoBox.AI is undergoing scheduled maintenance.
</Kendala>
