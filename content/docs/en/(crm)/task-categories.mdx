## **Adding a Task Category**

To create a new category, navigate to the **[CRM]** module, select the **[Tasks]** menu ➔ **[Categories]** ➔ **[Add Task Category]**.

An **[Add Task Category]** dialog will appear.

### Add Task Category Dialog View

### Task Category Form Fields

Complete the required field(s); for example:

| Field | Description    | Example |
| ----- | -------------- | ------- |
| Name  | Category Title | VIP     |

Enter a category name that clearly defines the group of tasks it represents.

Once you have entered the information, click **[Save]** to store the new category.

## **Editing and Deleting Category Data**

### Editing Task Categories

To modify a category name, select the target record from the list. Update the name and click **[Save]**.

The changes will be applied immediately across all tasks currently linked to that category.

### Deleting Task Categories

To remove a category, select the entry and click **[Delete]**. A confirmation dialog will appear to prevent accidental removal.

Click **[Yes]** to confirm or **[No]** to cancel.

<Callout type="info" title="Important Information">
  Deleting a task category is permanent. Ensure the category is no longer needed
  in historical reports before proceeding with deletion.
</Callout>

## **How to Apply Task Categories**

Task categories can be assigned directly when creating or updating any CRM task.

Follow these steps:

1. Open the **[Tasks]** menu and select an existing task or create a new one.

2. Click on the **Category** dropdown field; this will display all previously defined categories.

<Tips>
## **Usage Tips**

To optimize your task management organization:

- Use clear, intuitive names that the whole team understands.
- Keep the number of categories manageable to avoid analysis paralysis.
- Periodically audit categories to see which ones are rarely used.
- Align categories with your core business departments (e.g., Sales, Support, Operations).
- Standardize category usage across all agents for consistent reporting.

</Tips>

## **Troubleshooting**

If you experience issues within the Task Categories module:

- Refresh the page to synchronize with the latest server state.
- Verify that your internet connection is stable.
- Confirm your account has "Supervisor" permissions to manage categories.
- Try logging out and back in if the "Save" function seems unresponsive.
- Contact support if you notice categories disappearing or not saving correctly.
