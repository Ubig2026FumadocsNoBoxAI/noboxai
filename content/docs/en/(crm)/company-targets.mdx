## **Adding Company Target Data**

To create a new company target, navigate to the **[CRM]** module, click the **[Companies]** menu, then select **[Company Targets] ➔ [Add Company Target]**.

An **[Add Company Target]** dialog will appear. Complete the fields based on your company's planning requirements.

### Add Target Dialog View

<ImageZoom
  src="/images/id/Tampilan%20tambah%20Target%20perusahaan.png"
  alt="Adding a New Company Target"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

This form is used to enter detailed marketing target data for a company.

### Add Target Form Fields

| Field              | Description                 | Example      |
| ------------------ | --------------------------- | ------------ |
| Company            | Select the target company   | PT SEJAHTERA |
| Year               | Select the target year      | 2024         |
| Currency           | Select the currency         | Rp           |
| January - December | Enter monthly target values | 1,200,000    |

Each field must be completed accurately so that the system can calculate and display achievement data correctly.

Ensure that the target amounts are aligned with the company's current conditions and capacity.

Once all information is entered, click **[Save]** to store the company target data.

## **Editing and Deleting Company Targets**

### Editing Target Data

To edit a company target, select the target record from the list. Update the values as needed and click **[Save]** to apply the changes.

The updated data will be saved immediately and reflected in reports.

### Deleting Target Data

To remove a target entry, select the record and click **[Delete]**. A confirmation prompt will appear.

Click **[Yes]** to confirm or **[No]** to cancel.

<Callout type="info" title="Important Information">
  Deleting target data is permanent and cannot be undone. Ensure that the data
  is no longer required before deletion.
</Callout>

## **Current Revenue**

### Understanding Current Revenue

**Current Revenue** is an indicator that shows the amount of income achieved based on deal data.

Current revenue is calculated and displayed when a deal associated with that company reaches a **[Won]** status.

### Revenue Calculation Example

Example:  
If a deal for **PT SEJAHTERA** with a value of **18,000,000** is marked as **Won**, the Company Target view will reflect a current revenue of **18,000,000** for that target period.

This feature allows users to easily compare targets against actual performance.

## **Usage Tips**

To optimize your target management process, consider these tips:

- Perform monthly evaluations of goal achievements.

- Keep data regularly updated.
