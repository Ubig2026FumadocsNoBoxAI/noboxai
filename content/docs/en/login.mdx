## **Preparation Before Login**

Before attempting to log in, ensure the following requirements are met to prevent any obstacles during the process:

*   Your NoBox.AI account is registered and verified via email.
*   Your username and password are ready.
*   Your device is connected to a stable internet connection.
*   The NoBox Desktop application has been downloaded.
*   Your computer's operating system meets the application's specifications.

---

## **Downloading the NoBox Desktop Application**

This section explains how to download the NoBox Desktop application from the official website.

### Steps to Download

1.  Visit the official NoBox.AI website and log in to your account.
2.  Once logged in, navigate to the **Subscription** menu.
3.  Select your active service package.
4.  Download the installer assigned to your computer's operating system.

Ensure the installer file is completely downloaded before proceeding to the next stage.

### Important Notes

*   Use a stable internet connection while downloading.
*   Ensure you have sufficient storage space.
*   Save the installer file in an easy-to-find location.

---

## **Application Installation Process**

After the installer file has been successfully downloaded, proceed with the installation process according to the following guide.

### Installation Steps

1.  Run the downloaded installer file.
2.  Follow the on-screen installation instructions.
3.  Choose the installation location as needed.
4.  Wait for the installation process to complete.

Once the installation is successful, the NoBox Desktop application is ready to use.

### Verifying the Installation

To ensure the installation was successful:

*   Check for the application icon on your desktop or in the Start menu.
*   Run the application to ensure no errors occur.
*   Confirm that the system folders have been created.

---

## **Logging into the NoBox.AI Application**

This section is for users who already have a registered account and wish to log in to the NoBox.AI system to access available services and features.

### Steps to Log In

1.  Open the NoBox Desktop application.
2.  Select the server you wish to use.
3.  Enter the username and password associated with your account.
4.  Click the **Login** button.
5.  Wait for the authentication and data synchronization process.

Upon successful login, you will be directed to the application's main page.

### Post-Login Process

After successfully logging in, the system will:

*   Load your user account data.
*   Adjust the interface according to your access rights.
*   Synchronize data with the server.
*   Display the main dashboard.

---

## **Folder Structure After Installation**

Following a successful installation, the system will create two main folders as part of the application's configuration.

### Main System Folders

*   **NoboxChatManager**  
    Used to manage the chat system, agents, and service configurations.

*   **NoboxChatSession**  
    Used to store session data, activity logs, and communication history.

Ensure both folders are present for the application to function normally.

### Folder Functions

The existence of these folders is critical for:

*   Storing temporary data.
*   Maintaining system stability.
*   Saving user configurations.
*   Storing activity history.

## **Usage Tips**

If you encounter issues while logging in, consider the following:

**Account Checks**
*   Double-check your username and password for typos.
*   Ensure your account is active and has a valid subscription.
*   Confirm that your email has been verified.

**System Checks**
*   Ensure your internet connection is stable.
*   Use the latest version of the application.
*   Close and reopen the application if an error occurs.
*   Restart your computer if necessary.

**Additional Solutions**
*   Attempt to log in again after a few minutes.
*   Check the NoBox.AI server status.
*   Ensure no firewall is blocking the application.

If issues persist, please contact the support team.

## **Troubleshooting**

If you experience obstacles during login, follow these steps:

- Re-verify the email address and password you entered.
- Ensure your internet connection is stable.
- Refresh the login page or use incognito mode.
- Check for the verification email in your inbox or spam folder.
- Ensure your account is active and has the necessary access rights.
- Contact the administrator if you have forgotten your password or if your account is locked.

## **Frequently Asked Questions (FAQ)**

- **Cannot log in despite correct data**  
  Ensure there are no casing errors (uppercase/lowercase) in your username and password.

- **Application will not open**  
  Try reinstalling the application and ensure your operating system is compatible.

- **Forgot password**  
  Use the "Forgot Password" feature on the official NoBox.AI website to initiate a reset.
