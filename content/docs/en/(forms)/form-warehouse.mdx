## **Add Form Warehouse**

To create a form model data, please click the **[Form Warehouse] ➔ \[Add Form Warehouse]** menu. This will display the **[Add Form Warehouse]** dialog.

<ImageZoom
  src="/images/id/Tambah%20Form%20Warehouse.png"
  alt="Tampilan Dasbor NoBox"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

Complete the required fields, for example as follows:

| **Field** | Description | **Example** |
| --------- | ----------------------- | ------------- |
| Name | Form warehouse name | Milan |
| Phone | Mobile number | 0813102007221 |
| Country | Indonesia |
| State | Province | East Java |
| City | Malang |
| Zip Code | Postal Code | 65124 |
| Address | Jl. Akordion |

Once you have completed the fields, click **[Save]** to save the form model data you just created.

## **Edit and Delete Form Warehouse**

To edit a form model data, first select the data you want to edit. Edit your data and click **[Save]** to save your changes.

To delete a form model data, select the data you want to delete, then click **[Delete]. A confirmation dialog will appear asking if you want to delete the data. If you do, click **[Yes]**; otherwise, click **[No]**.
