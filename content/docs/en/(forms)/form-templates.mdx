## **Creating a New Form Template**

1. Navigate to **[Form Templates]** ➔ **[Add Form Template]**.
2. A comprehensive **[Add Form Template]** dialog will appear.

### **Configuration Form Fields**

Complete the required parameters for your new form; for example:

| **Field**          | Description                              | **Example**                   |
| ------------------ | ---------------------------------------- | ----------------------------- |
| Name               | Internal title of the form template      | Zeyaa Wholesale Warehouse     |
| Type               | Transactional logic (e.g., Order Form)   | Order Form                    |
| Product            | Linked items to be sold via the form     | -                             |
| Video URL          | Embedded video for engagement            | -                             |
| Tag                | Categorization labels                    | Wholesale, Fashion            |
| Layout             | Visual structure of the form             | -                             |
| Logo               | Branding image for the form header       | -                             |
| Title              | Public-facing form title                 | Azza Collection               |
| Description        | Visible context for the respondent       | -                             |
| Fields             | Input types (e.g., Name, Phone, Email)   | Name, Phone, Email            |
| Payment Methods    | Accepted transactional gateways          | COD, Bank Transfer, E-Payment |
| Submit Button      | Color customization                      | Blue                          |
| Post-Submit Button | Text displayed on the button             | Order Now                     |
| Success Action     | Behavior after successful submission     | Redirect to Success Page      |
| Follow-Up Actions  | Automated response message to purchaser  | -                             |
| Options            | Logic for delivery of follow-up messages | -                             |

Click **[Save]** to finalize and activate the new form template.

## **Managing Existing Form Data**

### Editing Form Templates

To modify a form, select the target entry from your list. Update the configuration and click **[Save]**.

Updates are deployed immediately; any active link or embedded form will reflect the changes in real-time.

### Deleting Form Templates

To remove a template, select the item and click **[Delete]**. A confirmation prompt will appear to prevent accidental data loss.

Click **[Yes]** to confirm or **[No]** to cancel.

<Callout type="info" title="Important Information">
  Form template deletion is permanent. Ensure that the specific form is no
  longer linked in any marketing materials or websites before removal.
</Callout>

## **Optimization Tips**

- **Full Configuration**: Ensure all required fields and descriptions are set before publishing to look professional.
- **Categorization**: Use Tags consistently to organize dozens of different forms for better management.
- **Verification**: Always test video URLs and follow-up links in Preview mode.
- **Design Check**: Use the Logo feature to build brand trust with your respondents.
- **Performance Audits**: Regularly check Statistics to see which forms have the highest conversion rates.
- **Workflow Speed**: Clone high-performing forms to launch new products or campaigns faster.

</Tips>

<Kendala>
## **Troubleshooting**

If you encounter issues while configuring or deploying your form templates:

- Refresh the dashboard if newly created templates fail to appear in your list.
- Verify internet stability during the save process.
- Confirm your account has "Manager" or "Supervisor" level access.
- Log out and back in if the Preview or QR generator seems unresponsive.
- Ensure all mandatory configuration fields (\*) are filled with valid data.
- Contact NoBox support if QR codes fail to redirect or statistical data goes missing.
