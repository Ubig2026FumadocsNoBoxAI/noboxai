The Coupons Form is a menu on NoBox that is used to create, manage, and organize coupons or promo codes that can be used in the system.

## **Add Form Coupons**

To create form model data, please click the **[Form Coupons] ➔ [Add Form Coupons]** menu. This will display the **[Add Form Coupons]** dialog box.

<ImageZoom
  src="/images/id/Tambah%20Form%20Coupons.png"
  alt="Tampilan Dasbor NoBox"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

Complete the required fields, for example as follows:

| **Fill-in** | Description | **Example** |
| ----------------- | -------------------- | ------------------------- |
| Name | Form discount name | New Year Discount |
| Code | Discount code | NewYear2026 |
| Status | Status | Active |
| Discount For | Discount for | Shipping |
| Discount Type | Discount type | Percentage |
| Discount Value | Discount amount | 100% |
| Has Zone | Region-specific | |
| Origin Type | Region of Origin | State |
| Origin State | Province of Origin | East Java |
| Destination Type | Destination Region | State |
| Destination State | Destination Province | East Java |
| Max Usage | Usage Limit | 100 |
| Products | Product Name | Pashmina Voal |
| Payments | Payment Method | Bank Central Asia, Paypal |
| New Customer |

Once you have completed the fields, click **[Save]** to save the form model data you just created.

## **Edit and Delete Form Warehouse**

To edit a form model data, first select the data you want to edit. Edit your data and click **[Save]** to save your changes.

To delete a form model data, select the data you want to delete, then click **[Delete]. A confirmation dialog will appear asking if you want to delete the data. If you do, click **[Yes]**; otherwise, click **[No]**.
