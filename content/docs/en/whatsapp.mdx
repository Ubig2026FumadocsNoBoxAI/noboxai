## **Connecting a WhatsApp Account**

To start using this feature, you must link your WhatsApp number to the NoBox Desktop system.

### **Account Addition Steps:**

1. Click the **[Account]** module ➔ **[Add Account]** button.
2. Select the **WhatsApp** channel and complete the account name and category details.
3. Click **[Save]**. Wait for the QR code to appear on the NoBox Desktop screen.
4. Scan the code using the Linked Devices feature in the WhatsApp application on your phone.

## **Messaging and Interaction Management**

Every message entering the connected WhatsApp number will be automatically synchronized into the NoBox.AI system.

### **Inbox**
Navigate to the **[Messages]** module ➔ **[Inbox]** to monitor all conversations. You can reply to customer messages directly from the website dashboard without needing to open your phone.

## **Using for Promotions**

You can use the active WhatsApp account as a sender for mass promotional campaigns.

1. Open the **[Promotions]** module ➔ Create a new promotion.
2. In the **Channel** section, select WhatsApp and choose the account you previously registered.
3. Compose the message content or use available templates, then run the sending process.
4. Monitor the sending status (sent, failed, read) through the **Send Messages** tab.

## **Editing and Deleting Account Data**

### **Edit Account Data**
You can change the account name or reference notes through the **[Account]** menu by selecting the data you want to change then clicking the **[Save]** button.

### **Delete Account Data**
To permanently disconnect an account, select the relevant account then click the **[Delete]** button.

<Callout type="info" title="Important Information">
Deleting an account will disconnect all WhatsApp integrations with the NoBox.AI system. All unsent message queue history for that account may be canceled.
</Callout>

## **Usage Tips**

Optimize your WhatsApp Automation performance with these tips:

- Ensure the internet connection on the computer running NoBox Desktop remains stable.
- Avoid sending mass messages with too high a frequency to prevent the risk of blocking from WhatsApp.
- Use **Random** or **Synonym** features in message templates to vary message content.
- Close other WhatsApp Web tabs while the account is being used by NoBox to maintain connection stability.
- Monitor your phone's battery status; even with the latest system, a dead phone can interrupt synchronization.

</Tips>

<Kendala>
## **Troubleshooting**

If you encounter issues with the WhatsApp service, please do the following:

- **QR Code Does Not Appear**: Ensure the **NoBoxChatManager** application is running in the background of your computer.
- **Messages Not Sent**: Check if the account status in NoBox Desktop is "Logout" or "Connecting". Re-scan if necessary.
- **Delayed Message Synchronization**: Refresh your browser page or check the RAM usage on your computer.
- **Automatic Disconnection**: Ensure you do not "Log out" from linked devices through the WhatsApp application on your phone.
- **Media (Images/Videos) Failed to Send**: Check attachment file sizes; ensure they do not exceed the limits set by WhatsApp (max 16MB).

</Kendala>
