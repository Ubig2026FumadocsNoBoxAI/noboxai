## **Preparation Before Installation**

Before starting the installation, ensure the following requirements are met to guarantee a smooth process:

- Your NoBox.AI account is registered and active.
- Your device is connected to the internet.
- You have sufficient storage space.
- The operating system is compatible with the application specifications.
- Administrator access rights are available (if required).

## **Downloading and Running the Application**

This section explains the process of downloading and launching the NoBox Desktop application.

### Steps for Manual Download and Installation

Follow these steps to download and run NoBox Desktop:

1.  Download the application via Google Drive at the following link:
    [NoBox Release](https://bit.ly/mynobox-release)

2.  Once the file is successfully downloaded, extract the folder.

3.  Open the extracted folder, then navigate to the following directory:
    **NoboxDesktop → Bin**

4.  Locate the NoBox Desktop application file.

5.  Double-click the file to launch the application.

Once the application is open, you can proceed to the login stage.

### Download Notes

- Ensure the download process reaches 100% completion.
- Avoid moving individual files before extraction is complete.
- Save the application folder in an easily accessible location.

## **Logging into NoBox Desktop**

To use NoBox Desktop, you must log in using the account credentials created on the official NoBox.AI website.

  src="/images/id/Login%20NoBox%20Desktop.png"

### Steps to Log In

1.  Launch the NoBox Desktop application.
2.  Enter your NoBox.AI Web account username and password.
3.  Ensure the data entered is correct.
4.  Click the **Login** button.
5.  Wait for the authentication process to complete.

If you do not have an account yet, please follow the registration guide on the following page:
[Account Registration](crm-user-registration)

### Post-Login Process

After successfully logging in, the system will:

- Load your user account data.
- Perform initial synchronization.
- Adjust application settings.
- Display the NoBox Desktop main page.

## **Important Usage Notes**

Please take note of the following conditions while using the application:

- The NoBox Desktop application can only be run on one computer at a time.
- Using it on more than one device simultaneously can cause data conflicts.
- Always log out properly before moving your session to another device.
- Do not share your account credentials with others.

## **Troubleshooting Installation Issues**

If you encounter obstacles during installation or login, consider the following:

**File and System Checks**
- Ensure the downloaded file is not corrupt.
- Use the latest version of the application.
- Confirm that your operating system is supported.
- Check for available storage capacity.

**Access and Security Checks**
- Run the application as an administrator if necessary.
- Temporarily disable antivirus software if blocking occurs.
- Ensure the firewall is not obstructing the application.

**Additional Solutions**
- Restart your computer after installation.
- Attempt to reinstall the application.
- Verify your internet connection.
