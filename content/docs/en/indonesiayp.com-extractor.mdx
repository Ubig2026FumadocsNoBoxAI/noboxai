**Indonesiayp.com Extractor** is a specialized feature in NoBox Desktop that allows you to perform bulk identity extraction of business profiles (Yellow Pages) from the Indonesiayp.com website. With this tool, you can rapidly compile business information such as categories, physical addresses, phone numbers, and regional locations across Indonesia.

This is an ideal solution for B2B marketing teams looking to build verified prospect databases directly from one of Indonesia's largest business directory sources.

## **Connecting an Extractor Account**

To begin extraction, you must register a new extractor account within the NoBox Desktop system.

### **Account Addition Steps:**

1. Launch the **NoBox Desktop** application ➔ navigate to the **[Extractor]** module ➔ click **[Add Account]**.
2. Select the **Indonesiayp.com** channel in the resulting dialog.
3. Complete the account name and category details for internal identification.
4. Click **[OK]** to save.

### **Initial Configuration**
Open the newly created account. Ensure the **[Logs]** tab displays the **"Ready to start"** status. This indicates that the system is calibrated to navigate the Indonesiayp site automatically.

## **Managing the Extraction Process**

### **Configuration Page (Page Process)**
Before triggering the data pull, define your search parameters to ensure high-quality results.

| Configuration | Detailed Function |
| :--- | :--- |
| **Searching** | Business keywords (e.g., "Bags", "Restaurant"). |
| **Category** | Selecting the relevant industrial category. |
| **Limit Result** | Restricting the number of business records to be pulled. |
| **Save To List** | Designating the target contact folder in the NoBox.AI cloud. |
| **Options** | Enabling debug mode or profile image loading. |

## **Synchronization and Data Export**

### **Real-Time Results (Page Result)**
Every successfully pulled record will appear in the Result tab. If the final count is lower than your **Limit Result**, it means the system has automatically filtered and discarded duplicate entries.

### **Cloud Synchronization**
All extracted data is automatically saved to the **Contacts** menu on the NoBox.AI dashboard. Use the **Source ➔ Indonesiayp.com** filter to isolate this data.

## **Advanced Features and Recovery**

### **Multi-Tab Execution**
You can run multiple simultaneous searches in different locations or categories using the Multi-Tab feature in NoBox Desktop.

### **Session Recovery (Auto Recovery)**
In the event of an unexpected application closure (computer power loss), NoBox will detect the interrupted process. Upon restart, a notification will appear offering to resume or cancel the previous extraction session.

## **Usage Tips**

Optimize your Indonesiayp Extractor workflow with these guidelines:

- Use broad but specific keywords (e.g., "Laundry" instead of "Laundry Shoe Wash") to maximize hit rates.
- Utilize the **Save to List** feature with timestamped folder names to keep your database organized.
- Enable **Auto Refresh** on the NoBox.AI web dashboard to monitor incoming data traffic in real-time.
- Perform periodic Excel exports for offline monitoring or local database backups.

## **Troubleshooting**

If you encounter issues with the extractor service, please follow these steps:

- **Error Messages in Logs**: Verify your internet connection or ensure that Venezuelayp.com is accessible manually through a standard browser.
- **Data Not Syncing to Web**: Ensure your extractor account is set to "Active" (check status in account management).
- **Process Halts Unexpectedly**: Check system RAM availability; deactivating other unused accounts can free up necessary resources.
- **Empty Result Sets**: Try modifying the category or expanding your location search radius on the map.
- **Application Lag**: Clear old activity logs or delete temporary system files on your OS.

</Kendala>
