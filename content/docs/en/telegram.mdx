## **Connecting a Telegram Account**

Linking a Telegram account requires several authentication steps using an active phone number.

### **Account Addition Steps:**

1. Navigate to the **[Account]** module ➔ click the **[Add Account]** button.
2. Select the **Telegram** channel and enter account details.
3. Enter the phone number in the **Client ID** field using the international format (Example: +628123456789).
4. Click **[Save]**.

## **Telegram API Configuration**

Unlike WhatsApp, Telegram requires the creation of an API Key so that external systems can communicate.

### **Enabling API Features:**
- Open settings on the active Telegram session in NoBox Desktop.
- Select **API Development Tools**. If you don't have an API yet, complete the **App Title** and **Short Name** fields as desired.
- Click **Create Application**. The system will automatically display your **api_id** and **api_hash**.

### **Save Session:**
- Click the **[Save Session]** button on the NoBox Desktop interface.
- Enter the verification code sent by the Telegram system to your main Telegram application.
- Once successful, the message "Save session Telegram Automation successfully" will appear in the logs tab.

## **Monitoring and Message Usage**

### **Reply via Inbox**
All messages received by your Telegram account will be synchronized into the **[Messages]** ➔ **[Inbox]** menu. You can monitor and reply to chats centrally from one dashboard.

### **Telegram Promotional Campaigns**
Use your Telegram account to send mass promotions by selecting the Telegram channel in the **Promotions** module. You can monitor delivery success through the **Send Messages** tab.

## **Editing and Deleting Account Data**

### **Edit Account Data**
You can change account notes or naming through the **[Account]** menu navigation on the NoBox.AI website. Select the data you want to adjust and click **[Save]**.

### **Delete Account Data**
Select the account you want to disconnect, click the **[Delete]** button, and confirm your action in the dialog that appears.

<Callout type="info" title="Important Information">
Deleting a Telegram account will permanently remove all authentication sessions (*session files*) from the NoBox.AI system. You will need to reconfigure from scratch if you wish to reconnect it.
</Callout>

<Tips>
## **Usage Tips**

Use Telegram Automation optimally with this guide:

- Ensure your Telegram account does not have **Two-Step Verification** enabled with a password, or ensure that password is entered when prompted.
- Use **API Development Tools** only in a secure NoBox environment to maintain your `api_id` confidentiality.
- Perform **Save Session** immediately after creating a new account so the connection is not lost when the application restarts.
- Utilize the **Inbox** feature to stay connected with customers without having to open your phone or native Telegram desktop application.

</Tips>

<Kendala>
## **Troubleshooting**

If you encounter issues in Telegram integration, please follow these steps:

- **Verification Code Does Not Appear**: Check official Telegram applications on your phone or PC; codes are usually sent via the "Telegram" system chat.
- **Fail to Create API Application**: Ensure the **App Title** contains unique characters and has not been used before.
- **Sessions Disconnect Frequently**: Check internet connection stability on the PC running NoBox Desktop or ensure your NoBox application version is up to date.
- **Broadcast Messages Fail**: Check if target numbers are still active or if your account is hit by mass messaging limits from Telegram policies.
- **Telegram Application Hangs**: Try doing a **Clear Session** and re-login using your phone number.

</Kendala>
