## **Editing and Deleting Quick Reply Data**

### Editing Quick Reply Data

To edit a specific quick reply, select the data you wish to modify first. Make your edits and click **[Save]** to store the changes.

Changes are saved immediately and will be ready for further use in conversations.

### Deleting Quick Reply Data

To delete a quick reply, select the item from the list and click **[Delete]**; a confirmation prompt will appear asking if you want to delete the data.

Click **[Yes]** to confirm or **[No]** to cancel.

<Callout type="info" title="Important Information">
Deleting quick reply data is permanent and cannot be undone. Ensure that the data to be deleted is no longer needed.
</Callout>

## **How to Use Quick Replies**

The Quick Reply feature can be used directly on the conversation page within the Inbox.

To send a quick reply, follow these steps:

1. Select the **[Messages ➔ Inbox]** module to view all incoming conversations. Then, select one of those messages.

2. In the message input field, press the **"/"** key on your keyboard.

<Tips>
## **Usage Tips**

To optimize the use of Quick Replies, consider the following tips:

- Create templates that address common customer needs.
- Periodically update your quick reply templates.
- Use command triggers that are easy for the team to remember.
- Avoid making messages excessively long.
- Tailor the content to the context of the conversation.

</Tips>

## **Troubleshooting**

If you experience issues while using the Quick Reply feature, you can try the following:
