## **Reports Menu View**

The Reports menu displays various types of statistical data and conversation activity that occurs within the NoBox.AI system.

On this page, users can monitor agent performance, message distribution, and the efficiency of customer service.

---

## **Agent Status**

### Agent Status Information

One of the submenus in NoBox.AI contains information regarding all agents who change their status to **online** or **offline**, as well as data on the last time an agent logged into the system.

This information is crucial for tracking agent attendance, their readiness to serve customers, and assisting supervisors in managing work schedules and task distribution more effectively.

## **Transfer Conversations**

### Transfer Conversation Information

This menu contains information related to the process of handing over or transferring messages from one agent to another.

This feature is typically used when the agent currently handling a conversation cannot continue the service, encounters an obstacle, or when handling by a more suitable agent is required.

## **Agent Conversations**

### Agent Conversation Information

This submenu displays a list of agents along with the number of conversations currently being handled and those that have been completed by each agent.

Data includes the number of incoming chats, chats still in progress, and chats resolved by each team member.

## **Agent Performance**

### Agent Performance Information

This menu contains data related to each agent's performance, such as the total number of successfully completed conversations, the number of sent messages, and the number of messages assigned to the agent.

Performance data is presented in summary and detailed formats according to specific time periods.

## **Tag Conversations**

### Tag Conversations Information

This submenu displays data on the use of conversation tags in the form of reports based on the specified day, month, and year.

Data is presented in graphs or tables showing the number of tags used in each time period.

This information can be filtered to meet specific user needs for easier analysis of conversation trends.

## **Usage Tips**

To maximize the use of the Reports menu, consider the following tips:

- Perform routine monitoring.
- Use report data as a basis for evaluation.
- Compare data across different time periods.
- Observe agent performance trends.
- Save analysis results for documentation purposes.

</Tips>
