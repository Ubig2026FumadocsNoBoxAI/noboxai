## **Adding a Tag**

To create a new tag, click the **[Message Tags] ➔ [Add Message Tag]** menu. A **[Add Message Tag]** dialog will appear; enter the tag name in this field.

**Add Message Tag View**

/>

In the Add Message Tag dialog, users can fill in the tag name according to their desired message category.

Once the information is complete, click **[Save]** to store the tag data.

## **Editing and Deleting Message Tag Data**

### Editing Message Tag Data

To edit a specific tag, select the data you want to modify first. Make the necessary changes and click **[Save]** to store them.

The changes will be saved immediately and will be ready for further use in the system.

### Deleting Message Tag Data

To delete tag data, select the tag you wish to remove and click **[Delete]**; a confirmation dialog will appear asking if you want to delete the data.

Click **[Yes]** to confirm or **[No]** to cancel.

<Callout type="info" title="Important Information">
Deleting message tag data is permanent and cannot be undone. Ensure that the data to be deleted is truly no longer needed.
</Callout>

## **How to Use Message Tags**

Message Tags can be applied directly on the conversation page to assist with grouping and monitoring messages.

Follow these steps to use Message Tags:

1. Open the **[Inbox]** menu and select a conversation.

2. Click the [ <img src="/images/id/information%20icon.png" alt="Information" className="inline h-5 w-auto mx-1" /> ] icon to display the details of that conversation.

3. Click on the Message Tags section; this will display previously created tags, or you can add a new one.

4. Select one or more tags for that conversation.

## **Usage Tips**

To optimize your use of Message Tags, consider the following tips:

- Use clear and consistent tag names.
- Avoid creating an excessive number of tags.
- Perform periodic evaluations of your tags.
- Align tags with your operational needs.
- Remove tags that are no longer in use.

</Tips>
