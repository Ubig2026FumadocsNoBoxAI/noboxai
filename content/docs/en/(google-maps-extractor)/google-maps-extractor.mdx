With the automated extraction system, you can harvest information such as business names, physical addresses, phone numbers, ratings, and industry categories in just minutes.

### **Configuration Page (Page Process)**

| Feature | Function Description |
| :--- | :--- |
| **Limit Result** | Restricting the number of records to be pulled to fit your quota. |
| **Save To List** | Directing extraction results to a specific contact group/list in NoBox.AI. |
| **Option Debug** | Displaying highly detailed system activity logs for auditing. |

## **Export and Data Recovery**

### **Export to Excel/JSON Formats**
You can download search results in `.xlsx`, `.csv`, or `.json` formats via the **Action** tab on the Results page for offline data processing.

### **Unexpected Shutdown Handling**
If the application is forced to close while a process is running, NoBox saves the last *checkpoint*. When the application is reopened, a notification will appear to resume the pending process.

<Tips>
## **Usage Tips**

Optimize your data extraction with these tips:

- Use the **Multi Tab** feature to run several different keyword searches simultaneously.
- Set varied (random) intervals so that activity appears more natural to Google's systems.
- Ensure you select the correct list in the **Save To List** section to avoid mixing data with older databases.
- Enable the **Auto Refresh** feature on the NoBox.AI website to monitor incoming data traffic live.

</Tips>

<Kendala>
## **Troubleshooting**

If you encounter technical issues, please perform the following:

- **Data Fails to Pull**: Check your internet connection or ensure your Google account does not require captcha verification.
- **Low Hit Count**: Try using more general keywords or expand the search radius on the Google Maps.
- **Application Closes Suddenly**: Check your computer's RAM availability and close other unnecessary applications.
- **Data Not Syncing to Web**: Ensure the account status in NoBox Desktop is "Active" and cloud synchronization is not blocked by a firewall.

</Kendala>
