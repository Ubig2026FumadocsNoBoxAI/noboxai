**Google Maps Extractor** can extract business prospect data (potential leads) from Google Maps with superior speed and quality. This software extracts a wide range of available data from Google Maps, including links, business names, addresses, phone numbers, websites, ratings, total views, opening hours, image links, latitude, longitude, plus codes, and business categories. Data is automatically saved to NoBox.AI. You can export results in Excel (.xlsx, .csv) and JSON (.json) formats.

## **Adding an Account**

To add a new Extractor, launch the **NoBox Desktop application ➔ [Add Account]**. The **[Add Account]** dialog will appear.

<ImageZoom
  src="/images/en/New%20Account%20Extractor.png"
  alt="NoBox Dashboard View"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

| Field   | Description                          | Example Data   |
| ------- | ------------------------------------ | -------------- |
| Channel | The selected platform channel        | Google Maps    |
| Name    | Your Account Name                    | Google Maps #1 |
| Notes   | Optional notes regarding the account | -              |
| Status  | Account activation status            | Active         |

Once you have completed the fields, click **[OK]** to save the new account.

## **Basic Operations**

Before proceeding to scrape data, you must understand the basic operations and features of Google Maps Extractor:

<ImageZoom
  src="/images/en/Dasar.png"
  alt="NoBox Dashboard View"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

- Launch the NoBox Desktop application.
- Enter your search query in the **[Search Google Maps]** field.
- Navigate to the **[Process]** tab.
- Enter a **[Limit Result]** to determine the total number of records to be harvested.
- Select a target list to store the extraction results in **[Save to List]** or create a new list.
- Select **[Options]** to enable debug mode or product image loading.

| Field        | Description                                | Example Data  |
| ------------ | ------------------------------------------ | ------------- |
| Limit Result | Specify the number of records to be pulled | 5             |
| Save to list | Select/add a list to store harvested data  | Laptop Malang |
| Options      | Check for debugging and image loading      | -             |

## **Scraping Data**

Scraping Google Maps data allows you to extract information from business owners or advertisers across various product and service categories with speed, ease, and quality.

<ImageZoom
  src="/images/en/Scrape%20data.png"
  alt="NoBox Dashboard View"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

- Once the configuration is complete, click **[Start]** to begin data extraction.
- Wait for the notification indicating the extraction process is finished.
- If the harvested data count is less than the specified limit, it is likely that duplicate data from the same source was filtered out.
- Extracted data is automatically saved to the NoBox.AI **Contacts** menu.

The Google Maps feature provides several specific filters to refine your search:

### **Filter By Category**

You can filter data based on specific business categories:

- Open the Google Maps channel and select the category to search.

## **Editing and Deleting Account Data**

### **Editing Account Data**

To edit an account, log in to the [id.NoBox.AI](https://id.NoBox.AI/) website and navigate to the **[Accounts]** menu. Select the target account, adjust the data, and click **[Save]**. Changes are saved immediately and reflected in the system.

### **Deleting Account Data**

To delete an account, select the target record and click **[Delete]**. Confirm the action in the resulting dialog.

<Callout type="info" title="Important Information">
  Account deletion is permanent and cannot be undone. Ensure the data being
  deleted is no longer required.
</Callout>

<Tips>
## **Usage Tips**

Maximize your Google Maps scraping results with these tips:

- Set a realistic **Limit Result** based on your distribution requirements.
- Leverage Opening Hours and Rating filters to obtain higher quality business leads.
- Use specific naming conventions for lists (e.g., "Pharmacy - South Jakarta") for easier database management.
- Enable **Load Images** if you require visual location verification.
- Periodically clear logs to maintain optimal NoBox Desktop performance.

</Tips>

<Kendala>
## **Troubleshooting**

If you encounter issues during scraping, check the following:

- If results are lower than the limit, the system likely filtered out duplicate data.
- Ensure the map is focused on the correct area before clicking **[Start]**.
- If the application lags, consider reducing the limit or disabling the **Load Images** option.
- Verify account status on the NoBox.AI website if data fails to synchronize.
- Restart the NoBox Desktop application if it freezes during a search.

</Kendala>
