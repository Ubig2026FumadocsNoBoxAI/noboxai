With a user-friendly interface, you can efficiently perform category and location-based searches across all regions of Indonesia.

## **Primary OLX Extractor Features**

The software is equipped with various intelligent features to enhance speed and security during data extraction.

### **1. Auto Login**

OLX Extractor supports an automated login feature. By saving your OLX account credentials, the system can access the platform without repeatedly typing passwords, facilitating easier access to advertisement data that requires authentication.

## **Configuration Management (Page Process)**

Before initiating the extraction process, you must define search parameters in the **Process** tab.

| Feature          | Function Description                                                              |
| :--------------- | :-------------------------------------------------------------------------------- |
| **Limit Result** | Determining the maximum number of advertisement records to be harvested.          |
| **Save To List** | Selecting the contact category or group to store results on the NoBox.AI website. |
| **Option Debug** | Enabling deep technical logging to monitor system performance.                    |
| **Load Images**  | Defining whether to load advertisement images to manage system resources.         |

<Tips>
## **Usage Tips**

Optimize your OLX Extractor performance with these steps:

- Always utilize the **Auto Login** feature for smoother daily operations.
- Leverage **Multi-Tab** functionality if you need to research competitors across different regions simultaneously (e.g., Jakarta, Surabaya, Bandung).
- Ensure you select a specific **Save To List** file based on product categories to keep your database organized.
- Enable **Auto Refresh** on the NoBox.AI website to monitor incoming results without manual refreshes.
- Perform regular data exports for weekly or monthly market analysis.

</Tips>

## **Troubleshooting**

If you encounter issues while using the extractor, please perform the following:

- **Login Failed**: Verify email and password accuracy in the **Other Options** menu and check your internet connection.
- **Process Interrupted**: In the event of a power loss, reopen the application and click **[OK]** in the process recovery dialog.
- **No Data Appearing**: Check the **Logs** tab to see if errors occurred while accessing the OLX page.
- **Limit Reached**: If extraction stops, check your **Limit Result** settings or your subscription quota.
- **Sync Issues**: Ensure the extractor account status in NoBox Desktop is set to "Active".

</Kendala>
