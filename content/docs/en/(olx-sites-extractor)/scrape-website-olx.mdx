OLX IndonesiaX is a desktop application powered by Chrome browser integration that allows for the high-quality extraction of business prospect data (potential leads) from various countries within the OLX network.

This software extracts a comprehensive range of data from OLX, including links, business names, addresses, phone numbers, websites, ratings, total views, opening hours, image links, latitude, longitude, plus codes, and business categories. Data is automatically saved to NoBox.AI. You can also export results in Excel (.xlsx, .csv) and JSON (.json) formats.

## **Adding an Account**

Ensure you have a subscription to the OLX Extractor for the specific country you wish to target. To add a new tool instance, launch the **NoBox Desktop application ➔ [Add Account]**. The **[Add Account]** dialog will appear.

<ImageZoom
  src="/images/en/New%20Acccount%20OLX.png"
  alt="NoBox Dashboard View"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4"
/>

| Field   | Description                          | Example Data      |
| ------- | ------------------------------------ | ----------------- |
| Channel | The selected platform channel        | OLX IndonesiaX    |
| Name    | Your Account Name                    | OLX IndonesiaX #1 |
| Notes   | Optional notes regarding the account | -                 |
| Status  | Account activation status            | Active            |

Once the fields are complete, click **[OK]** to save the new account.

## **Editing and Deleting Accounts**

### **Editing an Account**

To edit an account, log in to the [id.nobox.ai](https://id.nobox.ai/) website and navigate to the **[Accounts]** menu. Select the target account, adjust the information, and click **[Save]**.

### **Deleting an Account**

To delete an account, select the target record and click **[Delete]**. Confirm the action in the resulting notification dialog.

## **Configuration**

OLX supports an automated login feature, allowing you to bypass manual password entry. To configure this, click **[Other Options]**. In the resulting **[Option]** dialog, enter your OLX account username and password.

## **Basic Operations**

After configuring auto-login, you can proceed to basic operations:

- Enter your search query.

- Choose a target list in **[Save to List]** or create a new one.

## **Scraping Data**

Scraping the OLX website allows you to efficiently harvest contact information from sellers or advertisers across diverse product categories.

- Once configuration is complete, click **[Start]** to begin extraction.
- Wait for the completion notification.
- If fewer results are returned than requested, the system likely filtered out duplicates from the same poster.
- Data is automatically synchronized to the NoBox.AI **[Contacts]** menu.

<Tips>
## **Usage Tips**

Tips for streamlining the data scanning process on OLX:

- Ensure you define a specific location before starting to improve data accuracy.
- Use the most relevant categories for your research targets.
- Set **Limit Result** according to your daily data needs to maintain stability.
- Only enable **Load Images** if visual data is necessary, to minimize memory consumption.
- Organize extraction results using the 'Save to List' feature to keep projects distinct.

</Tips>

<Kendala>
## **Troubleshooting**

If you encounter issues while scraping OLX:

- If the process fails to start, verify that **Auto Login** credentials are correct.
- If the Chrome browser instance experiences an error, restart the application.
- Fewer results than the limit often indicate that the system is filtering ads from the same owner.
- Check your internet connection if real-time synchronization to NoBox.AI fails.
- Restart NoBox Desktop if you encounter synchronization issues between multiple extractor tabs.

</Kendala>
