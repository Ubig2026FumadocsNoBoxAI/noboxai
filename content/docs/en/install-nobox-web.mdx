This page serves as a complete technical guide for installing and configuring **NoBox Web**, including the deployment of the Chat Manager and Chat Session modules.

Our goal is to assist you through every setup phase to ensure your system is fully operational and ready to support your business communication needs.

---

## **Preparation Before Installation**

Before starting the installation process, ensure the following requirements are met to guarantee a smooth experience:

- Your NoBox.AI account is registered and active.
- Your subscription package is still valid.
- Your device is connected to the internet.
- You have sufficient storage space.
- Account login details are ready.
- Administrator access rights are available if required.

---

## **Downloading the NoBox Installer**

This section explains the process of downloading the installer from the official NoBox website.

### Steps to Download the Installer:

1.  Visit the official NoBox website and log in to your account.
2.  Navigate to the **Subscription** menu.
3.  Select your active service package.
4.  Download the installer assigned to your computer's operating system.

## **Application Installation Process**

After the installer file has been successfully downloaded, follow the installation process as described below.

### Installation Steps:

1.  Run the downloaded installer file.
2.  Select the server you wish to use.
3.  Enter your NoBox.AI account username and password.
4.  Follow the on-screen installation instructions.
5.  Specify the installation location according to your needs.

- Check that the installation process concluded without errors.

- Attempt to launch the main application.

## **Installation Folder Structure**

If the installation is successful, the following two folders will be created:

- **NoboxChatManager**  
  Used to manage chat integration, agents, and automation.

- **NoboxChatSession**  
  Used to manage account login sessions and authentication.

Ensure both folders are present for the application to function normally.

### System Folder Functions:

These folders serve to:

- Store system configurations.
- Save session data.
- Maintain application stability.
- Support the automation integration process.

---

## **NoBox Chat Manager**

NoBox Chat Manager is used to integrate web chat, social media, and instant messaging into a single centralized system.

This application simplifies the management of customer interactions and business leads.

### Main Functions of Chat Manager:

- Manages customer service agents.
- Connects automation accounts.
- Monitors chat activity.
- Manages incoming and outgoing messages.

### How to Run NoBox Chat Manager:

1.  Open the NoBox installation folder.
2.  Enter the **NoboxChatManager** folder.
3.  Open the **Bin** folder.
4.  Run the **NoboxChatManager.exe** file.

## **NoBox Chat Session**

NoBox Chat Session is used to manage the login processes of various platforms, such as WhatsApp, Telegram, Shopee, Tokopedia, and others.

This application functions to maintain the stability of account sessions so they remain active.

### Main Functions of Chat Session:

- Keeps login sessions active.
- Prevents automatic logouts.
- Stabilizes account connections.
- Manages platform authentication.

### How to Run NoBox Chat Session:

1.  Open the NoBox installation folder.
2.  Enter the **NoboxChatSession** folder.
3.  Open the **Bin** folder.
4.  Run the **NoboxChatSession.exe** file.

## **Troubleshooting Installation Issues**

If you encounter obstacles during installation or usage, follow these steps:

**Basic Checks**
- Verify your internet connection.
- Ensure antivirus software is not blocking the application.
- Confirm that the correct server has been selected.

**System Checks**
- Run the application as an administrator.
- Check available storage capacity.
- Ensure the operating system is compatible.

**Additional Solutions**
- Restart your computer.
- Reinstall the application.
- Update the application to the latest version.
- Contact the support team if errors persist.
