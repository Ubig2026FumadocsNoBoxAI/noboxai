Links is a diagnostic and operational module in NoBox.AI used to manage the various communication channel accounts connected to your customer contact data.

This menu helps users view all external links—such as WhatsApp numbers, Email addresses, Telegram usernames, and other platform identifiers—in a centralized hub. Every registered link enables the system to route messages, track deep conversation histories, and monitor the delivery status of broadcast campaigns directed at that specific account.

## **Adding a Functional Link**

To integrate a new communication pathway for a contact:

1. Navigate to the **[Contacts]** module ➔ **[Links]** menu ➔ click **[Add Link]**.
2. Complete the technical fields in the **[Add Link]** configuration dialog.

### **Link Entry Field Definitions**

| Field           | Description                                                                                                                                                 | Example        |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------- |
| Channel         | The target communication platform (WhatsApp, Email, Telegram, Shopee, etc.).                                                                                | Email          |
| External Name   | The profile name or customer username as it appears on the target platform.                                                                                 | Sasi Biz Email |
| External Id     | The absolute identifier required for delivery. (e.g., phone number with country code, or a full email address). This must be accurate for message delivery. | sasi@gmail.com |
| Conversation Id | The technical ID linked to ongoing chat threads. Usually populated automatically by the system upon the first interaction.                                  | -              |
| Profile Photo   | URL for the contact's platform-specific profile image (optional).                                                                                           | -              |
| Additional Info | Internal comments or qualitative context regarding this specific communication link.                                                                        | -              |

Click **[Save]** to finalize the link integration.

## **Managing Link Records**

### Editing Links

To update a username or correct an ID typo, select the target record and modify the form entries. Click **[Save]** to apply updates. Changes are immediately active for all system-wide features using that link.

### Deleting Links

To remove a gateway, select the link and click **[Delete]**. A verification prompt will appear. Click **[Yes]** to confirm.

<Callout type="info" title="Important Information">
  Deleting a link disconnects NoBox.AI from that specific communication channel
  for the contact. Ensure the record is truly obsolete before finalizing
  deletion.
</Callout>

## **Deep Conversation Tracking**

The **Conversations** tab within the link profile houses the complete historical logging of every interaction processed through that specific gateway.

This tab includes chat logs across all statuses (New, Assigned, Done). All non-archived conversations remain monitored and connected to the link record, allowing for full relationship audit trails.

- Monitor the current active assignment and status of the thread.
- Identify "New" or unhandled interactions that require immediate team attention.

## **Broadcast Success Monitoring**

The **Send Message** tab provides specialized telemetry for marketing broadcasts directed at this specific communication link.

Users can audit the outcome of mass messaging campaigns, identifying which specific attempts reached the recipient, which are pending, and which encountered delivery errors.

<Tips>
## **Operational Best Practices**

- **Multi-Channel Redundancy**: Map multiple links (e.g., WhatsApp + Email) to a single contact to ensure you can reach them even if one platform is offline.
- **ID Precision**: Use the absolute international format for phone numbers (e.g., +62... for Indonesia) to avoid "Failed" statuses during broadcasts.
- **Interaction Audits**: Regularly check the Conversations tab to identify high-interest leads who frequently engage with your content.
- **Failure Analysis**: Use the Send Message tab to troubleshoot habits; if a contact consistently fails on WhatsApp but succeeds on Email, prioritize the latter.
- **Data Hygiene**: Update link identifiers immediately if a customer reports a change in their primary contact number or email address.

</Tips>

<Kendala>
## **Troubleshooting**

- **Persistent Broadcast Failures**: Re-verify the "External ID" formatting (specifically country codes for mobile numbers).
- **Stagnant Histori**: Click your browser's refresh button if the Conversations or Send Message tabs fail to display recent activities.
- **Latency Errors**: Ensure your local internet is stable when modifying master link records to avoid data conflicts.
- **Account Restrictions**: If you cannot create or remove links, verify your user role permissions for "Supervisor" access.
- **Synchronization Gaps**: Log out and log back in if your session has timed out during a high-volume data update.
- **Broken Media**: Clear browser cache if profile photos fail to render or if the interface layout appears misaligned.

</Kendala>
