**Contacts** is a primary module in NoBox.AI designed to store, manage, and organize all customer contact data acquired from various sources.

This menu helps users handle promotion targets, store customer information in a structured manner, and support CRM needs such as task management, deals, and interaction notes. With the available features, users can perform bulk data imports, add contacts manually, and export data to external platforms like Google Contacts and Facebook Audiences.

## **How to Import Excel & CSV Files**

This feature allows for mass data migration from external spreadsheet files into your NoBox.AI environment.

1. Navigate to the **[Contacts]** menu ➔ click the **[Import Excel & CSV]** tool.

<ImageZoom
  src="/images/en/impor%20excel%20&#x26;%20csv.png"
  alt="Bulk Import Tool Integration"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

2. Click **[Download Model]** first to obtain the correctly formatted template, or upload your existing Excel/CSV file. Click **[Select File]** ➔ choose your file, then click **[Import]**.

<ImageZoom
  src="/images/en/Impor%20data%20excel&#x26;csv.png"
  alt="File Selection and Upload Process"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

3. The **[Import Excel & CSV]** configuration dialog will appear. Complete the fields for channel, list, country, category, and tags as required for your campaign.

<ImageZoom
  src="/images/en/hasil%20impor%20excel&#x26;csv.png"
  alt="Import Configuration and Metadata Dialog"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

4. Map your Excel columns to the corresponding NoBox.AI fields. Mapping ensures that data from your file is placed accurately within the system.

**Critical Step**: Ensure you map the **[External ID]** field for unique identifiers like phone numbers or emails to prevent duplication. Click **[Save]** to finalize the import.

## **Communication Links**

The **Links** tab is used to house multiple communication identifiers connected to a single contact, such as WhatsApp numbers, Email addresses, and Telegram IDs.

By connecting one contact to multiple channels, you ensure that marketing broadcasts can be delivered across various platforms from a integrated historical record.

### Adding a New Link

1. Click **[Add Link]** to open the configuration dialog.
2. Complete the technical details for the new channel.

<ImageZoom
  src="/images/en/Tambah%20Link.png"
  alt="Adding a Communication Gateway"
  width={800}

  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "

### **Link Configuration Fields**

| Field             | Description                                                                                       | Example           |
| ----------------- | ------------------------------------------------------------------------------------------------- | ----------------- |
| Channel           | The target communication platform (e.g., Telegram).                                               | Email             |
| External Name     | The profile name or username displayed on that specific platform.                                  | Sasi Biz Email    |
| External Id       | The unique ID (Phone, Email, Shopee ID) required for delivery. This must be accurate.               | sasi@gmail.com    |
| Conversation Id   | Optional ID linked to a specific historical chat thread.                                          | -                 |
| Profile Photo     | URL or upload for the contact's platform profile picture.                                          | -                 |
| Additional Info   | Internal comments specific to this communication link.                                            | -                 |

Click **[Save]** to finalize the link integration.

### Managing Links

#### Editing Links
To update a username or ID, select the link in the table and click **Save** after making changes. Updates reflect across the system immediately.

#### Deleting Links
Select the obsolete link and click **[Delete]**. Confirm the action in the prompt.

<Callout type="info" title="Important Information">
Link deletion is permanent. Ensure the channel is no longer required for promotional activities before deletion.
</Callout>

<Tips>
## **Workflow Strategies**

- **Bulk Efficiency**: Always use the **[Import Excel/CSV]** tool for large database migrations. Review the model template first to ensure data alignment.
- **Data Integrity**: Prioritize mapping the **[External ID]** (Phone/Email) correctly to prevent accidental account duplication.
- **Dynamic Sharing**: Use the **[vCard]** feature to rapidly transfer leads to mobile devices for on-the-go communication.
- **Clean Database**: Perform a **[Merge]** audit monthly to keep your customer lists compact and efficient.
- **Task Synergy**: Connect **[Tasks]** and **[Deals]** to contacts to create a clear operational map of your sales efforts.
- **Contextual Knowledge**: Leave detailed internal notes after significant phone calls to help other team members provide better service.

</Tips>

<Kendala>
## **Troubleshooting**

- **Import Errors**: If bulk upload fails, verify your file strictly adheres to the downloaded model template's column headers.
- **Duplicate Data**: If duplicates occur, check if the "External ID" was correctly mapped during the last import session.
- **UI Lag**: Refresh your browser or toggle the "Auto-Refresh" switch if manual additions don't appear in the list instantly.
- **Restricted Actions**: If "Edit" or "Delete" buttons are inactive, verify that your account role has the necessary permissions for data management.
- **External Sync Issues**: Maintain a stable internet connection when syncing with Google or Facebook platforms.
- **Broken Links**: If WhatsApp or Email "Links" fail to open, check the account authentication status in your Business Profile settings.

</Kendala>
