## **Feature Breakdown**

### 1. Conversation Feature

This feature is used to monitor communication activity with customers through annual statistics and charts.

This data helps users evaluate the performance of their services.

### 2. Contact Feature

The Contact feature displays the number and location of customers within a map.

This information is useful for understanding customer distribution geographically.

### 3. List Feature

The List feature functions to display the total number of lists stored within the system.

This data serves as a reference for information management.

### 4. Send Message Feature

The Send Message feature presents statistics on message delivery within a specific period.

Users can monitor the effectiveness of their communication through this feature.

### 5. Task Feature

The Task feature is used to manage daily work activities.

Users can add, complete, or delete tasks according to their needs.

## **Troubleshooting**

If you encounter issues while using the Dashboard, follow these steps:

- Refresh the system page if data does not update automatically.
- Ensure your internet connection is stable.
- Double-check any active date or category filters.
- Log out and log back in if features are not functioning normally.
- Contact the support team if charts or statistics fail to appear.
