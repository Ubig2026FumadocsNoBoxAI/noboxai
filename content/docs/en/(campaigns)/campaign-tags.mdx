**Campaign Tags** (Tag Promosi) is a dedicated organizational module in NoBox.AI used to apply labels or categories to every marketing initiative. By utilizing tags, you can group campaigns based on specific themes, timeframes, or assigned departments.

This feature significantly improves data retrieval, segmentation, and long-term campaign performance analysis.

## **Adding a Campaign Tag**

To create a new organizational label, follow these steps:

1. Navigate to the **[Campaigns]** module ➔ **[Campaign Tags]** menu.
2. Click the **[Add Campaign Tag]** button.
3. Enter the desired tag name in the dialog box.

### Add Tag Dialog View

---

## **Managing Tag Records**

### Editing Tags

Select the tag you wish to rename, enter the new title, and click **[Save]**. This update will automatically reflect across all historical and active campaigns carrying that specific tag.

### Deleting Tags

To remove an obsolete label, select the tag and click **[Delete]**, then confirm the action. Removing a tag detaches the label from all linked campaigns but does not delete the actual campaign records themselves.

<Callout type="info" title="Important Information">
  Tag deletion is permanent and cannot be reversed. Ensure the tag is no longer
  required for any reporting or filtering before confirming deletion.
</Callout>

## **Applying Tags to Your Campaigns**

Once created, tags can be applied during the creation or editing phase of any campaign:

1. Open the **[Campaigns]** dashboard and select the target mission.
2. In the **Tag** field, select one or more relevant labels from the dropdown list.
3. Click **[Save]** to apply the configuration.

## **Advanced Filtering by Tag**

Efficiency is the primary goal of tagging. Filter your campaign list to isolate specific marketing segments:

1. Open the **[Campaigns]** menu.
2. Click the **Tag Filter** button at the top of the campaign table.
3. Select the tags you wish to view. The system will refresh the list automatically to show only relevant results.

<Tips>
## **Organization Best Practices**

- **Naming Conventions**: Maintain a consistent system (e.g., "Event_Summer", "Client_Tier1", "Promo_Flash").
- **Granular Labeling**: Don't hesitate to apply 2-3 tags to a single campaign for multi-dimensional filtering (e.g., time + theme + team).
- **Performance Benchmarking**: Use tags to track performance variations between different marketing divisions.
- **Hygiene**: Delete obsolete tags once a specific year or event is over to keep your filtering menus clean and relevant.

</Tips>

<Kendala>
## **Troubleshooting**

- **Target Tag Not Visible**: Ensure the tag has been successfully saved in the master Campaign Tags menu before attempting to apply it.
- **Permission Denied**: Modifying master tags usually requires "Supervisor" level access.
- **Filtered List is Empty**: Check if other active filters (like Date Range or Channel) are conflicting with your tag selection.
- **Synchronization Delay**: If a renamed tag doesn't show up immediately, refresh the page to force a data update.
- **Interface Overflow**: Avoid extremely long tag names; use clear abbreviations to keep the dashboard tables readable.

</Kendala>
