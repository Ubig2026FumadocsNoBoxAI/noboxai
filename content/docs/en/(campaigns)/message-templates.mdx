**Message Templates (Message Models)** is a feature in NoBox.AI that allows you to save the design and content of promotional messages for future reuse. By using templates, you don't need to retype the message content every time you want to create a new campaign, making the marketing process much faster and more consistent.

These templates can include text, personalization variables, and media attachments such as images or documents.

## **How to Create a New Message Template**

1. Open the **[Promotions]** module and select the **[Message Templates]** menu.
2. Click the **[+ Create Message Model]** button.
3. A form dialog will appear for you to fill in the template details.

### Template Form Fields

Complete the following fields to create an effective template:

| Input Field | Description | Example |
| :--- | :--- | :--- |
| **Name** | A unique name to identify this template internally. | RAMADAN Order Confirmation |
| **Channel** | Select the communication channel supported by this template. | WhatsApp |
| **Subject** | The subject or topic of the message (especially for email). | Payment Confirmation |
| **Message Format** | Determine whether to use Text or Media format. | Text / Media |
| **Message** | The main message content you want to convey. | Hello [Name], thank you for ordering... |
| **Attachment** | Upload supporting files such as catalogs or invoices (if any). | promo_catalog.pdf |

Once completed, click **[Save]** to store the data.

---

## **Using Templates in Promotions**

Once a template is created, you can apply it when creating a new promotional campaign:

1. In the **[Promotions]** module, create or edit a campaign.
2. Click the **[Template]** button or select a message model from the dropdown.
3. Choose the desired message model from the list that appears.
4. Click **[Use Model]**. All message fields will be automatically filled according to the template.

## **Editing and Deleting Templates**

- **Edit**: Select a template from the list, click the edit icon (pencil) or the options button, update the fields, and click **[Save]**. Changes to a template will not affect *currently running* campaigns but will apply to *future* campaign use.
- **Delete**: Select the template you wish to remove, click the delete icon (trash can), and confirm by clicking **[Yes]**.

<Tips>
## **Tips for Creating Engaging Templates**

- **Use Variables**: Insert placeholders like `[Name]` to make messages feel more personal to customers.
- **Concise Content**: Ensure the core message is at the beginning so it is immediately read by the customer.
- **High-Quality Media**: If using the Media format, ensure attached images or videos have good resolution but reasonable file sizes.
- **Categorize Names**: Use name prefixes like `[PROMO]`, `[INFO]`, or `[CS]` to keep the template list organized.
</Tips>

<Kendala>
## **Troubleshooting**

- **Attachment Failed to Upload**: Ensure the file format is supported (PNG, JPG, PDF) and its size doesn't exceed the system limit.
- **Template Doesn't Appear When Selected**: Ensure the **Channel** selected in the template matches the **Channel** you are using for the promotional campaign.
- **Variables Not Working**: Double-check the variable syntax; ensure it matches the tags available in NoBox so contact data is correctly retrieved.
</Kendala>
