The **Direct Messages** feature in NoBox.AI allows you to send messages instantly to one or more specific contacts without having to go through the complex process of creating a full promotional campaign. This feature is ideal for personal communication, quick follow-ups, or sending urgent information to selected customers.

Even though sent directly, you can still monitor the delivery status of these messages through the monitoring menu to ensure the message has reached its destination.

## **Steps to Send a Direct Message**

To send a direct message, follow these simple steps:

1. Open the **[Promotions]** menu ➔ select **[Direct Messages]**.
2. Fill in the available form fields as needed.
3. Click the **[Submit Now]** button to execute the delivery.

### Field Input Guide

Here is the detailed information you need to complete:

| Field | Function Description | Example |
| :--- | :--- | :--- |
| **Contact** | Select one or more contacts from your contact list. | John Doe, Jane Smith |
| **Channel** | Select the communication channel you wish to use. | WhatsApp |
| **Account** | Choose a sender account already connected in NoBox. | Global Admin CS |
| **Message Format** | Determine if the message is Text or Media. | Text |
| **Short Link** | Enable the link shortening feature to track clicks (if there is a link). | Active |
| **Message** | Write the content of the message you want to convey. | Hello, your order is ready for shipment... |
| **Attachment** | Upload a file or image as a message supplement. | payment_receipt.jpg |

<Tips>
## **Direct Messaging Tips**

- **Use Short Links**: If you insert a link (e.g., payment link or catalog), always enable the short link feature so you can track whether the customer clicked it or not.
- **Personal Messages**: Since this is a direct message, use more personal language and address the customer by name to increase their comfort.
- **Check Contact Status**: Ensure the contacts you select have valid numbers on the chosen channel (e.g., WhatsApp numbers must have an active WhatsApp account).
</Tips>

---

<Kendala>
## **Troubleshooting**

- **Message Stuck**: If the delivery status does not change for a long time, ensure your NoBox Desktop application is active (online).
- **Failed to Send to Many Contacts**: Sending direct messages to too many contacts at once through this feature is not recommended if the content is identical (bulk). For bulk messages, use the **Campaign** feature for safer delivery.
- **Account Selection Error**: If a message fails to send, check if the channel account is still correctly linked in NoBox.
</Kendala>
