**Contact Tool** (Tool Contact) is a centralized administrative module in NoBox.AI designed for high-volume customer data management within the settings environment.

This menu allows authorized users to manually create new contact records, perform bulk imports from external sources, execute complex database searches, and purge obsolete contact data. Through the Contact Tool, your entire customer database can be audited and refined to support CRM outreach, messaging workflows, and promotional campaigns across the NoBox.AI platform.

## **Manual Contact Creation**

This feature is used to add individual new customer records directly into the system database.

To create a new entry:

1. Click the **[Add Contact]** button on the Contact Tool dashboard.
2. Complete all required fields in the profile form that appears.
3. Click **[Save]** to commit the record to your CRM.

## **Contact Form Fields**

The contact profile is divided into several logical categories. Here is a breakdown of the metadata fields:

| Category | Field         | Description                                                                                                                                      |
| -------- | ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| General  | Name          | The primary identifier for the contact. This is a **mandatory** field.                                                                           |
|          | Category      | High-level classification (e.g., Individual, Corporate, Reseller) to help organize data by customer type.                                       |
|          | List          | Organizational segments used for targeting marketing campaigns. Contacts can belong to multiple lists simultaneously.                              |
|          | Tag           | Custom labels for rapid identification and filtering (e.g., VIP, Hot Lead).                                                                      |
|          | Contact Person| The primary representative or point of contact, particularly useful for corporate accounts.                                                       |
|          | Notes         | Internal qualitative information or administrative reminders regarding the customer.                                                            |
| Channel  | Channel       | The external communication platform (WhatsApp, Telegram, etc.) associated with this record.                                                       |
|          | External Name | The customer's profile name as it appears on the target channel.                                                                                 |
|          | External ID   | The unique technical identifier (phone number or email) used to route messages correctly.                                                        |
| Address  | Address       | The physical street address of the customer.                                                                                                     |
|          | Geography     | Country, Province/State, and City details for regional reporting.                                                                                |
|          | Postcode      | The postal code for shipping validation.                                                                                                         |
|          | Coordinates   | Latitude and Longitude data for precise geographic mapping.                                                                                      |
| Status   | Blocked       | A toggle to mark a contact as "Active" or "Blocked." Blocked contacts will be automatically excluded from all promotional broadcasts. |

## **Bulk Contact Ingestion**

The **Import** feature allows for massive database expansions through various integrated methodologies:

| Method             | Workflow Description                                                                                             |
| ------------------ | ---------------------------------------------------------------------------------------------------------------- |
| Excel & CSV        | Upload structured spreadsheet files using the system-provided template to ensure correct column mapping.        |
| Google Contacts    | Synchronize your existing Google contact lists directly into the NoBox.AI ecosystem.                              |
| Facebook Audiences | Import target audience segments directly from the Facebook Ads manager for localized retargeting.               |
| WhatsApp Import    | Ingest contacts directly from active conversation threads on your connected WhatsApp business accounts.          |

**Note**: Always use the provided templates to ensure data integrity during the acquisition process.

## **Advanced Filtering & Search**

Database searches can be narrowed down to specific sub-segments to help you find precisely the contacts you need for immediate outreach.

**You can filter the database by:**
- **Category** — Filter by customer type classification.
- **Geography** — Isolate contacts in specific Cities, States, or Nations.
- **Data Origin** — Filtering by the platform/source of the contact record.
- **Segments** — Filtering by associated Lists and custom Tags.
- **Temporal Data** — Isolate contacts created during specific months or campaigns.

Click **[Filter]** to apply your logic, or **[Reset]** to restore the full database view.

---

## **Database Maintenance (Deletion)**

To remove obsolete or incorrect records:

1. Select one or more entries from the master contact list.
2. Click the **[Delete Selected]** button.
3. Confirm the permanent removal in the system prompt.

<Callout type="info" title="Important Information">
- **Validation**: Ensure that "Name" and "External ID" are accurate before saving manual records to prevent delivery failures.
- **Integration**: Saved contacts are immediately available for use in the Inbox and Promotions modules.
- **Permanence**: Deletions in this module are final and cannot be reverted once confirmed.
</Callout>

## **Optimization Tips**

- **Rapid Acquisition**: Use the **[WhatsApp Import]** tool to instantly capture new leads who have reached out to your agents but haven't been indexed yet.
- **Granular Targeting**: Apply specific tags like "Interested in [Product X]" so you can easily filter for focused follow-up broadcasts.
- **De-duplication**: When importing Excel files, double-check that your "External ID" (phone/email) matches your previous records to avoid creating duplicate profiles.
- **Growth Auditing**: Use the "Created Date" filter at the end of each month to measure the success of your list-building efforts.
- **Spam Control**: Use the "Blocked" status for disruptive accounts to ensure your marketing performance metrics remain untainted by non-responders.

</Tips>

<Kendala>
## **Troubleshooting**

- **Template Rejection**: If an Excel import fails, verify that you haven't renamed the column headers from the original template.
- **Display Lag**: If a manually added contact doesn't appear immediately, refresh the page to force a server-side sync of the list view.
- **API Latency**: If Google or Facebook imports feel sluggish, check your internet speed and ensure you haven't exceeded the provider's daily API limits.
- **Field Mismatch**: Ensure the "External ID" follows the international mobile format (+62...) if you encounter delivery issues after adding a contact.
- **Filter Malfunction**: Log out and log back in if the filter menu fails to refresh the table results accurately.
- **Interface Errors**: Clear your browser cache if the contact creation form layout appears broken or misaligned.

</Kendala>
