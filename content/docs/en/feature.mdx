## **Manage Chat**

The **Manage Chat** feature is the monitoring center for all conversation data from connected accounts. Here you can see message status, interaction type, and participant details.

| Information Column | Function Description |
| :--- | :--- |
| **Act** | Action buttons to send messages, open chat windows, or use *extra* features. |
| **Name** | Identity or profile name of the contact or group. |
| **Type** | Chat classification (Personal Chat, Group, or Channel). |
| **Participants** | Total number of members if the chat type is Group or Channel. |
| **Username** | Unique account ID/username on the target platform. |

## **Get Group Members**

NoBox Desktop allows you to automatically extract contact databases from groups or channels you follow into the NoBox.AI Cloud system.

### **Extraction Steps:**
1. Select a group/channel in the **[Chats]** menu.
2. Click the **[Act]** button ➔ select **[Get Group Members]**.
3. Check the members you want to take, then click **[Actions]** ➔ **[Add to NoBox.AI]**.
4. Select the target storage folder (list), then click **[OK]**.

## **Core Tools**

NoBox Desktop provides various utility tools to ensure application performance remains optimal and to avoid blocking:

### **1. Tools Setting (Security Settings)**
Used to adjust the **Interval Send Messages**. This feature is crucial for providing pauses between messages so that platform systems (like Telegram/WhatsApp) detect sending as authentic human behavior.

<ImageZoom
  src="/images/en/Settings.png"
  alt="Sending Interval Configuration"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

### **2. Debug Process (Detailed Monitoring)**
Activates an advanced logs window displaying real-time behind-the-scenes activity. Useful for technical teams to monitor message delivery processes.

### **3. Clear Cache (Cleanup)**
Deletes junk files in the internal application browser to resolve slowing display or synchronization issues.

<Tips>
## **Feature Optimization Tips**

- Use an **Interval Send Messages** of at least 10-15 seconds to avoid strict spam detection.
- Perform **Clear Cache** at least once a week to maintain application navigation performance.
- Always use the **Test** feature to send messages to your own personal number before starting a large campaign.
- Utilize **Export to Excel** to have a backup of group contact data you've extracted.

</Tips>

## **Troubleshooting**

If you encounter issues with the features above, please check these steps:

- **Group List Does Not Appear**: Wait a few moments for the initial synchronization process to complete, or click the **Reload Page** button.
- **Fail to Get Group Members**: Ensure you have access rights to see the member list in the relevant group/channel (e.g., not a group with restricted privacy).
- **Message Sending Process Paused**: Check the **Debug Process** tab to see specific error messages, usually due to an expired login session.
- **Application Becomes Slow**: Close unnecessary tabs or chat windows and perform **Clear Cache**.
