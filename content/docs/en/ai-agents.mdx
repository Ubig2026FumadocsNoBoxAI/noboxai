## **Create AI Agent**

To create a new AI Agent, click the **[Add AI Agent]** button and complete the form provided.

### Add AI Agent Dialog

<ImageZoom
  src="/images/id/Dialog%20Tambah%20AI%20Agents.png"
  alt="Add AI Agent"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

### AI Agent Form Fields

| Field                         | Description                                                                 |
| :---------------------------- | :-------------------------------------------------------------------------- |
| **Name**                      | The name or identity of your AI Agent.                                      |
| **Model**                     | The AI engine used to process conversations.                                |
| **Instructions**              | Defines the AI’s personality, role, and response behavior.                  |
| **Category ID**               | Groups agents based on business functions.                                  |
| **Notes**                     | Internal notes for administrative reference.                                |
| **Welcome Message**           | Automatic greeting message when a conversation starts.                      |
| **Agent Transfer Conditions** | Conditions or keywords that trigger transfer to a Human Agent.              |

After saving, the AI Agent will be in **Draft** status. Click **Publish** when you are ready to activate it.

## **Using AI Agent Templates**

The **Template** feature allows you to quickly create AI Agents using predefined configurations.

### Template Dialog

<ImageZoom
  src="/images/id/Dialog%20Template%20AI%20Agents.png"

### **Steps to Use a Template**

1. On the **Add AI Agent** page, click the **[Template]** button.
2. The Template dialog will appear.
3. Select a template from the **Template** dropdown.

<ImageZoom
  src="/images/id/Template%20AI%20Agents.png"
  alt="Template Selection"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

4. Once selected, the system will automatically populate several fields such as:
   - Name
   - Model
   - Instructions
   - Category
5. Review the populated data:
   - Ensure it matches your business needs.
   - Adjust the instructions if necessary.
6. Click **[Apply]** to apply the template to the main form.
7. Continue customizing other fields such as:
   - Welcome Message
   - Transfer Conditions
8. Click **[Save]** to store the AI Agent.

## **Edit and Delete AI Agent**

### Edit AI Agent

Select the agent you want to modify from the list, update the instructions or other configurations, then click **[Save]**.

### Delete AI Agent

Select the AI Agent you want to delete, click **[Delete]**, and confirm your action in the dialog that appears.

<Callout type="info" title="Important Information">
  Deleting an AI Agent is permanent. The agent will immediately stop serving customers and all configurations will be removed from the system.
</Callout>

## **Usage Tips**

Optimize your AI Agent performance with the following tips:

- Provide detailed **Instructions** so the AI understands its scope and limitations.
- Use **Welcome Message** to gather basic customer information automatically.
- Define clear **Transfer Conditions** to ensure smooth escalation to human agents.
- Perform testing in **Draft** status before publishing.
- Use the latest AI model for more natural and accurate responses.

</Tips>

## **Troubleshooting**

If you encounter issues with AI Agents, follow these guidelines:

- **AI Not Responding**: Ensure the agent status is **Published** and your communication channel (WhatsApp/Telegram) is properly connected.
- **Irrelevant Responses**: Review and refine the instructions for better clarity.
- **AI Credits Exhausted**: Check your subscription and top up if necessary.
- **Failed Transfer to Human Agent**: Ensure there is an available Human Agent online.
- **Changes Not Saved**: Check your internet connection or refresh the page.

</Kendala>
