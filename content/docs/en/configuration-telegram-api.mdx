API (*Application Programming Interface*) is a protocol that allows the NoBox application to communicate securely with Telegram servers. To use the Telegram Automation feature, you are required to have your own **API ID** and **API Hash** so that your message sending activity has a unique and trusted application identity.

The process of creating this API is free and only needs to be done once through the official Telegram developer portal.

---

## **Stage 1: Telegram Account Authentication**

The first step is to log in to the Telegram API management portal using your phone number.

1. Visit the official site: [https://my.telegram.org/auth](https://my.telegram.org/auth).
2. Enter the phone number registered in Telegram (Use international format, e.g., +6281xxxxxx).
3. A confirmation code will be sent via your **Telegram chat** (not SMS). Enter the code and click **[Sign In]**.

## **Stage 2: Create a New Application**

After successfully logging in, you must register an application identity for your account.

1. On the main **[Your Telegram Core]** menu, select the **[API development tools]** option.
2. You will be directed to the application creation page (**Create New Application**).
3. Fill in the available fields according to the following instructions:

| Field | Function Description | Example Data |
| :--- | :--- | :--- |
| **App title** | Your application name (any name). | NoBox Automation |
| **Short name**| Short application name. | NBAuto |
| **URL** | Your website address (can be skipped). | - |
| **Platform** | Application usage platform. | Desktop |
| **Description**| Brief application description. | Automation for leads |

4. Click the **[Create Application]** button to confirm creation.

## **Stage 3: Getting and Saving API**

After the application is successfully created, the system will display your API details.

1. You will see **App api_id** (numerical) and **App api_hash** (alphanumeric combination).
2. Copy both data items and save them in a safe place.
3. Use this data when you add a new Telegram account in the NoBox Desktop application.

<Tips>
## **API Security Tips**

- Keep your **api_hash** confidential; do not give this code to any party outside the NoBox.AI system.
- Use an application name that looks common or personal so as not to trigger Telegram's security system suspicion.
- Ensure the phone number used to create the API is the same number you want to set as the automated sender.

</Tips>

<Kendala>
## **Handling API Registration Issues**

If you encounter obstacles when creating an API, please check the following points:

- **"Create Application" Failed**: Telegram sometimes restricts API creation on new phone numbers. Try using a number that has been active for at least 1-2 weeks.
- **"Invalid Name" Error**: Ensure *Short Name* and *App Title* only use alphabets and numbers without special characters or excessive spaces.
- **Site Not Responding**: Use the latest Google Chrome browser in *Incognito* (Private) mode to avoid cache data conflicts.
- **OTP Code Not Received**: Ensure the Telegram application on your phone remains open, as code is only sent through Telegram's internal chat system.

</Kendala>
