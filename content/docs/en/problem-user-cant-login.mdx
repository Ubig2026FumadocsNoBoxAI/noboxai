## **Account Registration Check**

Before trying to log in, ensure you have completed the entire account registration process on our official website.

### Correct Registration Steps:

1. Visit [https://NoBox.AI/](https://NoBox.AI/) and click the **Login** button.
2. Select the server location closest to your operational area for the best access speed.
3. Use the **Sign Up** menu and complete the new tenant registration form.
4. **Important:** Check your email inbox or spam folder, then click the verification link we sent. Your account cannot be used until this verification process is complete.

## **Login Issue Solutions**

If you already have an account but still cannot log in, please follow these guidelines:

### 1. Web Dashboard Login

Use the email and password registered at [id.nobox.ai](https://NoBox.AI/). In this dashboard, you can transparently monitor all data and your subscription package status.

### 2. NoBox Desktop App Login

The desktop application requires the same credentials as your website account. If you just registered, wait about 1-2 minutes for the database to synchronize before trying to log in to the desktop app.

<Callout type="info" title="Important Information">
  One NoBox account can only be active in one primary desktop login session. If
  you try to log in on a new device, the session on the previous device may be
  disconnected automatically for data security.
</Callout>

<Tips>
## **Safe & Smooth Login Tips**

- Always save your password in a secure password manager to avoid typos.
- Ensure the **Caps Lock** key is not active when entering your password.
- Use a private internet connection; using a VPN or Proxy can sometimes hinder the server authentication process.
- Clear your browser cache periodically if the login page feels slow or unresponsive.

</Tips>

<Kendala>
## **Advanced Problem Handling**

If you still cannot log in to your account, try these steps:

- **Forgot Password**: Use the [Forgot Password](forgot-password) feature to reset your password via email.
- **Verification Email Not Received**: Click the "Resend Verification Email" button on the login page or contact admin support if the email still doesn't arrive within 10 minutes.
- **"Tenant Not Found" Error Message**: Ensure you log in through the correct server URL used during initial registration.
- **Desktop Application Blank**: Ensure the application has full internet access permissions and is not blocked by antivirus.

</Kendala>
