This menu is specifically designed to manage operational team members who handle and monitor customer interactions.

## **Inviting New Agents**

Adding Human Agents is done through an invitation system to ensure company data security.

### How to Invite an Agent

1. Click the **[Copy Invitation]** button located at the top right of the page.

<ImageZoom
  src="/images/id/Salin%20Undangan%20Human%20Agents.png"
  alt="Copy Invitation Button"
  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "
/>

2. A unique invitation link will be automatically copied to your clipboard.
3. Send the link to your team member via your preferred communication channel (WhatsApp, Email, etc.).
4. Once the agent registers through the link, their account will appear in the list with an active status.

The **[Unlimited]** label indicates that you can add as many agents as needed without additional cost per agent.

## **Edit and Delete Human Agent Data**

### Edit Agent Data

To modify agent information such as role assignment, select the desired agent from the list. Click the icon  
[ <img src="/images/id/role.png" alt="Edit Role" className="inline h-5 w-auto mx-1" /> ]  
to open the **[Edit User Access Role]** dialog.

### Edit User Access Role Dialog

<ImageZoom
  src="/images/id/Edit%20Role%20Human%20Agents.png"

  width={800}
  height={450}
  className="rounded-lg border shadow-sm mb-4 my-8 mx-auto "

Make changes to the agent’s access permissions, then click **[OK]**.

### Delete Agent Data

Select the agent you want to remove, click the delete icon  
[ <img src="/images/id/Tong%20Sampah.png" alt="Delete" className="inline h-5 w-auto mx-1" /> ],  
and confirm your action in the dialog that appears.

<Callout type="info" title="Important Information">
  Deleting a Human Agent is permanent. The removed agent will no longer be able
  to access the system and all permissions will be revoked immediately.
</Callout>

## **Usage Tips**

Here are some best practices for managing Human Agents:

- Assign roles according to each team member’s responsibilities to maintain data security.
- Use official company email addresses for a more professional setup.
- Regularly monitor the status column to ensure all team members have verified their accounts.
- Utilize the unlimited invitation feature to scale your team when message traffic increases.

</Tips>

<Kendala>
## **Troubleshooting**

If you encounter issues while managing agents, follow these guidelines:

- **Invitation Link Not Working**: Regenerate the link by clicking the **[Copy Invitation]** button again.
- **Agent Not Appearing in the List**: Ensure the user has completed the registration process via the invitation link.
- **Verification Email Not Received**: Ask the agent to check the Spam folder or resend verification.
- **Unable to Change Roles**: Make sure you are logged in as an authorized user (Owner/Manager).
- **Forgot Password**: Direct the agent to use the "Forgot Password" option on the login page.

</Kendala>
