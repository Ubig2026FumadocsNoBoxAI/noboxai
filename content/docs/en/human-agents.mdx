Human Agents at NoBox.Ai serve as the control center for managing all human users who have access to your platform. This feature allows you to invite team members, assign roles, and manage their access rights. If AI Agents are virtual robots, then Human Agents are your real-world team—such as customer service staff, managers, and administrators—responsible for operating, supervising, and managing customer interactions.

## **Inviting New Agents**

Adding Human Agents is done through an invitation system to ensure security. To invite a new agent, please click the ➔ **[Copy Invitation]** button. This will copy a unique invitation link. Send this link to prospective team members. Once they register via the link, their accounts will automatically be added to the Human Agents list with **[VERIFIED]** status. The **[Unlimited]** label next to it indicates that you can invite as many agents as you need without any quota limits.

## Edit and Delete Human Agent Data

To edit a Human Agent's data, please first select the data you want to edit (for example, by clicking on the agent's name). Edit the data such as roles and click **[Save]** to save the changes. To delete a Human Agent's data, select the Human Agent data you want to delete, then click the **Delete (trash can)** icon, and a confirmation dialog will appear asking if you want to delete the data. If yes, click **[Yes]** if not, click **[No]**.
